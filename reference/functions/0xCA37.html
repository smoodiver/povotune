Summary

Multi-stage load threshold evaluator that determines whether the engine is operating above critical load levels and, when conditions are met, drives a ramp output (0x3120) and scaled diagnostic values. Uses ramp_enable_flags (0x3023) as its primary state register. Stage 1 compares airflow_load against throttle_position via a hysteresis-gated 2D lookup (table 0x43E0) to set 0x3023 bit 3. Stage 2 performs a second 2D lookup (table 0x43A0) on airflow_load, scales the result and compares against engine_load (0x313C) with optional ROM offset (0x4042) to set 0x3023 bit 2. Stage 3, when both bits 2 and 3 are set, classifies engine_run_counter (0x30B6) into four speed tiers via ROM thresholds and compares a counter (0x30D1) against speed-tier-indexed thresholds (0x48A8) to determine the final ramp enable (0x3023 bit 0) or ramp-up-pending (0x3023 bit 6). The output at 0x3120 is right-shifted 5 bits and stored to 0x374E and IO port 0x013E.

Inputs

  • 0x306F (airflow_load): primary input for both 2D lookups
  • 0x3044 (throttle_position): comparison target for stage 1
  • 0x313C (engine_load): comparison target for stage 2
  • 0x30B6 (engine_run_counter): speed tier classifier input
  • 0x30D1 (load_ramp_counter): ramp comparison counter
  • 0x305A (system_control_flags): bit 2 gates early clear of 0x30D1
  • 0x3051 (map_mode_control): bit 2 gates early clear of 0x30D1
  • 0x305C (system_control_flags_b): bit 2 — inhibit gate
  • 0x402E (ROM): all-bits check — alternate inhibit gate
  • 0x3023 (ramp_enable_flags): state register — bits read and written

Outputs

  • 0x3023 (ramp_enable_flags): bits 0–6
    • bit 0 (0x01): ramp enabled (load thresholds exceeded, counter at max)
    • bit 2 (0x04): engine_load threshold exceeded
    • bit 3 (0x08): throttle load threshold exceeded (hysteresis)
    • bit 6 (0x40): ramp-up pending (threshold exceeded, counter not at max)
  • 0x30D1 (load_ramp_counter): set 0xFF when conditions met, 0x00 when not
  • 0x3120 (ramp_accumulator): cleared to 0 when conditions not met
  • 0x374E (load_ramp_output): 0x3120 >> 5 — scaled output
  • 0x013E (IO port): same as 0x374E
