Summary

Computes a closed-loop O2 sensor feedback correction factor for fuel injection. The function performs three major phases:

  1. AFM floor clamp (0xB811–0xB822): Loads the stored AFM value ($301A), clamps it to a minimum from ROM $4C1D, and stores the result to $32F4 as the base enrichment reference.

  2. Base fuel correction (0xB823–0xB948): Loads airflow load ($306F), subtracts a 0x20 offset (saturating to 0), multiplies by ROM coefficient $4C1F to produce a 16-bit scaled value stored at $3008. Then performs a 1D table lookup on coolant temperature ($3093) using ROM table at $4C30 via JSR $6000, storing the result at $32E2. The lookup result minus $3009 (clamped to minimum $4C21) becomes the correction span at $32E4. This is used to compute a complement value at $32E6 = (0x100 - correction), and an interpolated blend factor at $32EA. The function then computes an IIR-filtered running average at $32F0 using multi-precision multiply-accumulate operations with intermediate storage at $3008–$300B.

  3. O2 feedback integration (0xB955–0xBA6B): When the blend factor ($32EA) is nonzero and AFM ($301A) is nonzero, the function checks whether conditions allow O2 feedback correction. It reads speed control flags ($302A bit 3), a state register at $32F8, injection sequencer state ($31B4 bit 0), and various engine condition values ($3071, $3044/$3045, $3268, $32DD). If closed-loop conditions are met, it computes a signed correction delta, scales it by the blend factor, clamps the result to [0xB000..0x7FFF] centered at 0x8000, and stores the feedback integrator at $32F2. The final output writes shifted copies of $32F0 and $32EC to SSM diagnostic ports $0114, $0116, and $0118.

Inputs

  • $301A (afm_stored_value): current AFM reading
  • $306F (airflow_load): airflow-derived engine load
  • $3093 (coolant_temp): coolant temperature for table lookup
  • $302A (speed_control_flags): bit 3 gates feedback, bit 5 checked for state
  • $31B4 (inj_seq_state): bit 0 enables feedback integration
  • $305A (system_control_flags): bit 4 checked for mode gating
  • $32F8: O2 feedback state flags (bit 6: lean-detect latch, bit 7: inhibit flag)
  • $3268: RPM delta (signed, from 0xBA6C)
  • $32DD: AFM rate-of-change (signed, from 0xBA6C)
  • $3044 (throttle_position), $3045: throttle comparators
  • $3071 (vehicle_speed): speed threshold check

Outputs

  • $32F0: filtered base correction value (IIR averaged)
  • $32F2: O2 feedback integrator (0x8000 = neutral, >0x8000 = rich correction, <0x8000 = lean correction)
  • $32F4: clamped AFM reference
  • $32E2: coolant temp correction lookup result
  • $32E4: correction span
  • $32E6: correction complement (0x100 - span)
  • $32E8: enrichment factor
  • $32EA: blend factor
  • $32EC: combined correction output
  • $32EE: absolute delta (for threshold check)
  • $0114: SSM diagnostic — filtered correction >> 7
  • $0116: SSM diagnostic — combined correction >> 4
  • $0118: SSM diagnostic — feedback integrator high byte
