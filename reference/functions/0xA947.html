0xA947 — Knock Sensor Mode Classifier

Field Value
Address 0xA947
Size 93 bytes (0xA947–0xA9A3)
Blocks 15
Type subroutine
Callers 1 — Timer A ISR Phase 0 (0x98A1 at 0x99D6)
Confidence confirmed

Summary

Classifies knock sensor operating mode and sets up the knock correction enable flag ($31C2) and sensor activity flag ($31C1). Also evaluates RPM and airflow conditions to determine whether to enable knock correction feedback via $31FC bit 6.

The function performs three tasks:

  1. Knock correction enable: Tests $316B (lookup_input_b) against threshold 0x57 and external acquisition state to set $31C2 to 0x01 or 0x00.
  2. Sensor activity: Tests direct-page $0B bit 2 to determine sensor mode — sets $31C1 to 0x00 (active mode) or 0x88 (inactive/diagnostic mode).
  3. Knock feedback gate: Checks airflow load ($306F) and a signed RPM-derived value ($326A) against ROM thresholds. If both exceed thresholds, loads a ROM constant and optionally enables feedback.

Disassembly

; --- knock correction enable ---
A947  LDA $316B           ; load lookup_input_b
A94A  CMP #$57            ; compare against threshold
A94C  BCC $A962           ; if < 0x57, -> skip enable check
A94E  BBS $325A,#$80,$A958 ; if ext_acq running, -> enable
A953  BBS $3253,#$20,$A95E ; if port bit 5 set, -> disable
A958  LDM $31C2,#$01      ; enable knock correction
A95C  BRA $A962
A95E  LDM $31C2,#$00      ; disable knock correction
; --- sensor activity ---
A962  LDA #$00            ; default: active mode
A964  BBS $0B,#$02,$A96A  ; if direct-page $0B bit 2 set, keep 0x00
A968  LDA #$88            ; else: inactive mode = 0x88
A96A  STA $31C1           ; store sensor_activity_flags
A96D  STA $0216           ; mirror to diagnostic output
; --- cylinder ID update ---
A970  LDA $31CB           ; load knock_correction_combined
A973  STA $31C3           ; store as current_cylinder_id
; --- knock feedback gate ---
A976  LDA $306F           ; load airflow_load
A979  CMP $445A           ; compare against ROM airflow threshold
A97C  BCC $A995           ; if below threshold, -> check existing
A97E  LDX $326A           ; load RPM-derived value (signed)
A981  BMI $A995           ; if negative, -> check existing
A983  CPX $4458           ; compare against ROM RPM threshold
A986  BCC $A995           ; if below, -> check existing
A988  LDA $445B           ; load ROM feedback constant
A98B  STA $32C9           ; store to knock feedback register
A98E  BBC $35A4,#$01,$A995 ; if diag self-test bit 0 clear, -> check existing
A993  BRA $A99F           ; else -> enable feedback
; --- check existing state ---
A995  LDA $32C9           ; load current knock feedback register
A998  BNE $A99F           ; if nonzero, -> enable feedback
A99A  CLB $31FC,#$40      ; clear knock enable flag
A99E  RTS
; --- enable feedback ---
A99F  SEB $31FC,#$40      ; set knock enable flag
A9A3  RTS
