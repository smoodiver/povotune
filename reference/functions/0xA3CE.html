0xA3CE — RPM Hysteresis Gate (Phase 2)

Field Value
Address 0xA3CE
Size 38 bytes (0xA3CE–0xA3F3)
Blocks 8
Type subroutine
Callers 1 — Timer A ISR Phase 2 (0x98A1 at 0x9A98)
Confidence confirmed

Summary

RPM-based hysteresis gate called during ISR Phase 2. Tests the RPM period ($3062) against two ROM thresholds with hysteresis to set or clear bit 2 of $31A1 (engine_running_flags). When the engine is above the high-speed threshold, bit 2 is set; when below the low-speed threshold, bit 2 is cleared. Also clears $31CC (af_ratio_mode) to zero when leaving the high-speed zone.

An early-exit check on $305A bit 4 (system_control_flags) bypasses the comparison entirely, forcing the bit clear and zeroing $31CC.

Disassembly

A3CE  LDX $3062           ; load rpm_period
A3D1  BBS $305A,#$04,$A3EB ; if system_control_flags bit 4 set, -> clear
A3D6  BBS $31A1,#$02,$A3E6 ; if already in high-speed zone, check exit threshold
A3DB  CPX $4456           ; compare period against high-speed entry threshold
A3DE  BCC $A3EF           ; if period < threshold (faster), -> enter zone
A3E0  SEB $31A1,#$02      ; set bit 2 (high-speed zone active)
A3E4  BRA $A3F3           ; -> return
A3E6  CPX $4454           ; compare against low-speed exit threshold
A3E9  BCS $A3F3           ; if period >= threshold (still fast enough), stay
A3EB  CLB $31A1,#$02      ; clear bit 2 (exit high-speed zone)
A3EF  LDM $31CC,#$00      ; clear af_ratio_mode to 0
A3F3  RTS

Inputs

  • $3062 (rpm_period) — current RPM period (lower = faster)

Outputs

  • $31A1 bit 2 — set when RPM period below entry threshold, cleared when above exit threshold
  • $31CC — cleared to 0 when exiting high-speed zone
