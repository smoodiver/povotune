0xA6EB — RPM Range Classifier (Limiter Flags)

Field Value
Address 0xA6EB
Size 90 bytes (0xA6EB–0xA744)
Blocks 13
Type subroutine
Callers 1 — Timer A ISR Phase 2 (0x98A1 at 0x9A9E)
Confidence confirmed

Summary

Classifies the combined timing value ($31AA:$31AC) against ROM thresholds to set limiter control flags in $314E. Also reads speed limit flags ($31E0) as a secondary input. The function produces a 4-bit flag pattern in $314E that controls the fuel cut / rev limiter system.

The classification has three RPM ranges:

  1. Very low RPM / overflow ($31AC nonzero): Clears bits 0+1, then checks speed flags for secondary classification.
  2. Below low threshold ($31AA <= $441C): Sets bits 0+1 (both limiter zones active).
  3. Between thresholds ($441C < $31AA <= $4418): Sets bit 0 only, clears bit 1.

For the secondary flags (bits 6 and 7):

  • Speed limit bit 1 set -> bit 6 set, bit 7 clear (partial limit)
  • Speed limit bit 2 set -> bit 6 clear, bit 7 set (full limit)
  • Neither -> bits 6 and 7 both cleared (no limit)

Disassembly

A6EB  LDA $31AC           ; load combined_timing_high
A6EE  BNE $A71C           ; if nonzero (overflow), -> low RPM path
A6F0  CLM                 ; 16-bit mode
A6F1  LDA $31AA           ; load combined_timing_low
A6F4  CMP $4418           ; compare against upper threshold
A6F7  BEQ $A6FB
A6F9  BCS $A71C           ; if timing > upper threshold, -> low RPM
A6FB  CMP $441C           ; compare against lower threshold
A6FE  BEQ $A702
A700  BCS $A70F           ; if timing > lower threshold, -> mid range
; --- very low timing (high RPM) ---
A702  SEB $314E,#$01      ; set limiter bit 0
A707  SEB $314E,#$02      ; set limiter bit 1
A70C  SEM                 ; back to 8-bit
A70D  BRA $A73B           ; -> set bit 7
; --- mid range ---
A70F  SEB $314E,#$01      ; set limiter bit 0
A714  CLB $314E,#$02      ; clear limiter bit 1
A719  SEM
A71A  BRA $A731           ; -> set bit 6
; --- low RPM / overflow ---
A71C  SEM
A71D  CLB $314E,#$03      ; clear bits 0+1
A721  BBS $31E0,#$02,$A73B ; if speed limit bit 1 set, -> full limit
A726  BBS $31E0,#$01,$A731 ; if speed limit bit 0 set, -> partial limit
A72B  CLB $314E,#$C0      ; clear bits 6+7 (no limit)
A72F  BRA $A743           ; -> return
; --- secondary limit flags ---
A731  SEB $314E,#$40      ; set bit 6 (partial limit)
A735  CLB $314E,#$80      ; clear bit 7
A739  BRA $A743
A73B  CLB $314E,#$40      ; clear bit 6
A73F  SEB $314E,#$80      ; set bit 7 (full limit)
A743  SEM
A744  RTS

Inputs

  • $31AA (combined_timing_low) — 16-bit combined timing sum low word
  • $31AC (combined_timing_high) — timing sum high word (nonzero = overflow)
  • $31E0 (speed_limit_flags) — bits 0+1: speed-based limit inputs

Outputs

  • $314E (limiter_flags) — bits 0,1,6,7 set/cleared based on RPM range and speed
