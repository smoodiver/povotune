0x79C7 — Blink Pattern Generator

Size ~79 bytes (0x79C7–0x7A15)

Type subroutine

Confidence confirmed

Summary

Converts a DTC code (0x3192) into a timed blink sequence for the MIL lamp. Implements a two-phase blink pattern: the DTC code value directly controls the number of blinks, with the lamp toggling ON and OFF for each count.

The function operates as a state machine driven by 0x35A5 bit 5 (current lamp state):

First entry (lamp OFF, bit 5 clear): Reads DTC code from 0x3192. If zero, exits without action. If >= 0x10 (16), sets long ON duration (0x17 = 23 ticks). If < 0x10, sets short ON duration (0x04 = 4 ticks). Sets lamp ON (0x35A5 bit 5) and stores duration in 0x3191.

Subsequent entries (lamp ON, bit 5 set): Decrements the DTC code:

  • If code >= 0x10: subtracts 0x10, stores remainder. If remainder is non-zero, sets OFF duration (0x06 = 6 ticks). If remainder is zero, sets long pause (0x23 = 35 ticks) — end of tens digit. Clears lamp (bit 5 clear).
  • If code < 0x10: decrements by 1. If non-zero, sets OFF duration (0x06 = 6 ticks). If zero, sets long pause (0x23 = 35 ticks) — end of code. Clears lamp (bit 5 clear).

This creates a two-digit blink pattern: the tens digit blinks with long ON pulses (0x17 ticks each), followed by a pause, then the ones digit blinks with short ON pulses (0x04 ticks each), followed by a longer pause before the next code.

Inputs

  • 0x3192: DTC code value (set by fault_code_bit_scanner 0x7928)
  • 0x35A5 bit 5: current MIL lamp state

Outputs

  • 0x3191: blink phase timer (countdown value for MIL lamp driver)
  • 0x3192: decremented DTC code (consumed during blink sequence)
  • 0x35A5 bit 5: toggled MIL lamp state

Calls

  • 0x920B (port_2000_shadow_write): updates physical port after lamp state change
