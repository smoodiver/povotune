cross-reference Heritage Warning

Labeled "ISC Valve Position Sampler" from cross-reference SVX heritage. On the BD5A (EJ20H twin turbo), this function reads ADC channel AN5 which feeds the turbo feedback sensor (0x32AE), used as proportional controller feedback by the twin turbo duty controller (0xCC9F). The physical sensor identity on BD5A needs board-level pin trace verification.

Summary

Samples the ISC (Idle Speed Control) valve position from the ADC result register and computes a 4-sample moving average used for idle control feedback. Called every ~4th scheduler tick (P2 priority). The function reads the ADC result from dp+$22, applies a scaling factor, and maintains a 4-deep history buffer at 0x3368–0x336E. The averaged result is written to 0x3370 (16-bit) and mirrored to IO port 0x011E and to 0x3796 (8-bit, shifted right 4).

Flow

  1. Early exit check: Reads 0x32B7 (ISC activity counter); if below ROM threshold 0x48C6 (0x0A = 10), checks 0x35A4 bit 0 — if clear, returns early (not enough samples yet).
  2. ADC capture: Busy-waits on dp+$20 bit 3 (ADC ready flag), then loads X from dp+$22 (the raw ADC result). Stores X to 0x3008 (scratch) and conditionally to 0x3360 (if 0x35A4 bit 0 is set) or 0x329E (ISC valve position, if 0x31DF bit 3 is clear).
  3. Scaling: Switches to 16-bit mode. Multiplies 0x3008 by constant 0x01F4 (=500), stores result. Subtracts 0x012F (=303) from the high word, clamping to 0 minimum. Stores scaled value to 0x3362.
  4. RPM gate: If 0x3062 >= 0x60 (RPM above threshold), skips averaging and clears 0x3166 bit 6 (history-valid flag).
  5. History buffer (sliding window): If 0x3166 bit 6 is already set (history valid), shifts values down: 0x3368 -> 0x336A -> 0x336C -> 0x336E, inserts 0x3362 at 0x3368. If bit 6 is clear (first time), initializes all 4 slots to the current value and sets bit 6.
  6. Averaging: Sums 0x3368 + 0x336A + 0x336C + 0x336E, divides by 4 (two LSR). Stores 16-bit result to 0x3370 and IO 0x011E. Shifts right 4 more to get 8-bit value, stores to 0x3796.
