0xAE20 — Stall/Cranking Detector

Field Value
Address 0xAE20
Size 126 bytes (0xAE20–0xAE9D)
Blocks 18
Type subroutine
Callers 1 — scheduler ISR (0xAAD1) P4 dispatch at 0xAB4A
Callees 1 — JSR $A3C7 (shift_right_BA_by_2, divide by 4)
Confidence confirmed

Summary

Detects engine stall and cranking conditions, and computes a boost startup / cranking injection base value. Runs in the scheduler P4 priority slot (every 20th tick). The function monitors the cylinder event counter ($319C), system flags ($305A bit 4), and knock sensor reading ($306E) to determine cranking state. When cranking is detected, it sets the cranking-active flag ($3020 bit 3) and the cranking-init-request flag ($330A bit 2). It also computes the boost startup value ($3130) by selecting from a ROM lookup table ($43F8) based on knock level and operating conditions, then writes a scaled copy to the IO diagnostic port ($018A).

Detailed Flow

Stall detection (0xAE20–0xAE40):

AE20  LDA $319C           ; cylinder event counter
AE23  BEQ $AE38           ; if zero → engine stalled
AE25  BBC $305A,#$10,$AE41 ; if system bit 4 clear → normal path
AE2A  LDA $312C           ; stall counter
AE2D  INA                 ; increment (saturating at 0xFF)
AE2E  BEQ $AE33           ; if wrapped to 0, don't store
AE30  STA $312C           ; store incremented counter
AE33  CLB $3020,#$08      ; clear cranking-active flag
AE37  RTS

When $319C is zero (no cylinder events), the function clears the cranking flag, zeros the stall counter, and returns. When $305A bit 4 is set (system in a known-good state), it simply increments the stall counter and clears cranking.

Cranking detection (0xAE41–0xAE5E):

AE41  BBS $3020,#$08,$AE62 ; if already cranking → skip to boost calc
AE46  LDA $306E           ; knock sensor reading
AE49  CMP $4402           ; compare against low threshold
AE4C  BCS $AE38           ; if above → not cranking, clear and return
AE4E  LDA $312C           ; stall counter
AE51  CMP $4405           ; compare against stall count threshold
AE54  BCC $AE38           ; if below → not enough stall counts yet
AE56  LDM $312C,#$00      ; reset stall counter
AE5A  SEB $3020,#$08      ; set cranking-active flag
AE5E  SEB $330A,#$04      ; set cranking-init-request flag

Cranking is detected when: (1) the knock sensor ($306E) reads below the low threshold ($4402), AND (2) the stall counter ($312C) has reached the required count ($4405). This prevents false cranking detection from brief signal dropouts.

Boost startup calculation (0xAE62–0xAE9D):

AE62  LDX #$43F8          ; base of 4-entry ROM table
AE65  LDA $306E           ; knock sensor reading
AE68  CMP $4404           ; compare against high threshold
AE6B  BCC $AE6E           ; if below, use base index
AE6D  INX                 ; +1 for high knock
AE6E  BBC $302A,#$08,$AE75 ; check operating condition flag
AE73  INX                 ; +2 for alternate condition
AE74  INX
AE75  LDA $00,X           ; load table value
AE77  MPY $312F           ; multiply by correction factor
AE7B  JSR $A3C7           ; divide by 4 (shift right B:A by 2)

The table index selects from 4 entries: {base, high-knock, alt-condition, alt+high-knock}. After multiplication and scaling:

AE86  LDA $3008           ; scaled result
AE89  CLC
AE8A  ADC $3056           ; add offset
AE8D  BCC $AE92
AE8F  LDA #$FFFF          ; saturate on overflow
AE92  STA $3130           ; store as boost_startup_value
AE95  LSR A               ; divide by 16 (4x LSR)
AE96  LSR A
AE97  LSR A
AE98  LSR A
AE99  SEM                 ; 8-bit mode
AE9A  STA $018A           ; write to IO diagnostic port
AE9D  RTS

Inputs

Source Address Description
RAM $319C Cylinder event counter (from Timer A ISR)
RAM $305A System flags — bit 4 = system initialized/running
RAM $3020 Engine state flags — bit 3 = cranking active
RAM $306E Knock sensor reading
RAM $312C Stall counter (incremented each call while running)
RAM $312F Boost startup correction factor (multiplier)
RAM $302A Operating condition flags — bit 3 selects table offset
RAM $3056 Boost startup offset added to scaled result
ROM $4402 Knock threshold low — below this = potential cranking
ROM $4404 Knock threshold high — selects higher startup value
ROM $4405 Stall count threshold — minimum stall counts before cranking detect
ROM $43F8 4-entry boost startup multiplier table

Outputs

Address Description
$3020 Bit 3: cranking-active flag (set on cranking detect, cleared on stall/running)
$330A Bit 2: cranking-init-request flag (set on cranking detect, consumed by 0xAE9E)
$312C Stall counter — incremented when running, zeroed on stall or cranking detect
$3130 Boost startup / cranking injection base value (16-bit)
$018A IO diagnostic port — boost startup value >> 4 (8-bit)
