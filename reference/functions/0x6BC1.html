0x6BC1 — Starter/Battery Voltage Check

Field Value
Address 0x6BC1
Size 89 bytes
Blocks 15
Type subroutine
Callers 1 (health_check_dispatcher 0x6B14)
Confidence confirmed

Summary

Checks starter motor engagement and battery voltage during cranking. First tracks whether the system_control_flags (0x305A bit 2) has been set, using 0x32C4 bit 0 as a latch and 0x32C5 as a saturating up-counter. When the ISR event flag (0x322C bit 6) indicates a crank signal has fired, checks 0x32C4 bit 1 for a prior fault. If no crank signal, waits for 0x32C5 to reach 0x28 (40 counts) before performing an AD converter read: disables interrupts, configures AD register 0xB5 (channels 0x14/0x12), reads the result from direct page 0xA4, stores it to 0x300C, and compares against threshold 0x18. If the reading is below 0x18 (low voltage), clears fault bit 0x3330 bit 0; if at or above, sets 0x3330 bit 0 (starter/battery fault).

Inputs

  • 0x32C4 bit 0: system ready latch
  • 0x32C4 bit 1: prior starter fault flag
  • 0x32C4 bit 2: AD read already performed flag
  • 0x305A bit 2: system control flag (closed-loop gate)
  • 0x32C5: cranking event counter
  • 0x322C bit 6: ISR crank signal event flag
  • Direct page 0xA4: AD converter result register
  • Direct page 0xB5: AD converter control register

Outputs

  • 0x32C4 bits 0,1,2: state tracking flags
  • 0x32C5: cranking counter (incremented, saturating)
  • 0x300C: stored AD converter reading
  • 0x3330 bit 0: starter/battery voltage fault flag

RAM Reads

  • 0x32C4, 0x32C5, 0x305A, 0x322C

RAM Writes

  • 0x32C4, 0x32C5, 0x300C, 0x3330
