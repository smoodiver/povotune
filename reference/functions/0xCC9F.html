0xCC9F — Twin Turbo Duty Controller

Field Value
Address 0xCC9F
Size 2989 bytes (0xCC9F–0xD84B)
Blocks 445
Type subroutine
Callers 1 — from scheduler at 0xABB6
Confidence confirmed

Summary

Master sequential twin turbo duty cycle controller. Manages the PWM solenoid valve duty cycles that control exhaust gas routing, pre-spin, and transition timing for the EJ20H sequential twin turbo system. Uses two adjustment map lookups (Call 1 and Call 2) with a proportional step-adjustment controller to drive 6 PWM timer outputs and a digital port output.

This is the largest function in the ROM (445 blocks). Called from the scheduler — not part of the 23-function main loop or periodic tasks.

Call 1 — Primary Turbo Duty Control

Maps: 0x4790 (twin mode) or 0x4940 (single mode) — 8×8, range 0x84–0xB6 Output: 4 PWM channels (0x01D0, 0x01D2, 0x01DA, 0x01DC) Algorithm: Proportional step controller

  1. Look up target from map using RPM × throttle position
  2. Scale by $31E5 × $31E6 → $3296
  3. Compare $3296 (target) vs $32AE (feedback from 0xB44B)
  4. If target > feedback: use increase step table (0x48E0), set $328A bit 5
  5. If feedback > target: use decrease step table (0x4730), clear $328A bit 5
  6. Magnitude thresholds at 0x4854–0x4857 select step size (4 levels)
  7. Scale existing duty values by factor from 0x4980, saturating at 0xFF
  8. Write final duty values to I/O ports

Call 2 — Secondary Turbo Duty Clamp

Maps: 0x4F00 (single mode) or 0x4F40 (twin mode) — 8×8, range 0x00–0xFF Output: 2 PWM channels (0x01D4, 0x01D8) and port 0x2200 Algorithm:

  1. Look up clamp from map → $316E
  2. Perform 1D graph lookups using $306F (airflow) at tables 0x4720 and 0x4750
  3. Clamp combined duty value at $316E (at 0xD2F8)
  4. Apply minimum ($4859) and maximum ($482F) bounds
  5. Conditional logic based on $316D bits, $31DE flags, $3020 state
  6. Final output: $3293 → 0x01D4, $316F → 0x01D8, $3275 → 0x2200

Turbo Mode Switching ($3291 bit 3)

Bit 3 is managed internally by this function:

  • Set at 0xD779 (along with bits 4, 5): transitions to twin-turbo mode. Triggered when accumulated conditions in the mode-switching section are met.
  • Cleared at 0xD4AB: transitions to single-turbo mode. Clears several working variables ($332A, $332B, $3280, $3381, $3382).

When bit 3 changes, the next execution cycle uses a different set of maps and axes, switching between single-turbo and twin-turbo calibration.

Why Modified These Tables

Increased the Call 1 map values (0x4790/0x4940) to raise the PWM duty cycle targets for the turbo solenoid valves. Higher targets mean the proportional controller drives the solenoids harder — resulting in:

  • More exhaust gas routing to the secondary turbo
  • Faster spool-up of the secondary turbine
  • More aggressive turbo transition between single and twin modes

The Call 2 maps (0x4F00/0x4F40) set the upper duty clamp for the secondary turbo. Higher values allow the secondary turbo to reach full duty sooner, enabling peak performance at lower RPM points.

Combined with the higher max boost duty limit (0x4BC0), these changes create a more aggressive twin turbo system that transitions faster and builds boost more aggressively than stock.
