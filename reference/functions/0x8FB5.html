0x8FB5 — Idle Duty Controller

Confidence likely

Summary

Idle speed control (ISC) valve duty cycle controller. Manages a stepped ramp of the ISC duty target (0x351E) with qualification gating, clamping, and coolant temperature correction. Checks master enable (0x4B7E), qualifies conditions (warm engine, idle state, timer interval), then computes new duty from airflow correction + offset, clamped to ROM limits. Applies coolant-temp correction by looking up table 0x4B20 and scaling around 0x4000 midpoint bias. Final corrected value written to I/O port 0x021E.

Inputs

  • 0x3092 (coolant_temp_raw): qualification check
  • 0x3093 (coolant_temp): correction factor lookup
  • 0x305A, 0x305B, 0x305E: engine state qualification
  • 0x316B (lookup_input_b): load qualification
  • 0x30F8: airflow-derived correction

Outputs

  • 0x351E (idle_duty_target): updated duty target
  • 0x3520 (idle_duty_step_counter): step state
  • 0x32CC (idle_duty_corrected): final corrected value
  • 0x021E (I/O port): ISC valve output

Calls

  • 0x633D — timer comparison
  • 0x906F — coolant correction subroutine (split from this function by flood-fill; contains the 0x6000 table lookup for coolant temp correction via table 0x4B20)

ROM References

  • 0x4B20 (idle_coolant_correction): correction factor table
  • 0x4B70–0x4B7E: controller constants (initial, min, max, thresholds)
