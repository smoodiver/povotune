Summary

The largest single function in the BD5A ROM at 954 bytes across 138 blocks. Controls the sequential twin-turbo transition for the EJ20H engine by managing the secondary turbo's wastegate duty cycle ($30A2) and associated ramp timing. The function implements a multi-state transition controller with entry/exit qualification, duty cycle ramping with rate limiting, and target tracking with asymmetric gain.

The function has six major phases:

  1. Gain calculation (0xBDFF): Calls JSR $C1B9 (turbo_transition_gain_calculator) to compute RPM-dependent and speed-dependent ramp rates for the transition.

  2. Entry qualification (0xBE02–0xBE34): Tests whether transition conditions are met:

    • Checks $305A bit 4 (stall/cranking flag) — if set, clears transition permission and skips.
    • Compares $306F (airflow_load) against $45F8 and $3071 (vehicle_speed) against $45F9 — both must exceed thresholds.
    • Tests $3029 bit 5 (adjustment map condition) — must be clear.
    • Compares $45FC (max speed threshold) against $30A2 (duty accumulator) — duty must be below threshold.
    • On qualification, loads $45FA into $30A2 (initial duty) and sets $3029 bit 5.

  3. Differential pressure capture and mode dispatch (0xBE34–0xBE50): Reads hardware port $20 bits via CLM/SEM mode switch, captures differential pressure reading $22 -> $3040, computes $3040 >> 2 -> $3750 (8-bit mirror) and SSM port $014A. Then dispatches based on $4039: if nonzero, jumps to the abort/reset path at 0xBF85.

  4. Transition state machine (0xBE50–0xBF9E): A complex multi-branch state machine controlling the secondary turbo transition:

    • Overboost detection (0xBE50–0xBE8C): Compares RPM counter $310A against thresholds $4D18/$4D1A. Sets $3028 bit 7 on high RPM. Separately checks warmup counter $3239 against $4E42 with $3103 for conditional $3028 bit 6 (warmup gate).
    • Deceleration management (0xBE8C–0xBEEF): When $3026 bit 4 is set (decel condition), manages two deceleration timers ($30D7, $30D8) with ROM delays ($4D39, $4D3B, $4D3C). Sets $3029 bit 6 and loads target into $3010. Handles timer expiry transitions.
    • Transition commit (0xBEEF–0xBF72): When $3028 bit 0 is clear (not yet committed), qualifies on coolant temp ($3092 >= $4D0C), differential pressure ($3040 in range $4D0D–$4D11), and absence of deceleration/inhibit flags. On commit: sets $3028 bit 0, clears bits 1 and 4, zeros timers $30D7/$30D8, loads initial targets from ROM ($4D13 -> $309E, $4D15 -> $310C/$30A0, $4D17 -> $3110), clears accumulators $3136/$323E/$3103, and jumps to exit.
    • Transition abort (0xBF75–0xBF9E): Loads abort delay $4D00 -> $30D9 and/or $4D01 -> $30DA. Sets duty to $8000 (50% / neutral). Jumps to exit.

  5. Duty cycle ramping (0xBFB5–0xC168): The core control loop that ramps $30A2 toward the target $3010:

    • Rate-limited approach (0xBFB8–0xBFDC): Compares current duty $30A2 against target $3010. If above, subtracts step size $4D0A (clamped to target). If below, adds step size $4D0A (clamped to target). Implements smooth rate-limited transitions.
    • Above-target path (0xBFE0–0xC108): When $3040 >= $30A0, sets $3028 bit 2. Selects base duty from $310C or $310E (depending on $3028 bit 7 / overboost flag). Subtracts the up-ramp rate $3316/$3318 (computed by 0xC1B9) from $30A2, clamped to zero.
    • Below-target path (0xC10A–0xC168): When $3040 < $30A0, clears $3028 bit 2. Adds the down-ramp rate $3317/$3319 to $30A2, clamped to 0xFFFF. Sets target ceiling from $310C + $4D24 (or $3302 for overboost mode), clamped to 0xFFFF.

  6. Target step-down and exit (0xC16B–0xC1B8): Clamps $30A2 to range [$4D1E, $4D20]. Manages a step-down timer for $310C: compares against minimum $4D22, decrements with reload period $4D17 via counter $3110. Writes final duty byte $30A3 to SSM port $0148 and diagnostic mirror $377E.
