0x7F22 — Adjustment Map Lookup

Field Value
Address 0x7F22
Size 44 bytes
Blocks 2
Type subroutine
Callers 1 call site
Confidence likely

Summary

Sets up an 8×8 bilinear interpolation map lookup using RPM (0x3062) and adj_map_input (0x3044) through ROM map 0x4CC0 with axes 0x4CB8/0x4CB0. Calls graph_lookup_3d in 8×8 mode (entry 0x6093) and stores the result to 0x3024 and I/O port 0x0112.

Inputs

  • 0x3062 (rpm): RPM axis input
  • 0x3044 (adj_map_input): load axis input

Outputs

  • 0x3024 (adj_map_result): interpolated map result
  • 0x0112 (I/O port): same result written to hardware

Calls

  • 0x6093 — graph_lookup_3d (8×8 mode): bilinear interpolation

RAM Reads

  • 0x3062 (rpm)
  • 0x3044 (adj_map_input)

RAM Writes

  • 0x3072 (map_table_ptr)
  • 0x3074 (rpm_axis_ptr)
  • 0x3076 (load_axis_ptr)
  • 0x307A (rpm_lookup_value)
  • 0x3078 (load_lookup_value)
  • 0x3079 (load_lookup_value_hi)
  • 0x3024 (adj_map_result)

ROM References

  • 0x4CC0 (adjustment_map_8x8): 8×8 data map
  • 0x4CB8 (adjustment_map_rpm_axis): RPM axis
  • 0x4CB0 (adjustment_map_load_axis): load axis
