0xC637 — Engine Stall Recovery

Field Value
Address 0xC637
Size 62 bytes (0xC637–0xC674)
Blocks 5
Type subroutine
Callers 1 — scheduler P5 (0xAAD1 at 0xABB0)
Confidence confirmed

Summary

Monitors for engine stall condition and performs a soft reset when detected. Two paths based on map_mode_control ($3051) bit 2:

Engine not running (bit 2 clear): Loads default counter values from ROM into RAM: ROM[$4D00] (= 0x05) to 0x30D9, ROM[$4D01] (= 0x28 = 40) to 0x30DA. Clears engine_mode_flags ($3028) bit 0. Returns normally.

Engine running (bit 2 set): Checks timer_c2_overflow_total ($31E4). If < 0x40 (64), returns normally (engine still running). If >= 0x40, the engine has stalled and a soft reset is performed:

  1. SEI (disable interrupts)
  2. Clear map_mode_control ($3051) bit 2 (engine no longer running)
  3. Clear scheduler_flags ($3027) bit 7
  4. Clear engine_running_flags ($31A1) bit 1
  5. Clear system_event_flags ($3021) bit 1
  6. Set engine_running_flags ($31A1) to 0x00
  7. Set reentry_counter ($30D0) to 0x00
  8. Reset stack pointer to 0x37FF
  9. JMP $66C7 (re-enter main initialization)

Inputs

  • 0x3051 (map_mode_control): bit 2 = engine running flag
  • 0x31E4 (timer_c2_overflow_total): stall detection timer

Outputs

  • 0x30D9: loaded with ROM[$4D00] (0x05) on non-running path
  • 0x30DA: loaded with ROM[$4D01] (0x28) on non-running path
  • 0x3028 (engine_mode_flags): bit 0 cleared on non-running path
  • On stall: multiple flag registers cleared, stack reset, execution jumps to 0x66C7

ROM References

  • 0x4D00: default counter value (value = 0x05)
  • 0x4D01: default counter value (value = 0x28 = 40)

RAM Reads

  • 0x3051 (map_mode_control)
  • 0x31E4 (timer_c2_overflow_total)

RAM Writes

  • 0x30D9, 0x30DA: default values loaded
  • 0x3028 (engine_mode_flags): bit 0 cleared
  • 0x3051 (map_mode_control): bit 2 cleared (stall path)
  • 0x3027 (scheduler_flags): bit 7 cleared (stall path)
  • 0x31A1 (engine_running_flags): bit 1 cleared then zeroed (stall path)
  • 0x3021 (system_event_flags): bit 1 cleared (stall path)
  • 0x30D0 (reentry_counter): zeroed (stall path)
