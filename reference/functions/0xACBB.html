0xACBB — Target A/F Ratio

Field Value
Address 0xACBB
Size 38 bytes (0xACBB–0xACE0)
Blocks 7
Type subroutine (ISR callee)
Callers 1 — irq_handler_AAD1 P4 task group (0xAAD1 at 0xABB1)
Callees 0
Confidence confirmed

Summary

Calculates the target air-fuel ratio byte and writes it to 0x32C8. Reads the engine load value from 0x313C, shifts it right by 5 bits (divides by 32) to produce a 3-bit index, and compares it against the ROM threshold at 0x4D87. If the shifted value is below the threshold, the default A/F ratio from ROM 0x4D88 is used; otherwise a fixed value of 0x20 is used. Bit 7 of the result is then set or cleared based on the af_ratio_mode flag at 0x31CC.

Disassembly

ACBB  CLM                    ; 16-bit accumulator mode
ACBC  LDA $313C              ; load engine_load (16-bit)
ACBF  LSR A                  ; shift right x5 (divide by 32)
ACC0  LSR A
ACC1  LSR A
ACC2  LSR A
ACC3  LSR A
ACC4  SEM                    ; back to 8-bit mode
ACC5  CMP $4D87              ; compare against af_ratio_threshold
ACC8  BCS $ACCF              ; if >= threshold, use fixed value
ACCA  LDA $4D88              ; load af_ratio_default
ACCD  BRA $ACD1              ; skip to mode flag check
ACCF  LDA #$20               ; fixed value 0x20 (rich target)
ACD1  LDB $31CC              ; load af_ratio_mode
ACD5  BNE $ACDB              ; if nonzero, set bit 7
ACD7  AND #$7F               ; clear bit 7
ACD9  BRA $ACDD
ACDB  ORA #$80               ; set bit 7
ACDD  STA $32C8              ; write target_af_ratio
ACE0  RTS

Logic

  1. Load and scale: Engine load (0x313C) is loaded in 16-bit mode, then shifted right 5 times to extract the upper bits as a coarse load index (0–7 range from a 0x00–0xFF byte).
  2. Threshold comparison: The scaled load is compared against ROM 0x4D87. Below threshold selects the ROM default from 0x4D88 (lean/cruise target); at or above threshold selects the fixed value 0x20 (rich/power target).
  3. Mode flag: If 0x31CC (af_ratio_mode) is nonzero, bit 7 is set in the output, signaling a different fueling mode to downstream consumers. If zero, bit 7 is cleared.
  4. Output: Final value stored to 0x32C8 (target_af_ratio).
