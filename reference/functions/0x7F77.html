0x7F77 — Warmup Enrichment

Confidence likely

Summary

Computes warmup fuel enrichment based on engine temperature and a time-based ramp. If adj_map_input exceeds a threshold, enrichment is bypassed. Otherwise, a ramp counter (0x3098) builds up during warmup conditions (gated by RPM, speed limiting, and temperature). The ramp value is multiplied by two temperature-dependent correction factors: one from coolant_temp (selected between tables 0x4320/0x4420 by mode flag), and one from coolant_temp_raw (selected between three tables 0x42F0/0x4330/0x4430 by temperature thresholds). The product is stored as the enrichment value (0x3058) and output to I/O port 0x010A. A final clamping stage tracks maximum enrichment (0x31D1).

Inputs

  • 0x3044 (adj_map_input): bypass threshold check
  • 0x305A (system_control_flags): bit 4 = speed limiting
  • 0x305C (mode_select_flags): bit 2 = enrichment table A/B select
  • 0x3062 (rpm): RPM for conditioning
  • 0x3092 (coolant_temp_raw): temperature for table selection and second lookup
  • 0x3093 (coolant_temp): first enrichment factor lookup
  • 0x3098 (warmup_ramp_counter): ramp state
  • 0x30D2 (warmup_condition_value): conditioning state

Outputs

  • 0x3058 (warmup_enrichment_16): computed enrichment
  • 0x010A (I/O port): scaled enrichment output
  • 0x3098 (warmup_ramp_counter): updated
  • 0x30D2 (warmup_condition_value): updated
  • 0x31D0 (warmup_clamp_reference): updated
  • 0x31D1 (warmup_clamp_current): updated

Calls

  • 0x6000 — graph_lookup_2d: 1D table interpolation (called twice for dual correction factors)

  • 0x404E (rpm_enrichment_threshold): RPM conditioning
  • 0x4021 (warmup_ramp_base): ramp base value
  • 0x4023 (warmup_ramp_scale): ramp scale factor

  • 0x42F0, 0x4330, 0x4430 (enrichment_cold/mid/hot): coolant_temp_raw factor tables
  • 0x4013, 0x4014 (temp_thresholds): table selection breakpoints
