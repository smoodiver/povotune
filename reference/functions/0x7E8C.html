0x7E8C — Boost Output Controller

Field Value
Address 0x7E8C
Size 60 bytes
Blocks 7
Type subroutine
Callers 1 call site
Confidence likely

Summary

Controls I/O port 0x013C output with two operating modes. In overheat-protection mode (0x305A bit 4 set): looks up coolant_temp through one of two ROM tables selected by vehicle speed vs threshold 0x4051 — at low speed, table 0x43C0 applies a coolant-temperature-based boost reduction (preventing overheating without ram air cooling); at road speed, table 0x4770 is all zeros (no reduction needed). Scales the result to 0x30B4 and clears the output port. In normal mode: passes 0x30B4/0x30B5 through processing function 0x6314 and writes the result to I/O port 0x013C.

Inputs

  • 0x305A (system_control_flags): bit 4 selects operating mode
  • 0x3071 (vehicle_speed): table selection in speed-limiting mode
  • 0x3093 (coolant_temp): lookup input in speed-limiting mode
  • 0x30B4 (boost_output_base): input in normal mode
  • 0x30B5 (boost_output_base_hi): input in normal mode

Outputs

  • 0x30B4 (boost_output_base): updated in speed-limiting mode
  • 0x013C (I/O port): actuator output (cleared in speed-limiting, driven in normal)

Calls

  • 0x6000 — graph_lookup_2d: coolant temp → table lookup (speed-limiting path)
  • 0x6314 — unknown: processes 0x30B4/0x30B5 → output value (normal path)

RAM Reads

  • 0x305A (system_control_flags)
  • 0x3071 (vehicle_speed)
  • 0x3093 (coolant_temp)
  • 0x30B4 (boost_output_base)
  • 0x30B5 (boost_output_base_hi)

RAM Writes

  • 0x300C (tracker_lower_bound): scratch
  • 0x30B4 (boost_output_base)

ROM References

  • 0x43C0 (coolant_to_boost_normal): normal speed table
  • 0x4770 (coolant_to_boost_highspeed): high-speed table
  • 0x4051 (vehicle_speed_boost_threshold): table selection threshold
