0x8088 — Overrun Detector

Summary

Detects engine overrun (deceleration) conditions and sets/clears bit 4 of fuel_cut_flags (0x3026). Checks a cascade of conditions: speed control active (0x302A bit 3), coolant temp above minimum, vehicle speed >= 4, RPM above threshold, and finally that engine_load is below an RPM-indexed table value at 0x4D4F. If all conditions pass, the engine is in overrun and fuel cut can activate.

Inputs

  • 0x302A (speed_control_flags): bit 3 = speed control active (auto-set)
  • 0x3092 (coolant_temp_raw): temperature check
  • 0x3071 (vehicle_speed): minimum speed check
  • 0x3062 (rpm): RPM threshold and table index
  • 0x305A (system_control_flags): bit 4 = speed limiting (auto-set)
  • 0x313C (engine_load): compared against RPM-indexed limit

Outputs

  • 0x3026 (fuel_cut_flags): bit 4 set/cleared

RAM Reads

  • 0x302A (speed_control_flags)

  • 0x3071 (vehicle_speed)

  • 0x305A (system_control_flags)
  • 0x313C (engine_load)

RAM Writes

ROM References

  • 0x4D36 (overrun_coolant_threshold): minimum temp for fuel cut
  • 0x4D37 (overrun_rpm_threshold): minimum RPM for fuel cut
  • 0x4D4F (overrun_load_limit_table): RPM-indexed max load for overrun
