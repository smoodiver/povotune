0x9852 — Knock Learning State Restore / Init

Field Value
Address 0x9852
Size ~79 bytes (0x9852–0x98A0)
Blocks 6
Type subroutine
Callers 1 call site (reset_handler 0x6431 at 0x66B4)
Confidence confirmed

Summary

Restores or initializes knock sensor learning state during ECU reset, depending on whether the EEPROM learning system was previously initialized (0x3030 bit 6). This is a companion to sub_97CF and runs immediately after it.

Cold-start path (bit 6 clear — fresh EEPROM): Zeroes the knock learned values table from 0x3630 to 0x366F (32 words / 64 bytes). Loads knock multiplier initial values from ROM (0x4D97 to 0x3503, 0x4D98 to 0x3502). Copies ignition zone control (0x3033) to ign_zone_status_mirror (0x3500), sets 0x3034 bit 3, then copies 0x3034 to ign_control_status_mirror (0x3501).

Warm-start path (bit 6 set — EEPROM valid): Restores state from EEPROM-backed mirrors: copies 0x3500 through AND mask 0x48 back to 0x3033 and 0x3500 (preserving only bits 6 and 3), copies 0x3501 through AND 0x04 then OR 0x08 back to 0x3034 and 0x3501 (preserving bit 2, forcing bit 3).

Common tail (0x989C): Sets 0x3036 bits 0 and 4 (ign_zone_flags = 0x11).

Inputs

  • 0x3030 (throttle_state_flags): bit 6 — EEPROM initialized flag
  • ROM 0x4D97: knock multiplier initial value 1
  • ROM 0x4D98: knock multiplier initial value 2
  • 0x3033 (ign_zone_control): current ignition zone (cold-start)
  • 0x3500 (ign_zone_status_mirror): saved ignition zone (warm-start)
  • 0x3501 (ign_control_status_mirror): saved ignition control (warm-start)

Outputs

  • 0x3630–0x366F (knock_learned_values): zeroed on cold-start
  • 0x3503 (knock_multiplier): loaded from ROM 0x4D97 (cold-start)
  • 0x3502: loaded from ROM 0x4D98 (cold-start)
  • 0x3500 (ign_zone_status_mirror): initialized or restored
  • 0x3501 (ign_control_status_mirror): initialized or restored
  • 0x3033 (ign_zone_control): restored from mirror (warm-start)
  • 0x3034 (ign_control_flags): initialized or restored with bit 3 set
  • 0x3036 (ign_zone_flags): bits 0,4 set (0x11)

ROM References

  • 0x4D97: knock multiplier initial value 1 -> 0x3503
  • 0x4D98: knock multiplier initial value 2 -> 0x3502
