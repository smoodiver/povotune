Summary

Sequential dispatcher that unconditionally calls 25 diagnostic test subroutines via back-to-back JSR instructions. No conditional selection — all 25 tests run every invocation, with each test self-gating based on preconditions. The first callee (0x6B60) is a precondition gate that checks engine-running state, warmup progress (0x316B >= 0x64), and sensor readiness before enabling remaining tests via 0x3031 bit 7.

Inputs

  • 0x3051 bit 4: engine running (master gate)
  • 0x316B: warmup counter (>= 0x64 required)
  • 0x306E: knock sensor (>= 0x08 required)
  • 0x30B6: engine run timer (>= 0x000A required)

  • 0x332E, 0x332F, 0x3330: diagnostic fault flags
