Summary

Multi-condition qualification gate for boost closed-loop operation. Checks three conditions in sequence: (1) mode flag 0x305E bit 3 must be set, (2) RAM value 0x3178 must be >= ROM threshold 0x4602, (3) coolant_temp_raw must be >= ROM threshold 0x4BF6, (4) secondary_condition_value must be >= ROM threshold 0x4604. If all pass, sets 0x305F bit 5; if any fail, clears it.

Disassembly

8E38  BBC $305E,#$08,$8E5A  ; if bit 3 clear, fail
8E3D  LDX $3178              ; load timer/counter value
8E40  CPX $4602              ; compare vs threshold
8E43  BCC $8E5A              ; if below, fail
8E45  LDA $3092              ; load coolant_temp_raw
8E48  CMP $4BF6              ; compare vs temp threshold
8E4B  BCC $8E5A              ; if below, fail
8E4D  LDX $30B6              ; load secondary_condition_value
8E50  CPX $4604              ; compare vs threshold
8E53  BCC $8E5A              ; if below, fail
8E55  SEB $305F,#$20         ; set bit 5 = qualified
8E59  RTS
8E5A  CLB $305F,#$20         ; clear bit 5 = not qualified
8E5E  RTS

Inputs

  • 0x305E bit 3: mode flag (must be set)
  • 0x3178: timer or counter value
  • 0x3092 (coolant_temp_raw): engine temperature
  • 0x30B6 (secondary_condition_value): load/pressure input

Outputs

  • 0x305F bit 5: boost qualification flag (set if all conditions met, cleared otherwise)

RAM Reads

RAM Writes

  • 0x305F (bit 5)

  • 0x4604 -- secondary condition threshold
