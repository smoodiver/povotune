0xCBA1 — Vehicle Speed Processor

Field Value
Address 0xCBA1
Size 254 bytes (0xCBA1–0xCC9E)
Blocks 38
Type subroutine
Callers 1 call site (main loop region at 0xC464)
Confidence confirmed

Summary

Processes differential pressure (0x3040) into vehicle speed high/low bounds (0x3114/0x3116) with a controlled convergence rate, then computes a speed range value (0x310A) and two scaled speed outputs (0x310E, 0x3302) with clamping. The function has two major phases. Phase 1 (0xCBA1–0xCBC8): when map_mode_control bit 2 is set, increments a warmup counter (0x3112) until it reaches ROM[0x4D26]; before that, initializes speed bounds from diff_pressure (clamped to 0x0100) and reloads a prescaler (0x3113) from ROM[0x4D27]. Phase 2 (0xCBC9–0xCC9E): once warmed up and gated by engine_mode_flags (0x3028) bits 3 and 6, decrements the prescaler; when it expires, narrows the high bound (0x3114) by 1 and widens the low bound (0x3116) by 1 (both 16-bit). Then updates both bounds from current diff_pressure. Computes speed range as high - low (clamped >= 0). A secondary prescaler (0x3118) reloaded from ROM[0x4D28] gates the final computation: two independent interpolations using ROM scale factors (0x4E3F, 0x4E40) applied to the speed range, added to the low bound, and clamped to ROM min/max (0x4D2A/0x4D2C). The results are stored to 0x310E and 0x3302 with half-values mirrored to IO ports 0x021A and 0x021C.

Inputs

  • 0x3051 (map_mode_control): bit 2 gates warmup counter
  • 0x3040 (diff_pressure): primary speed input — clamped to 0x0100 max
  • 0x3028 (engine_mode_flags): bits 3 and 6 gate convergence logic
  • 0x3112 (speed_warmup_counter): warmup state tracker
  • 0x3113 (speed_prescaler): convergence rate divider
  • 0x3114 (vehicle_speed_raw_hi): upper speed bound
  • 0x3116 (vehicle_speed_raw_lo): lower speed bound
  • 0x3118 (speed_output_prescaler): output computation rate divider
  • 0x310A (speed_range): high - low difference

Outputs

  • 0x3112 (speed_warmup_counter): incremented during warmup
  • 0x3113 (speed_prescaler): countdown timer
  • 0x3114 (vehicle_speed_raw_hi): updated upper speed bound
  • 0x3116 (vehicle_speed_raw_lo): updated lower speed bound
  • 0x310A (speed_range): computed as 0x3114 - 0x3116, clamped >= 0
  • 0x3118 (speed_output_prescaler): countdown timer
  • 0x310E (speed_output_primary): first scaled speed output
  • 0x3302 (speed_output_secondary): second scaled speed output
  • 0x021A (IO port): 0x310E >> 1
  • 0x021C (IO port): 0x3302 >> 1
