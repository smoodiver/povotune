0xD86E — Boost Target Pressure Adjustment

Field Value
Address 0xD86E
Size 66 bytes (0xD86E–0xD8AF)
Blocks 12
Type subroutine
Callers 1 — multi_rate_timer_dispatcher (0xC458) 10-tick path at 0xC47D
Confidence confirmed

Summary

Adjusts the 16-bit boost target pressure accumulator ($30F8) by adding or subtracting a correction value derived from $30E0. Called every 10 P5 scheduler ticks. The direction of adjustment is controlled by $305E bit 0: when set, the correction is subtracted (reducing boost target); when clear, it is added (increasing boost target). Result is clamped to 0x0000–0xFFFF.

Entry gate: If $35A4 bit 0 is set, returns immediately (system inhibit). Then checks $3051 bit 2: if clear, loads a constant 0x4000 into $30F8 and returns (reset to midpoint). If $305E bit 3 is clear, also returns early (precondition not met).

Correction path: Loads $30E0 (correction magnitude), masks to 8-bit (AND #$FF), stores to workspace $3008. Loads current $30F8, then branches on $305E bit 0:

  • Bit 0 set (subtract): $30F8 = $30F8 - $3008, clamped at 0x0000
  • Bit 0 clear (add): $30F8 = $30F8 + $3008, clamped at 0xFFFF

Disassembly

D86E  BBS $35A4,#$01,$D8AF  ; if system inhibit, return
D873  CLM                    ; 16-bit mode
D874  BBS $3051,#$04,$D87F  ; if map_mode bit 2 set, continue

D87D  BRA $D8AB              ; -> store and exit
D87F  BBC $305E,#$08,$D8AE  ; if engine_state bit 3 clear, exit (SEM+RTS)
D885  LDA $30E0              ; load correction magnitude
D888  AND #$FF               ; mask to 8-bit
D88B  STA $3008              ; store to workspace
D88E  LDA $30F8              ; load current boost target
D891  BBC $305E,#$01,$D8A2  ; if bit 0 clear, -> add path
D897  SEC                    ; subtract path
D898  SBC $3008              ; target = target - correction
D89B  BCS $D8AB              ; if no underflow, store
D89D  LDA #$00               ; clamp at zero
D8A0  BRA $D8AB
D8A2  CLC                    ; add path
D8A3  ADC $3008              ; target = target + correction
D8A6  BCC $D8AB              ; if no overflow, store
D8A8  LDA #$FFFF             ; clamp at max
D8AB  STA $30F8              ; store updated boost target
D8AE  SEM                    ; 8-bit mode
D8AF  RTS
