0x8E19 — Saturating Double and Lookup

Summary

Scales the throttle position value by doubling it twice (x4), saturating to 0xFF on overflow, then performs a 1D table interpolation. Used in the boost controller's passive path to convert throttle position into a duty floor value via a ROM table pointed to by X on entry.

Disassembly

8E19  LDA $3044          ; load throttle_position
8E1C  ASL A              ; shift left (x2)
8E1D  BCS $8E22          ; if carry, saturate
8E1F  ASL A              ; shift left (x4)
8E20  BCC $8E24          ; if no carry, continue
8E22  LDA #$FF           ; saturate to 0xFF
8E24  JSR $6000          ; graph_lookup_2d (X = table pointer)
8E27  RTS

Inputs

  • 0x3044 (throttle_position): raw throttle position value
  • X register: ROM table pointer (set by caller, e.g. 0x4B40)

Outputs

  • A register: interpolated table result
