0xBB8D — Injection Pulse Width & Cylinder Distribution

Field Value
Address 0xBB8D
Size 395 bytes (0xBB8D–0xBD17)
Blocks 57
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 at 0xAB68)
Confidence confirmed

Summary

Takes the base fuel pulse width from the enrichment calculator and applies O2 feedback correction, warmup enrichment, and cranking compensation to produce per-cylinder injection base pulse values. The function has four major phases:

  1. Load scaling (0xBB8D–0xBBAA): Multiplies the duty accumulator ($30A2) by engine load ($313C), producing a 32-bit result. The upper 16 bits (shifted right 4) are written to SSM diagnostic port $014C.

  2. O2 feedback application (0xBBAF–0xBBEC): Takes the load-scaled product and multiplies by the O2 feedback integrator ($32F2). If warmup enrichment base ($30B8) is nonzero, adds the boost pressure reading ($3094) with saturation, then multiplies again by the O2-feedback-scaled result. If the product exceeds 0x2000 in the high word, saturates to 0xFFFF; otherwise left-shifts by 3 via JSR $6304. The result is stored to $3128 as the enrichment-adjusted pulse width.

  3. Cranking/minimum pulse width (0xBBEC–0xBC83): Adds the duty complement ($3056) to the enrichment result with saturation, clamps to a minimum of 0x0019 (25 decimal), and stores to $300A. Computes $300A * 4 and writes the high byte to SSM diagnostic port $0108. Compares the warmup enrichment value ($3058) against either the computed pulse width or the ROM cranking value ($4410), selecting the larger, and stores to $30BE (cranking pulse width). Also computes a doubled version of the enrichment result + duty complement and compares against the doubled cranking value to set/clear $3022 bit 3 (hysteresis flag), storing the winner to $30BC. Writes $30BE * 4 high byte to SSM port $010E.

  4. Cylinder distribution (0xBC83–0xBD17): Distributes the injection pulse width to four per-cylinder base pulse registers ($3170–$3176) based on the closed-loop readiness level ($3211), limiter flags ($314E bits 6/7), and speed/map mode flags ($302A bit 4, $323F bit 0). The distribution logic selects between three strategies:

    • Normal: all 4 cylinders get the enrichment-compensated value (from $30BC)
    • Partial cut: cylinders 3-4 get $30BC, cylinders 1-2 get the duty complement ($3056)
    • Enriched cut: all 4 cylinders get the enriched value from JSR $BD18 (enrichment * 2 + duty complement)

    The final $30BC value is right-shifted 4 bits and written to SSM port $0104 and $3742. The $3170 value is right-shifted 6 bits and conditionally written to $3746 (gated by $3051 bit 2).

Inputs

  • $30A2: 16-bit duty accumulator
  • $313C (engine_load): engine load value
  • $32F2: O2 feedback integrator (from 0xB811)
  • $30B8: secondary fuel value (from 0xBAA3)
  • $3094 (throttle_correction_output): boost pressure / throttle correction
  • $3056 (duty_complement_16): 16-bit duty complement
  • $3058 (warmup_enrichment_16): 16-bit warmup enrichment
  • $3128: enrichment-adjusted pulse width (computed internally)
  • $3022 (isr_sync_flags): bit 3 hysteresis flag
  • $3051 (map_mode_control): bit 2 gates $3746 output, bit 3 selects cranking comparison
  • $3211 (closed_loop_readiness_level): determines cylinder distribution strategy
  • $314E (limiter_flags): bits 6/7 force partial-cut distribution
  • $302A (speed_control_flags): bit 3 inhibits, bit 4 selects distribution path
  • $323F: bit 0 forces partial-cut distribution

Outputs

  • $3128: enrichment-adjusted injection pulse width
  • $30BC: final injection base pulse width (selected via hysteresis)
  • $30BE (cranking_pulse_width): cranking/minimum pulse width
  • $3170 (inj_base_pulse_ch1): injection base pulse, cylinder 1
  • $3172 (inj_base_pulse_ch2): injection base pulse, cylinder 2
  • $3174 (inj_base_pulse_ch3): injection base pulse, cylinder 3
  • $3176 (inj_base_pulse_ch4): injection base pulse, cylinder 4
  • $3742: pulse width diagnostic (shifted)
  • $3746: injection diagnostic (shifted, conditional)
  • $0104: SSM diagnostic — pulse width >> 4
  • $0108: SSM diagnostic — minimum pulse * 4 high byte
  • $010E: SSM diagnostic — cranking pulse * 4 high byte
  • $014C: SSM diagnostic — load * duty >> 4
