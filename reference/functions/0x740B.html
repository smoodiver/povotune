0x740B — Turbo Solenoid B Fault Check

Summary

Dual-direction fault check for boost solenoid channel B. Identical structure to 0x73AF but checks different gate/feedback bits and writes to different fault positions. Reads 0x35A5 bit 3 (port_2000_shadow) for command state and 0x3806 bit 4 (ext_io_reg_1) for hardware feedback. Uses counter pair at 0x3336/0x3337 with threshold 0x28 (40 iterations). Faults reported in 0x333A bits 4+5; summary promoted to 0x3331 bit 2.

Prerequisites

  • 0x316B (lookup_input_b) >= 0x8A — engine must be warm enough
  • If below threshold: counter cleared to 0x0000 and early return

Per-Direction Logic

X = $3336 (counter base)
SEI

if 0x35A5 bit 3 set (solenoid B commanded on):
    if 0x3806 bit 4 set (solenoid B feedback active):
        CLI
        JSR $633D (increment counter at X)
        if count >= $28: SEB $333A bit 4
    else (feedback inactive while commanded = stuck-low):
        CLB $333A bit 4
        BRA clear_and_exit
else (solenoid B not commanded on):
    INX (X = $3337)
    if 0x3806 bit 4 NOT set (feedback inactive — expected):
        ; skip to exit
    else (feedback active while not commanded = stuck-high):
        CLI
        JSR $633D (increment counter at X+1)
        if count >= $28: SEB $333A bit 5
        BRA exit
    CLB $333A bit 5

clear_and_exit:
    CLI
    LDM $00,X,#$00 (clear counter)

exit:
    if $333A bits 4+5 both set:
        SEB $3331 bit 2
    else:
        CLB $3331 bit 2
    RTS

Inputs

  • 0x316B (lookup_input_b): warmup threshold >= 0x8A
  • 0x35A5 bit 3 (port_2000_shadow): solenoid B command state
  • 0x3806 bit 4 (ext_io_reg_1): solenoid B hardware feedback

Outputs

  • 0x3336 (boost_sol_b_counter): 16-bit debounce counter pair
  • 0x333A bit 4: solenoid B stuck-high fault
  • 0x333A bit 5: solenoid B stuck-low fault
  • 0x3331 bit 2: solenoid B summary fault (both directions confirmed)

Calls

  • 0x633D — saturating increment (2 calls)

None (threshold 0x28 is immediate).
