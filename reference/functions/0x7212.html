Summary

Checks the coolant fan (radiator fan) relay circuit for faults. First validates that airflow load (0x306F) is within a ROM-defined window [ROM 0x4878..ROM 0x4879) and vehicle speed (0x3071) is above ROM threshold 0x487A. If preconditions fail, clears both debounce counters (0x31D8, 0x31D9) and exits. When conditions are met, checks 0x305A bit 5 to select between two debounce paths: one path increments 0x31D8 (fan relay commanded ON side) while clearing 0x31D9, the other increments 0x31D9 (fan relay commanded OFF side) while clearing 0x31D8. When either counter reaches the ROM threshold at 0x487B, the corresponding fault bit is set or cleared in 0x332F bit 7.

Inputs

  • 0x306F (airflow_load): must be within [ROM 0x4878, ROM 0x4879)
  • 0x3071 (vehicle_speed): must be > ROM 0x487A
  • 0x305A (system_control_flags): bit 5 selects fan relay state path

Outputs

  • 0x31D8: debounce counter (fan-ON path)
  • 0x31D9: debounce counter (fan-OFF path)
  • 0x332F (temp_override_flags): bit 7 set/cleared

Calls

  • 0x633D — saturating_increment (debounce counter)
