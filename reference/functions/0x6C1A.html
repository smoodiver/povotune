0x6C1A — Coolant Sensor Pre-Check

Field Value
Address 0x6C1A
Size 17 bytes
Blocks 4
Type subroutine
Callers 1 (health_check_dispatcher 0x6B14)
Confidence confirmed

Summary

Simple pre-check that sets a warmup qualification flag once the coolant temperature sensor has reported sufficient warmup time. If the engine is running (0x3051 bit 4 set) and the timing confidence counter (0x31E9) has reached at least 0x24 (36), sets 0x3330 bit 2 to indicate the coolant sensor warmup condition has been met. This flag is used by downstream health checks (e.g. 0x6C2B) as a gating precondition.

Inputs

  • 0x3051 bit 4: engine running flag
  • 0x31E9: timing confidence counter

Outputs

  • 0x3330 bit 2: coolant sensor warmup qualified flag

RAM Reads

  • 0x3051, 0x31E9

RAM Writes

  • 0x3330
