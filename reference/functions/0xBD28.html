0xBD28 — Fuel Pulse Diagnostic Flag

Field Value
Address 0xBD28
Size 41 bytes (0xBD28–0xBD50)
Blocks 9
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 at 0xAB6B)
Confidence confirmed

Summary

Sets or clears bit 5 of the diagnostic auxiliary fault register ($333B) based on whether fuel injection is active and conditions are normal. If the fuel pulse accumulator ($3054) is zero, or any of six inhibit conditions are true, sets $333B bit 5 (indicating fuel cut/fault). Otherwise clears the bit (fuel normal).

The inhibit conditions checked are:

  • $314E bit 6 (hard rev limiter active)
  • $314E bit 7 (soft rev limiter active)
  • $302A bit 4 (speed control fuel cut)
  • $323F bit 0 (map selection fuel cut)
  • $3211 != 0 (closed-loop readiness not at level 0)

Inputs

  • $3054: fuel pulse width accumulator
  • $314E (limiter_flags): bits 6/7 — rev limiter states
  • $302A (speed_control_flags): bit 4 — speed control fuel cut
  • $323F: bit 0 — map selection fuel cut flag
  • $3211 (closed_loop_readiness_level): nonzero inhibits

Outputs

  • $333B (diag_aux_fault_bits): bit 5 set = fuel cut active, bit 5 clear = fuel normal
