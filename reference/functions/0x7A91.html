0x7A91 — Diagnostic Output Builder

Confidence confirmed

Summary

Builds a diagnostic output packet for the Subaru Select Monitor (SSM). Reads throttle position, vehicle speed, and numerous I/O port and status register bits, packs them into RAM registers (0x3780–0x3786) with I/O mirrors (0x020A–0x0214). Purely data marshalling — no table lookups or subroutine calls.

Inputs

  • 0x3360 (throttle_position_raw): TPS reading
  • 0x3114, 0x3116: vehicle speed counter values
  • 0x3028: switch state (bit 2 = idle contact)
  • 0x2000, 0x2200: I/O input ports
  • 0x3275 (port_control_mirror): output port bits
  • 0x35A5 (port_2000_shadow): shadow register bits
  • 0x31C1: sensor activity flags

Outputs

  • 0x3780 (throttle_position_scaled) → I/O 0x020A
  • 0x377A, 0x377B (speed_hi/lo_scaled) → I/O 0x0206, 0x0208
  • 0x3783–0x3786 (diag_switches_1–4) → I/O 0x020E–0x0214
  • 0x378F bit 0 (idle_contact_flag)
