0x6D15 — Knock Sensor Debounce Check

Field Value
Address 0x6D15
Size 121 bytes (0x6D15–0x6D8D)
Blocks 23
Type subroutine
Callers 1 call site (health_check_dispatcher 0x6B14, slot 7)
Confidence confirmed

Summary

Two-phase knock/detonation sensor health check. Phase 1 is a 16-bit saturating counter that tracks consecutive cycles where knock conditions are present. Phase 2 decides whether to set or clear the knock fault flag in 0x332F bit 5, factoring in diagnostic mode (0x3510) and a secondary counter at 0x318C.

Phase 1 — Knock condition counter (0x3186): Gated by multiple preconditions — exits early (resetting counter to 0) if any fail:

  • 0x3028 bit 3 must be clear (BBS -> bail)
  • 0x332F bits 0+4 must both be clear (intake air + other fault absent)
  • 0x3330 bits 0+2 must both be clear (VSS + coolant faults absent)
  • Coolant temp (0x3092) >= 0x6E
  • Knock counter (0x306E) >= 0x28
  • Vehicle speed (0x3071) >= 0x0A
  • Throttle position (0x3745) >= 0x27
  • 0x3028 bit 0 must be clear

If all pass, counter 0x3186 is incremented (16-bit saturating: INX/BNE/DEX pattern). When it reaches 0x0186 (390), pending fault 0x31CF bit 2 is set.

Phase 2 — Fault confirmation: If 0x31CF bit 2 is set and 0x3028 bit 0 is set, the pending fault is cleared and confirmed fault 0x3030 bit 3 is set. Otherwise, if diagnostic mode (0x3510 != 0) and counter 0x318C >= 0x030D, fault flag 0x332F bit 5 is set. If 0x31CF bit 2 is not set, 0x332F bit 5 is cleared.

Inputs

  • 0x3028: engine mode flags (bits 0, 3)
  • 0x332F: temp_override_flags (bits 0, 4)
  • 0x3330: diag_fault_flags_B (bits 0, 2)
  • 0x3092: coolant_temp_raw
  • 0x306E: knock sensor counter
  • 0x3071: vehicle speed
  • 0x3745: throttle position
  • 0x3186: 16-bit knock debounce counter
  • 0x31CF: diag_pending_faults
  • 0x3510: diagnostic mode control
  • 0x318C: diagnostic timer (16-bit)

Outputs

  • 0x3186: knock debounce counter (incremented or reset to 0)
  • 0x31CF bit 2: knock pending fault
  • 0x3030 bit 3: knock confirmed fault
  • 0x332F bit 5: knock fault flag for diagnostic output
