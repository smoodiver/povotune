0xC37B — System Status Update

Field Value
Address 0xC37B
Size 197 bytes (0xC37B–0xC43F)
Blocks 27
Type subroutine
Callers 2 — reset_handler (0x6431 at 0x66AE), scheduler ISR (0xAAD1 at 0xAB9B)
Confidence confirmed

Summary

Consolidates hardware port reads, speed processing, and system flag updates into a single status refresh routine. Called both during reset initialization and periodically from the scheduler ISR. The function has five phases:

  1. Flag snapshot (0xC37B–0xC37E): Saves current $305A (system_control_flags) to $3138 for edge detection later.

  2. Port read and mode flags (0xC381–0xC3AB): Reads hardware port $2200, masks and XORs to extract active-low signals, stores to $305C (mode_select_flags). Then reads port $2000, XORs bit 5, and conditionally modifies based on $305C bit 5 state:

    • When $305C bit 5 is set: XORs bit 0, then checks $2000 bit 2. If set, loads $400D (ROM) into $328C (debounce/filter). If clear, checks if $328C is non-zero and conditionally sets bit 0.
    • This implements a mode-switch debounce using ROM-configured initial values.

  3. Idle contact speed check (0xC3AD–0xC3DC): When $332F bit 4 is not set, computes intake air temperature compensation: $303C - $32AA (clamped to 0), then compares against ROM thresholds $42E2 or $42E4 (selected by $305A bit 4). Sets or clears bit 4 of the accumulator (idle contact / closed throttle flag). Stores to $3740 (diagnostic mirror).

  4. Engine running condition (0xC3DF–0xC401): Checks $3330 bit 1 and $3051 bit 2 for special conditions. If not in special mode, compares $306F (airflow_load) against $4CAE and $316B (lookup_input_b) against $4CAF. Clears bit 2 if either is below threshold (engine not sufficiently loaded).

  5. Speed and output update (0xC401–0xC43F): Writes the assembled flags to $305A and SSM port $0100. Copies $309B (speed_value) to $3071 (vehicle_speed) and SSM port $01FE. Copies $3167 (speed_snapshot) to diagnostic mirror $3776. Computes $3042 >> 2 and stores to $316B (lookup_input_b), SSM port $01FC, and $3775. Finally, detects edges on bit 4 of $305A (stall/cranking flag): on a falling edge (was set, now clear), clears $3300 and $3301 to zero.

Inputs

  • $2200 — hardware port (mode switches)
  • $2000 — hardware port (sensor/switch inputs)
  • $305A — system_control_flags (previous state)
  • $306F — airflow_load
  • $309B — speed_value
  • $303C — intake air temperature ADC
  • $3042 — throttle position ADC
  • $3167 — speed_snapshot
  • $332F — temp_override_flags (bit 4 gates idle check)
  • $3330 — control flags
  • $3051 — map_mode_control
  • $316B — lookup_input_b
  • $32AA — intake air temp offset
  • $328C — mode switch debounce

Outputs

  • $305A — system_control_flags (updated)
  • $305C — mode_select_flags (from port $2200)
  • $3071 — vehicle_speed (from $309B)
  • $316B — lookup_input_b ($3042 >> 2)
  • $3138 — previous system_control_flags snapshot
  • $3300 — cleared on stall-to-run transition
  • $3301 — cleared on stall-to-run transition
  • $328C — mode switch debounce value
  • $3740 — diagnostic mirror of assembled flags
  • $3775 — diagnostic mirror of lookup_input_b
  • $3776 — diagnostic mirror of speed_snapshot
  • $0100 — SSM port: system_control_flags
  • $01FC — SSM port: lookup_input_b
  • $01FE — SSM port: vehicle_speed

Calls

None (leaf function).
