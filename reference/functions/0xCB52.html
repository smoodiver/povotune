0xCB52 — Three-Map Selection

Field Value
Address 0xCB52
Size 79 bytes
Blocks 7
Type subroutine
Callers 1 call site
Confidence confirmed

Summary

Selects one of three 16×16 maps (0x5200, 0x5300, or 0x5400) based on knock_multiplier (0x3503) and map_control_register (0x3291) bit 3, then performs a bilinear interpolation using RPM and engine_load through the selected map. Stores the scaled result to 0x3184.

Inputs

  • 0x3503 (knock_multiplier): selects map tier (above/below threshold 0x4045)
  • 0x3291 (map_control_register): bit 3 selects between maps 1 and 2
  • 0x3062 (rpm): RPM axis input
  • 0x313C (engine_load): load axis input

Outputs

  • 0x3184 (three_map_result): scaled interpolation result

Calls

  • 0x6080 — graph_lookup_3d: bilinear interpolation

RAM Reads

  • 0x3503 (knock_multiplier)
  • 0x3291 (map_control_register)
  • 0x3062 (rpm)
  • 0x313C (engine_load)

RAM Writes

  • 0x3072 (map_table_ptr)
  • 0x3074 (rpm_axis_ptr)
  • 0x3076 (load_axis_ptr)
  • 0x307A (rpm_lookup_value)
  • 0x3078 (load_lookup_value)
  • 0x300C (tracker_lower_bound): scratch
  • 0x3184 (three_map_result)

ROM References

  • 0x4045 (three_map_threshold): selection threshold
  • 0x5200 (three_map_1): selected when knock_multiplier >= threshold, bit 3 clear
  • 0x5300 (three_map_2): selected when knock_multiplier >= threshold, bit 3 set
  • 0x5400 (three_map_3): selected when knock_multiplier < threshold
  • 0x4100 (rpm_axis_primary): shared RPM axis
  • 0x4110 (engine_load_axis_primary): shared load axis
