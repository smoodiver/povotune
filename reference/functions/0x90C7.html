Summary

Name corrected from "EGR Temperature Control" — the BD5A EJ20H twin turbo has no EGR system. The cross-reference SVX analysis labeled dispatcher position 17 as "Test EGR Temperature" and that name propagated here incorrectly. On the BD5A, port 0x2200 bits 0 and 3 control sequential turbo solenoid valves, not EGR.

Solenoid valve controller. Uses three hysteresis comparators on speed (0x3071) and load (0x3092) to evaluate operating conditions. In diagnostic mode, cycles through four valve output states for testing. In normal mode, enables valve when speed/load/temperature conditions are met, then selects one of four duty cycle levels based on temperature input (0x3235) against three thresholds. Controls bits 1 and 8 of port_control_mirror (0x3275), flushed to I/O port 0x2200 via 0x9214.

Inputs

  • 0x3071 (vehicle_speed): speed hysteresis
  • 0x3092 (coolant_temp_raw): load/temp hysteresis
  • 0x3235 (egr_temp_value): duty cycle selection
  • 0x305A (system_control_flags): operating mode gates
  • 0x35A4 (diag_self_test_status): diagnostic mode

Outputs

  • 0x31DF (egr_condition_flags): hysteresis state
  • 0x325E (egr_output_flags): enable/active state
  • 0x3275 (port_control_mirror): bits 1,8 updated
  • I/O port 0x2200: via 0x9214

Calls

  • 0x9214 — port_2200_write: flushes port_control_mirror to hardware
