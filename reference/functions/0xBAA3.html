Callers 1 — scheduler P4 (0xAAD1 at 0xAB65)

Summary

Computes the base fuel injection pulse width from RPM correction, boost compensation, and enrichment factors, then stores the result and produces SSM diagnostic outputs. The function has three main phases:

  1. Precondition gate (0xBAA3–0xBADE): Checks multiple inhibit conditions. If $305A bit 3 is set, a debounce counter at $3225 is cleared and execution continues; otherwise the counter is incremented via JSR $633D and must reach 2 before proceeding. Six additional bit tests ($302A bits 3/5/7, $328B bits 3/4, $314E bit 3) any of which being set forces the output to zero via JMP $BB61.

  2. Base enrichment calculation (0xBADF–0xBB62): Loads the RPM-based correction ($309C) and adds the fuel value shifted ($3126), optionally adding boost output base ($30B4) if $305A bit 4 is clear. If $305C bit 2 is set, performs a coolant temperature table lookup ($3093 vs ROM table $4760 via JSR $6000) and scales the result against the working accumulator. The running sum is further adjusted by adding $3120 (ramp accumulator), subtracting $3320 (IO output value), adding $3122 (fuel value shifted), and clamping against the maximum at ROM $4018. If $305A bit 2 is set, applies a final scaling by ROM $401F. The result is stored to $3054 (the main fuel pulse width accumulator).

  3. Diagnostic output and secondary calculations (0xBB62–0xBB8C): Stores the clamped fuel value to $3054, writes the high byte to SSM port $0138. Then scales by $30A5 (an 8-bit factor), producing a secondary fuel value stored to $30B8, with its high byte written to SSM port $013A.

Inputs

  • $305A (system_control_flags): bit 3 bypasses debounce, bit 4 gates boost addition, bit 2 enables scaling
  • $3225: fuel enrichment debounce counter
  • $302A (speed_control_flags): bits 3/5/7 inhibit enrichment
  • $328B (turbo_ctrl_state): bits 3/4 inhibit enrichment
  • $314E (limiter_flags): bit 3 inhibits enrichment
  • $309C (temp_rpm_correction_scaled): RPM-based correction
  • $3126: fuel correction addend
  • $30B4 (boost_output_base): boost compensation
  • $305C (mode_select_flags): bit 2 enables coolant correction
  • $3093 (coolant_temp): coolant temperature for table lookup
  • $3120 (ramp_accumulator): accumulated ramp value
  • $3320 (io_output_value): subtracted from enrichment sum
  • $3122 (fuel_value_shifted): fuel correction addend
  • $30A5: secondary scaling factor

Outputs

  • $3054: fuel pulse width accumulator (16-bit)
  • $30B8: secondary fuel value (scaled by $30A5)
  • $0138: SSM diagnostic — fuel pulse high byte
  • $013A: SSM diagnostic — secondary fuel high byte
