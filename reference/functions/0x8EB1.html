0x8EB1 — Vehicle Speed Hysteresis Check

Field Value
Address 0x8EB1

Blocks 7
Type subroutine
Callers 1 call site (boost_control_main at 0x8B14)
Confidence confirmed

Summary

Speed hysteresis comparator for boost control mode selection. Compares vehicle_speed (0x3071) against a threshold with hysteresis: the threshold is a base value from ROM 0x4A4E, optionally increased by ROM 0x4A52 when the "already above" flag (0x3026 bit 7) is set. If speed >= threshold, sets the flag (entering high-speed mode); if speed < threshold, clears it (returning to normal mode). This prevents oscillation at the speed boundary.

Disassembly

8EB1  LDA $4A4E              ; load base speed threshold
8EB4  BBS $3026,#$80,$8EC1   ; if already above, skip hysteresis add
8EB9  CLC
8EBA  ADC $4A52              ; add hysteresis offset (raising entry threshold)
8EBD  BCC $8EC1              ; if no overflow, continue
8EBF  LDA #$FF               ; saturate to 0xFF
8EC1  CMP $3071              ; compare threshold vs vehicle_speed
8EC4  BEQ $8ECD              ; if equal, set flag
8EC6  BCC $8ECD              ; if speed > threshold, set flag
8EC8  CLB $3026,#$80          ; speed below: clear high-speed flag
8ECC  RTS
8ECD  SEB $3026,#$80          ; speed at/above: set high-speed flag
8ED1  RTS

Inputs

  • 0x3071 (vehicle_speed): current vehicle speed
  • 0x3026 bit 7: current high-speed state flag

Outputs

  • 0x3026 bit 7: updated high-speed state flag

RAM Reads

  • 0x3071, 0x3026

RAM Writes

  • 0x3026 (bit 7)

ROM References

  • 0x4A4E -- base speed threshold (low-to-high transition)
  • 0x4A52 -- hysteresis offset (added when transitioning from low to high)
