0x86DA — Dual Lookup Multiply

Field Value
Address 0x86DA
Size 39 bytes
Blocks 3
Type subroutine
Callers 1 call site
Confidence likely

Summary

Performs two independent 2D table lookups — 0x316B through ROM table 0x44C0 and airflow_load (0x306F) through ROM table 0x4620 — then multiplies the two results together. Stores the 32-bit product as high word to 0x3052 and low word to 0x3053, with interrupts disabled for atomicity.

Inputs

  • 0x316B (lookup_input_b): first lookup input
  • 0x306F (airflow_load): second lookup input

Outputs

  • 0x3052 (dual_lookup_product_hi): high 16 bits of product
  • 0x3053 (dual_lookup_product_lo): low 16 bits of product

Calls

  • 0x6000 — graph_lookup_2d: 1D table interpolation (called twice)

RAM Reads

  • 0x316B (lookup_input_b)
  • 0x306F (airflow_load)

RAM Writes

  • 0x300C (tracker_lower_bound): scratch for first lookup result
  • 0x3052 (dual_lookup_product_hi)
  • 0x3053 (dual_lookup_product_lo)

ROM References

  • 0x44C0 (lookup_b_to_factor): maps 0x316B to a factor
  • 0x4620 (airflow_to_factor): maps airflow_load to a factor
