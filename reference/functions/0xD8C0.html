0xD8C0 — Twin Turbo Switching Port Controller

Field Value
Address 0xD8C0
Size 74 bytes (0xD8C0–0xD909)
Blocks 17
Type subroutine
Callers 1 — multi_rate_timer_dispatcher (0xC458) 10-tick path at 0xC480
Confidence confirmed

Summary

Controls the twin turbo switching port bit ($3275 bit 2) and the pre-speed-control flag ($323F bit 4) using a debounce counter at $316C. Called every 10 P5 scheduler ticks. When conditions are met and the counter reaches threshold 0x0A (10), sets $3275 bit 2 (enables twin turbo switching solenoid) and $323F bit 4 (signals pre-speed-control). At threshold 0x14 (20), returns — holding state. When RPM ($3062) is below 0x20, resets the counter and clears the output bit.

Calls $9214 (port_2200_shadow_write) to flush $3275 to hardware port $2200 on every execution.

Entry gate: Three-level gate:

  1. If $35A4 bit 0 set: return (system inhibit)
  2. If $3051 bit 2 set: pass through to main logic
  3. If $3510 is not 0xFF: pass through to main logic
  4. Otherwise: return

Precondition: $305A bit 3 must be set (engine running). If clear, returns.

Debounce logic: If $323F bit 4 is already set OR RPM >= 0x20, increments $316C via JSR $633D (saturating increment). The counter value controls state:

  • Counter reaches 0x14 (20): return (hold, debounce complete)
  • Counter reaches 0x0A (10): set $3275 bit 2 and $323F bit 4 (activate)
  • Counter < 0x0A and RPM < 0x20: reset $316C to 0, clear $3275 bit 2 (deactivate)

Disassembly

D8C0  BBS $35A4,#$01,$D8CF  ; if system inhibit, return
D8C5  BBS $3051,#$04,$D8D0  ; if map_mode bit 2 set, continue
D8CA  BBC $3510,#$FF,$D8D0  ; if diagnostic_mode != 0xFF, continue
D8CF  RTS                    ; early return
D8D0  BBC $305A,#$08,$D8CF  ; if engine not running, return
D8D5  BBS $323F,#$10,$D8E2  ; if pre_speed_control bit 4 already set, skip RPM check
D8DA  LDX $3062              ; load RPM
D8DD  CPX #$20               ; compare against threshold (32)
D8E0  BCC $D8FE              ; if RPM < 32, -> reset path
D8E2  LDX #$316C             ; point to turbo_entry_gate counter
D8E5  JSR $633D              ; saturating increment
D8E8  CMP #$14               ; compare against upper threshold (20)
D8EA  BEQ $D8EE              ; if == 20, -> activation path
D8EC  BCS $D8CF              ; if > 20, return (hold)
D8EE  CMP #$0A               ; compare against lower threshold (10)
D8F0  BEQ $D8F4              ; if == 10, -> set bits
D8F2  BCS $D902              ; if > 10 but < 20, -> clear bit path
D8F4  SEB $3275,#$02         ; set port_control_mirror bit 2 (turbo switch ON)
D8F8  SEB $323F,#$10         ; set pre_speed_control_flags bit 4
D8FC  BRA $D906              ; -> write to port
D8FE  LDM $316C,#$00         ; reset counter to 0
D902  CLB $3275,#$02         ; clear port_control_mirror bit 2 (turbo switch OFF)
D906  JSR $9214              ; write $3275 to port $2200
D909  RTS
