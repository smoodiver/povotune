0xACE1 — Measure Airflow

Field Value
Address 0xACE1
Size 61 bytes (0xACE1–0xAD1D)
Blocks 4
Type subroutine (ISR callee)
Callers 1 — irq_handler_AAD1 P2 task group (0xAAD1 at 0xAB41)
Callees 2 — sub_63E0 (16-bit table interpolation), sub_AD1E (airflow_rate_calc)
Confidence confirmed

Summary

Reads the airflow meter (AFM) sensor via an ADC conversion result and processes it through a 16-bit lookup table interpolation. Waits for the ADC conversion to complete (busy-wait on dp+$20 bit 3), reads the raw result from dp+$22, performs a 16-bit table lookup against the ROM table at 0x4080, then stores the raw and processed results to multiple RAM locations and diagnostic mirrors. Calls sub_AD1E (airflow_rate_calc) for rate-of-change processing.

Disassembly

ACE1  BBC $20,#$08,$ACE1     ; busy-wait until ADC ready (dp+$20 bit 3)
ACE5  CLM                    ; 16-bit mode
ACE6  LDA $22                ; read ADC result from dp+$22 (16-bit)
ACE8  STA $303E              ; save raw ADC value
ACEB  LDX #$4080             ; table base address
ACEE  STX $300A              ; set table_base_ptr for sub_63E0
ACF1  JSR $63E0              ; 16-bit table interpolation
ACF4  STA $3018              ; store raw AFM result (16-bit)
ACF7  LDA $303E              ; reload raw ADC value
ACFA  LSR A                  ; shift right 2 (divide by 4, 16-bit)
ACFB  LSR A
ACFC  SEM                    ; back to 8-bit mode
ACFD  STA $3747              ; diagnostic mirror: AFM high byte / 4
AD00  STA $0110              ; IO diagnostic mirror
AD03  JSR $AD1E              ; call airflow_rate_calc
AD06  SEM                    ; ensure 8-bit mode
AD07  SEB $3025,#$02         ; set airflow_init_flags bit 2 (initialized)
AD0B  CLM                    ; 16-bit mode
AD0C  STA $301A              ; store processed airflow value (16-bit)
AD0F  LSR A                  ; shift right 7 (divide by 128, 16-bit)
AD10  LSR A
AD11  LSR A
AD12  LSR A
AD13  LSR A
AD14  LSR A
AD15  LSR A
AD16  SEM                    ; back to 8-bit mode
AD17  STA $3748              ; diagnostic mirror: processed AFM / 128
AD1A  STA $011A              ; IO diagnostic mirror
AD1D  RTS

Logic

  1. ADC wait (0xACE1): Busy-waits in a tight loop polling dp+$20 bit 3, which signals ADC conversion complete.
  2. Raw read (0xACE5–0xACE8): Reads the 16-bit ADC result from dp+$22 and saves it to 0x303E.
  3. Table lookup (0xACEB–0xACF4): Sets the table base pointer (0x300A) to 0x4080 and calls the 16-bit table interpolation routine (0x63E0). The result is the linearized AFM reading, stored to 0x3018 (afm_raw).
  4. Diagnostic mirror 1 (0xACF7–0xAD00): The raw ADC value is shifted right 2 bits (divide by 4) and the low byte is written to 0x3747 and IO port 0x0110 as a diagnostic output.
  5. Rate calculation (0xAD03): Calls airflow_rate_calc (0xAD1E) which computes rate-of-change with smoothing, returning the processed airflow value in A.
  6. Init flag (0xAD07): Sets 0x3025 bit 2 to indicate the airflow subsystem has been initialized (first valid reading taken).
  7. Processed output (0xAD0C–0xAD1A): The processed value is stored to 0x301A (afm_processed), then shifted right 7 bits and the low byte written to 0x3748 and IO port 0x011A as a second diagnostic output.
