0xAD1E — Airflow Rate Calculator

Field Value
Address 0xAD1E
Size 258 bytes (0xAD1E–0xAE1F)
Blocks 31
Type subroutine
Callers 1 — measure_airflow (0xACE1) at 0xAD03
Callees 0
Confidence confirmed

Summary

Computes a smoothed airflow rate-of-change value from the raw AFM reading, used for transient fuel enrichment and load prediction. The function selects a rate limit from ROM ($46FC or $46FE depending on $305A bit 4), multiplies the RPM-derived value ($320A) against a scaling factor ($46F9), applies bounding (min 0x0100), and runs an exponential smoothing filter to produce the final airflow rate output. On first call (when $3025 bit 2 is clear), it initializes the filter state from a ROM default ($46FA) and returns early.

Detailed Flow

Rate limit selection (0xAD1E–0xAD32):

AD1E  LDX $46FC           ; default rate limit (normal)
AD21  BBS $305A,#$10,$AD29 ; if bit 4 set, skip alternate
AD26  LDX $46FE           ; alternate rate limit
AD29  CPX $3240           ; compare with current stored limit
AD2C  BEQ $AD32           ; if same, skip flag
AD2E  SEB $3166,#$01      ; set rate-change-detected flag
AD32  STX $3240           ; store selected limit

RPM-based scaling (0xAD35–0xAD76):

AD35  LDA $46F9           ; 8-bit scale factor
AD38  CLM                 ; 16-bit mode
AD39  AND #$FF            ; zero-extend to 16-bit
AD3C  ASL A               ; multiply by 4
AD3D  ASL A
AD3E  STA $3008           ; temp = scale * 4
AD41  SEI                 ; disable interrupts
AD42  LDX $320A           ; load RPM-derived value (atomic)
AD46  LDA $320C           ; overflow check
AD4A  BEQ $AD4F           ; if zero, use raw value
AD4C  LDX #$FFFF          ; saturate to max
AD4F  CLI                 ; re-enable interrupts
AD50  TXA
AD51  MPY $3008           ; result = RPM_value * (scale * 4)

The 32-bit MPY result is split into high:low words. The function then selects the bounded result — taking the minimum of the computed value and the rate limit $3240, with a floor clamp at 0x0100:

AD6E  CPX #$0100          ; minimum bound
AD71  BCS $AD76
AD73  LDX #$0100          ; clamp to floor
AD76  STX $321C           ; store as airflow_rate_low

Init-or-update branch (0xAD7A–0xADA5): If $305A bit 4 is set OR $3166 bit 0 is set (rate-change detected), the rate high ($321E) is updated to match $321C. If $3025 bit 2 is clear (first call / not initialized), the function loads defaults from ROM $46FA into $321C/$321E, zeros $3222 (fractional accumulator), copies $3018 to $3220 (initial smoothed value), and returns.

Exponential smoothing filter (0xADA6–0xAE1F): The main filter logic at 0xADA6 computes a weighted blend between the current airflow rate and the previous smoothed value. It uses a shift-and-accumulate approach:

  1. Load $3018 (current AFM reading) into A
  2. Iteratively shift left, counting leading zeros (up to 6 iterations) — this normalizes the value
  3. Compute delta = rate_high - 0x0100 and multiply by the smoothed value ($3220)
  4. Add the product to the workspace accumulators ($3008/$300A)
  5. Saturate on overflow to 0xFFFF:FFFF
  6. Divide result by $321C (via AND operation), clamp on overflow
  7. Right-shift by the normalization count from step 2
  8. Store result to $3220 (smoothed airflow value)
  9. Copy $321C to $321E for next iteration
