0x8E28 — Weighted Accumulate (B_high >> 2)

Field Value
Address 0x8E28
Size 16 bytes (0x8E28–0x8E37)
Blocks 1
Type subroutine
Callers 9 call sites
Confidence confirmed

Summary

Arithmetic helper that takes the high byte of B, shifts it right by 2 (divides by 4), and adds the result to accumulator A. Used in boost_control_main to build a weighted sum of multiple correction terms (RPM correction, load correction, temperature offsets, etc.) by calling this function repeatedly with different values loaded into B before each call.

Disassembly

8E28  ANDB #$FF00        ; clear low byte of B, keep high byte
8E2C  LSRB A             ; logical shift right B
8E2E  LSRB A             ; logical shift right B (total: B >> 2)
8E30  STB $300E           ; store shifted B to workspace
8E34  ADC $300E           ; A = A + shifted_B
8E37  RTS

Inputs

  • A register: running accumulator (sum so far)
  • B register: 16-bit value whose high byte contains the correction term

Outputs

  • A register: accumulator + (B_high >> 2)
  • B register: shifted value (B_high >> 2, in high byte position after shift)

RAM Writes

  • 0x300E (tracker_upper_bound / workspace) — temporary storage for the shifted B value; reused as workspace

RAM Reads

  • 0x300E — read back immediately after write to add to A
