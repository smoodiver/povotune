0xA76F — Knock Retard Ramp Controller

Field Value
Address 0xA76F
Size 183 bytes (0xA76F–0xA825)
Blocks 25
Type subroutine
Callers 1 — knock_control_dispatcher (0xA683 at 0xA6C5)
Confidence confirmed

Summary

Manages the knock retard ramp — a progressive timing retard/advance controller that oscillates the knock multiplier ($3503) to probe for knock boundaries. The function implements a triangular wave pattern: it alternately adds and subtracts a step value ($3502) to/from the knock multiplier, with direction controlled by $3033 bit 7 (ramp direction) and bit 5 (half-step flag).

The ramp has three operating modes based on the knock correction flags:

  1. Fully enabled ($31C2 = 0xFF): Resets ramp state, clears counters.
  2. Sensor active ($31C1 nonzero): Active ramp — either subtracts step (decrementing phase) or adds step (incrementing phase), toggles direction at the limits. Monitors the multiplier value and a cycle counter ($31C9) to detect when the knock boundary has been found (sets $3033 bit 6).
  3. Neither: Counts intervals via $31CA, compares against $31C0 (rpm_timing_factor), triggers ramp update when count reaches threshold.
