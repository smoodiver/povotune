Summary

The largest health check callee (61 blocks). Tests all four knock sensor channels for both high-level and low-level faults, using 8 independent debounce counters and a dual-direction fault detection pattern. Reads knock sensor hardware registers from I/O ports $0C/$0D under interrupt protection (SEI/CLI), copies to workspace at 0x300C/0x300D, then processes each channel identically. Each channel checks an enable bit in 0x300C and a level bit in 0x300D, incrementing the appropriate counter (high-fault or low-fault) and comparing against threshold 0x32 (50 counts). Per-channel results aggregate into 0x31EB (8 bits, 2 per channel) and promote to 0x3330 bits 3–6 (one bit per channel, either direction triggers).

Structure

The function consists of:

  1. Enable/disable gate (0x6FC7–0x6FC9): LDA #$FF; BNE — always branches to main path (the #$FF is a patchable enable byte)
  2. Disable path (0x6FCB–0x6FDF): Clears all knock diagnostic state (0x31EB, 0x3512, 0x3330 bits 3-6, 0x3508 bits 3-6, 0x350C bits 3-6)
  3. Warmup gate (0x6FE0–0x6FF6): Requires 0x316B == 0xFF (fully warmed up); if not, clears all 8 counters (0x31F0–0x31F7) and returns
  4. Hardware read (0x6FF7–0x7004): SEI; LDA $0D; LDB $0C; CLI — atomic read of knock sensor I/O registers, stored to 0x300C (low) and 0x300D (high)
  5. Channel 1 (0x7005–0x7053): Counter pair 0x31F0/0x31F1, enable bit 0x300C.0, level bit 0x300D.7, fault bits 0x31EB.0/.4, output 0x3330.3
  6. Channel 2 (0x7053–0x70A1): Counter pair 0x31F2/0x31F3, enable bit 0x300C.1, level bit 0x300D.6, fault bits 0x31EB.1/.5, output 0x3330.4
  7. Channel 3 (0x70A1–0x70EF): Counter pair 0x31F4/0x31F5, enable bit 0x300C.2, level bit 0x300D.5, fault bits 0x31EB.2/.6, output 0x3330.5
  8. Channel 4 (0x70EF–0x713D): Counter pair 0x31F6/0x31F7, enable bit 0x300C.3, level bit 0x300D.4, fault bits 0x31EB.3/.7, output 0x3330.6

Per-Channel Logic (identical for all 4)

Using channel 1 as example (X = 0x31F0):

if 0x300C bit 0 set (channel enabled):
    if 0x300D bit 7 set (level high):
        JSR 0x633D (increment counter at X=0x31F0)
        if count >= 0x32: SEB 0x31EB bit 0 (high fault)
    else:
        MEM 0x31F0 = 0x00 (clear high counter)
        CLB 0x31EB bit 0 (clear high fault)
else (channel not enabled):
    if 0x300D bit 7 NOT set (level low):
        INX (X = 0x31F1)
        JSR 0x633D (increment counter at X+1=0x31F1)
        if count >= 0x32: SEB 0x31EB bit 4 (low fault)
    else:
        MEM 0x31F1 = 0x00 (clear low counter)
        CLB 0x31EB bit 4 (clear low fault)

; Aggregate to output
if 0x31EB bits 0+4 both set: SEB 0x3330 bit 3
else: CLB 0x3330 bit 3

Note: The aggregation check uses OR — if EITHER high or low fault bit is set for a channel, the channel fault bit in 0x3330 is set. The BBC check at e.g. 0x7044 tests mask 0x11 (bits 0+4), meaning either bit triggers the output.

Inputs

  • I/O port $0D: knock sensor level register (hardware, bits 7/6/5/4 = channels 1/2/3/4)
  • I/O port $0C: knock sensor enable register (hardware, bits 0/1/2/3 = channels 1/2/3/4)
  • 0x316B: warmup counter (must be 0xFF = fully warmed)

Outputs

  • 0x300C: knock enable snapshot (copied from port $0C)
  • 0x300D: knock level snapshot (copied from port $0D)
  • 0x31EB: knock sensor fault bits (8 bits: 4 high + 4 low)
  • 0x31F0–0x31F7: per-channel debounce counters (8 bytes, 2 per channel)
  • 0x3330 bits 3–6: per-channel confirmed knock sensor faults
  • 0x3512: diagnostic init state 2 (cleared on disable)
  • 0x3508 bits 3–6: diagnostic counter 2 (cleared on disable)
  • 0x350C bits 3–6: latched faults B (cleared on disable)
