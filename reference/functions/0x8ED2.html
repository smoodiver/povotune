Summary

Conditionally enforces a minimum duty floor (0x324E) for boost control. First validates multiple preconditions: (1) boost output value 0x30EF must be below a threshold at ROM[0x4B00], (2) RPM must be within a window defined by ROM[0x4B01..0x4B04] (16-bit), (3) secondary_condition_value must be >= ROM[0x4AFC]. If preconditions pass, selects a floor value from one of two ROM offsets (0x4B0A or 0x4B07, indexed from X=0x4B00) depending on 0x305F bits 1 and 3, and writes it to 0x324E if it is greater than the current value.

Inputs

  • 0x30EF: current boost output value
  • 0x3062 (rpm): engine speed (16-bit)
  • 0x30B6 (secondary_condition_value): load/pressure input
  • 0x305F bits 1, 3: boost rate-of-change state flags
  • 0x324E: current duty floor (16-bit)

Outputs

  • 0x324E: updated duty floor (only if new floor > current)
