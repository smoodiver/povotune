0xC458 — Multi-Rate Timer Dispatcher

Field Value
Address 0xC458
Size 65 bytes (0xC458–0xC498)
Blocks 14
Type subroutine
Callers 1 — scheduler P5 (0xAAD1 at 0xABA1)
Confidence confirmed

Summary

Divides the scheduler P5 tick rate into three slower sub-rates using counters. Each counter increments per call; when it reaches its threshold the associated sub-functions are called and the counter resets to 0.

Counter Threshold Period Sub-calls
0x3111 5 every 5 ticks 0xCBA1 (sub_CBA1), 0xC513 (counter_update_5tick)
0x311D 10 every 10 ticks 0xC499 (counter_update_10tick), 0xC54A (o2_heater_control), 0xD86E (boost_target_pressure_adjustment), 0xD8C0 (twin_turbo_switching_port_controller)
0x311A 49 (0x31) every 49 ticks 0xC532 (counter_update_49tick)

The 5-tick path also sets bit 4 of 0x31A9 before calling sub-functions.

Inputs

  • 0x3111 (timer_5tick): 5-tick counter
  • 0x311D (timer_10tick): 10-tick counter
  • 0x311A (timer_49tick): 49-tick counter

Outputs

  • 0x3111 (timer_5tick): incremented or cleared
  • 0x311D (timer_10tick): incremented or cleared
  • 0x311A (timer_49tick): incremented or cleared
  • 0x31A9: bit 4 set on 5-tick path

Calls

  • 0xCBA1 — sub_CBA1 (5-tick path)
  • 0xC513 — counter_update_5tick (5-tick path)
  • 0xC499 — counter_update_10tick (10-tick path)
  • 0xC54A — o2_heater_control (10-tick path)
  • 0xD86E — boost_target_pressure_adjustment (10-tick path)
  • 0xD8C0 — twin_turbo_switching_port_controller (10-tick path)
  • 0xC532 — counter_update_49tick (49-tick path)

RAM Reads

  • 0x3111, 0x311D, 0x311A: timer counters

RAM Writes

  • 0x3111, 0x311D, 0x311A: timer counters reset to 0 or incremented
  • 0x31A9: bit 4 set on 5-tick overflow
