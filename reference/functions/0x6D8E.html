0x6D8E — Idle Speed Control Valve Check

Field Value
Address 0x6D8E
Size 143 bytes (0x6D8E–0x6E1C)
Blocks 29
Type subroutine
Callers 1 call site (health_check_dispatcher 0x6B14, slot 8)
Confidence confirmed

Summary

Two-part health check covering the idle speed control (ISC) valve and idle speed fault detection. Part 1 validates the ISC valve position sensor (0x329E) is within operational range. Part 2 checks for sustained idle speed deviation indicating a stuck or failed ISC valve.

Part 1 — ISC valve range check (0x6D8E–0x6DC7): Gated by 0x31DF bit 3 (must be clear) and 0x32B7 (must be nonzero, and value-1 must be >= ROM threshold at 0x48C6 = 0x0A). If the gate fails, the function returns immediately. Then checks ISC position (0x329E) against range 0x0028–0x03D4. Out-of-range increments fault counter 0x31FA; when it trips the ROM threshold at 0x4C65 (0x27 = 39), sets 0x31F8 bit 2 (ISC valve fault). In-range clears the counter and fault bit.

Part 2 — Idle speed deviation check (0x6DC8–0x6E1C): Gated by 0x332F bit 4, 0x3330 bits 0+2, 0x305A bit 4 (closed-loop active), and 0x306F (airflow load must be > 0x14). Checks ISC position (0x329E) must be > 0x01F4 (500). If all conditions met, increments counter 0x32C7; when it trips ROM threshold at 0x4C66 (0x27 = 39), sets 0x31CF bit 0. If position <= 0x01F4, clears 0x31CF bit 0.

Final output decision: if 0x31F8 bits 0-2 (any of O2 front/rear/ISC faults) OR 0x31CF bit 0 are set, sets 0x332E bit 4 (idle speed fault confirmed). Otherwise clears it.

Inputs

  • 0x31DF bit 3: ISC check inhibit flag
  • 0x32B7: ISC activity counter (must be nonzero and decremented value >= ROM[0x48C6])
  • 0x329E: ISC valve position (16-bit)
  • 0x31FA: ISC range fault counter
  • 0x332F bit 4: intake air temp fault (inhibits part 2)
  • 0x3330 bits 0, 2: VSS/coolant faults (inhibit part 2)
  • 0x305A bit 4: closed-loop flag
  • 0x306F: airflow load
  • 0x32C7: idle deviation counter
  • 0x31F8: O2/ISC sensor status bits
  • 0x31CF: diagnostic pending faults

Outputs

  • 0x31FA: ISC range fault counter
  • 0x31F8 bit 2: ISC valve out-of-range fault
  • 0x32C7: idle deviation counter
  • 0x31CF bit 0: idle deviation pending fault
  • 0x332E bit 4: confirmed idle speed fault (output decision)
