Summary

Checks the fuel injector circuit for injector pair 1. Requires 0x316B >= 0x8A (engine running). Uses 0x3275 bit 4 (port_control_mirror) to select between two injector drive lines, then reads 0x3806 bit 5 (ext_io_reg_1) as the feedback signal. For each line:

  • If the drive is active and feedback confirms it, increments a saturating counter (0x3332 or 0x3333 via X-indexed) via JSR $633D. When counter reaches 0x28 (40), sets the corresponding bit in 0x333A (bit 0 or bit 1).
  • If feedback contradicts the drive command, clears the corresponding 0x333A bit and resets the counter to 0.

At exit, if both 0x333A bits 0+1 are set (mask 0x03), sets 0x3331 bit 0. Otherwise clears 0x3331 bit 0.

This is the first of a series of structurally identical injector check functions (0x7353, 0x73AF, 0x740B, 0x7467, 0x74C3) that each check a different injector pair using different bits of 0x3275/0x35A5 and 0x3806.

Inputs

  • 0x316B (lookup_input_b): >= 0x8A required (engine running gate)
  • 0x3275 (port_control_mirror): bit 4 (injector drive select)
  • 0x3806 (ext_io_reg_1): bit 5 (injector feedback signal)

Outputs

  • 0x333A: bits 0, 1 (individual injector circuit fault flags)
  • 0x3331 (control_reg_3331): bit 0 (combined injector pair 1 fault)
  • 0x3332: debounce counter for injector line A
  • 0x3333: debounce counter for injector line B (via X+1)

Calls

  • 0x633D — saturating_increment (x2, once per injector line)
