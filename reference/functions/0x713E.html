Summary

Monitors front and rear O2 sensor outputs and sets lean-detection status flags in 0x31F8. Gated by multiple prerequisites: lookup_input_b >= 0x8A, system_control_flags bit 2 clear, and load within ROM-defined window (0x4C04–0x4C06). Uses Port 6 bit 1 to select between front (0x320E) and rear (0x320F) sensor paths. Each path reads port 0x2000 bit 6: if lean, increments a saturating counter; when counter reaches 0x30 (48 iterations), sets the corresponding status bit. Rich readings clear the counter and status bit.

Inputs

  • 0x316B (lookup_input_b): >= 0x8A required
  • 0x305A (system_control_flags): bit 2 must be clear
  • 0x30E2 (closed_loop_load_input): must be within 0x4C04–0x4C06 window
  • Port 0x2000 bit 6: O2 sensor signal (clear = lean)
  • Port 0x0E bit 1: sensor selection (clear = front, set = rear)

Outputs

  • 0x31F8 (o2_sensor_status): bits 3,4 set/cleared
  • 0x320E (o2_front_lean_timer): front sensor counter
  • 0x320F (o2_rear_lean_timer): rear sensor counter
