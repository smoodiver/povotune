0x9D39 — Cylinder Pair Injection Sequencer

Field Value

Confidence confirmed

Summary

Injection sequencing controller called during Phase 1 of the Timer A ISR. Manages cylinder-pair injection ordering for the 4-cylinder sequential injection system. On first call for a new engine cycle, initializes per-cylinder injection schedule registers ($3197–$319A) based on the current cylinder and a previous-cylinder reference ($319B). On subsequent calls, dispatches the per-cylinder injection chain via 0x9E11.

The function implements a two-phase state machine controlled by $31B4 (injection sequencing state) and $31C6 (alternation toggle):

  1. Init phase ($31B4 bit 1 clear): Toggles $31C6; on the "active" toggle, saves current cylinder to $319B, clears all scheduling registers, and calls 0x9E11 to begin injection. On the "inactive" toggle, just clears $31B4/$31B5 and returns.
  2. Sequencing phase ($31B4 bit 2 set): Uses $319C (current cylinder) and $319B (previous saved cylinder) as a pair to look up the injection firing order, loads per-channel values into $3197–$319A, sets corresponding bits in $31B5, then calls 0x9E11 to dispatch.

Firing Order Table

The function encodes the 4-cylinder firing order through four case blocks, each mapping a (current, previous) cylinder pair to a channel assignment:

$319C $319B $3197 $3198 $3199 $319A $31B5 bits
1 4 05 09 07 03 0x09 (bits 0,3)
3 1 03 07 05 09 0x05 (bits 0,2)
4 2 07 03 09 05 0x0A (bits 1,3)
2 3 09 05 03 07 0x06 (bits 1,2)

Inputs

  • $319C — cylinder counter (current cylinder, 1–4)
  • $319B — previous cylinder reference
  • $31B4 — injection sequencing state (bits 0–2)
  • $31C6 — alternation toggle (flipped each call)

Outputs

  • $3197–$319A — per-cylinder injection schedule values
  • $31B4 — state bits updated (bit 2 set after sequencing)
  • $31B5 — output channel flags (cleared or set per firing order)
  • $319B — saved cylinder counter (init phase)
  • $31C6 — toggled alternation flag
