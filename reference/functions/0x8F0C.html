0x8F0C — Duty Floor Decay

Field Value
Address 0x8F0C
Size 18 bytes (0x8F0C–0x8F1D)
Blocks 3
Type subroutine
Callers 1 call site (periodic context at 0x6A61)
Confidence confirmed

Summary

Decrements the boost duty floor accumulator (0x324E) by a ROM constant (0x4B05) each call, clamping to zero on underflow. This provides time-based decay of the duty floor set by the boost duty floor enforcer (0x8ED2).

Disassembly

8F0C  CLM                    ; 16-bit mode
8F0D  LDA $324E              ; load duty floor accumulator
8F10  SEC
8F11  SBC $4B05              ; subtract decay step
8F14  BCS $8F19              ; if no underflow, keep result
8F16  LDA #$0000             ; clamp to zero
8F19  STA $324E              ; store updated floor
8F1C  SEM                    ; 8-bit mode
8F1D  RTS

Inputs

Outputs

  • 0x324E: decremented duty floor, clamped to 0
