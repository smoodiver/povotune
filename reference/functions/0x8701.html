0x8701 — Boost Control Main

Field Value
Address 0x8701
Size 1816 bytes

Type subroutine
Callers 1 call site
Confidence confirmed

Summary

Master wastegate boost control routine — the largest function in the entire ROM. Implements a complete closed-loop boost controller with 10 phases:

  1. Mode selection (0x8701): Selects between low-speed and high-speed max-duty-limit table sets based on vehicle speed (traction-aware torque management, not different boost targets)
  2. Cranking override (0x876C): Startup enrichment with timer pairs
  3. Rate-of-change smoothing (0x87CD): Asymmetric attack/decay to prevent boost spikes
  4. Closed-loop feedback AT (0x889A): Proportional control with configurable gains for automatic transmission
  5. Closed-loop feedback MT (0x896A): Manual transmission feedback path with snapshot correction
  6. Auxiliary corrections (0x8A3E): AC compressor load, overboost protection
  7. Engine protection (0x8AEE): Speed-based wastegate floor, decel fuel cut
  8. Active boost computation (0x8BD9): Acceleration enrichment, gain chain, component summation
  9. Passive boost computation (0x8D39): Simplified path without boost cut
  10. Final output (0x8DDB): PWM clamping (0x0332–0x3CBC), 8-bit conversion, I/O write

Outputs

  • 0x30E2 (boost_final_duty) → I/O 0x016E
  • 0x30F8 (boost_target_pressure) → I/O 0x017E (MSB)
  • 10 I/O ports (0x016C–0x0184): diagnostic outputs for each computation stage
