Field Value

Type subroutine

Confidence confirmed

Summary

Injection output and dwell timing calculator called during Phase 2 of the Timer A ISR (0x98A1). Performs three major tasks: (1) selects and writes the injector output value to IO port 0x2800 via a cylinder-indexed ROM table, (2) computes the injection pulse width from base fuel value, timing result, and various corrections, and (3) calculates dwell (coil charge) timing by scaling the timing result and subtracting a timer-derived offset. The function arms the injection timer hardware by writing computed period and pulse width values to direct-page timer registers.

Algorithm Detail

Phase 1 — Injector Output Select (0x9B99–0x9BE6)

  1. Reads cylinder counter $319C. If zero (no cylinder identified), checks $31A5 bit 3 (first tooth detected); if not set, returns immediately. If set, clears the bit and uses injector index 0.
  2. If $319C != 0 and $31A1 bit 2 is set, skips the output write and jumps to dwell re-arm at 0x9BF2.
  3. Clears $31A1 bit 5, decrements cylinder counter to get table index.
  4. Tests ROM $448E for 0xFF to select between standard table ($9D31, 4 bytes: 01 01 03 03) or alternate table ($9D35, 4 bytes: 09 0A 0B 0C).
  5. CLM (16-bit), masks index to 8-bit, adds to table base pointer to form X.
  6. Writes 0xFFFF to direct page $88 (timer compare register), loads table[X] in 8-bit mode, writes to IO $2800 (injector hardware port).
  7. Sets $31A1 bit 3.

Phase 2 — Injection Pulse Width Calculation (0x9BF2–0x9CD7)

  1. Calls $A9ED for pre-injection timing correction.
  2. Checks $35A4 bit 0 for alternate mode: if set, loads injection base from $3576; otherwise computes from ROM $4480 + $309A (saturating add), minus $3050 (clamped to 0).
  3. Multiplies result by $31B0 (timing_result_C), calls $A3C1 (shift right by 4), adds $3052 (saturating to 0xFFFF). Stores to $3236.
  4. Loads ROM $4464 (max scale = 0x64 = 100), multiplies by $31B0, shift-right-by-4. If overflow, saturates to 0xFFFF. Clamps result against $3236 (uses minimum).
  5. Stores final injection pulse width to $3146.

Phase 3 — Dwell Timing Calculation (0x9C49–0x9D01)

  1. Multiplies constant 0xB4 (180) by $31B0, shift-right-by-4. If overflow, saturates. Subtracts $3146 (injection pulse), clamps minimum to 0xFA (250).
  2. Stores dwell period to $3148.
  3. Computes corrected timing: 0x41 minus $3050 (or $3573 in alternate mode), clamped to minimum 0x0F.
  4. Multiplies by $31B0, shift-right-by-4. Stores to $3004.

Phase 4 — Timer Arm (0x9C92–0x9D30)

  1. Manipulates direct-page $B4 bits 2/4 (injection state machine toggles).
  2. Captures timer $A0, right-shifts by 4, adds 0x0C, stores as timer offset to $3006.
  3. Subtracts offset from $3004 (dwell period), clamps minimum to 0x0B. Stores to $314A.
  4. Compares $314A - $3146: if difference >= 0x14 (20), enters the "normal dwell" path setting direct-page timer registers $80, $84, $88 for the hardware timer. Tests $31A1 bit 3 and $0E bit 4 to determine output mode (0x0E or 0x02 loaded to A).
  5. If difference < 0x14, enters the "short dwell" path: loads $314A to $80, $3148 to $84, 0xFFFF to $88, loads 0x0D to A.
  6. Stores A to $96 (injection timer mode), sets $92 bit 7, clears $94 to 0. Returns.

Inputs

  • $319C — cylinder counter (0–4)
  • $31A1 — engine running flags (bits 2, 3, 5, 8 tested)
  • $31A5 — first tooth flags (bit 3)
  • $31B0 — timing_result_C (from Phase 2 of ISR)
  • $3050 — correction offset
  • $309A — injection base adjustment
  • $3052 — injection additive term
  • $35A4 — alternate mode flags (bit 0)
  • $3573 — alternate correction value
  • $3576 — alternate injection base
  • $00A0 — free-running timer (direct page)
  • $000E — IO port (bit 4: output mode select)

Outputs

  • IO $2800 — injector driver port (cylinder-specific output pattern)
  • $3146 — computed injection pulse width
  • $3148 — computed dwell period
  • $314A — dwell timing value (period minus timer offset)
  • $314C — dwell/injection difference
  • $3236 — intermediate injection maximum
