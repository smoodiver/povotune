0x82D6 — Ignition Timing

Summary

Computes the final ignition timing angle by looking up base and advance timing from two 16x16 ROM maps, applying knock correction from learned values, and enforcing zone-based limits. The function performs two map_lookup_16x16 calls (base map 0x5100, advance map 0x5000), sums them to get the raw combined timing, then runs an 8-zone RPM/load detection system that gates knock correction application. The final output at 0x31BB is the ignition timing value used by the hardware driver.

Inputs

  • 0x3062: current RPM — drives map lookup axis
  • 0x313C: engine load, 16-bit — drives map lookup axis
  • 0x3092: engine state value — compared against 0x4D8F
  • 0x3503: knock correction multiplier
  • 0x31C2: knock correction enable
  • 0x31B9: timing advance offset
  • 0x323A: knock correction channel 1
  • 0x323C: knock correction channel 2
  • 0x35A4 bit 0: self-test status — skips knock correction when active

Outputs

  • 0x31BB: final ignition timing — drives hardware
  • 0x3017: combined base + advance timing
  • 0x31C8: base timing from map 0x5100
  • 0x31C7: advance timing from map 0x5000
  • 0x31EE: map lookup index copy
  • 0x31BE: load zone value — shifted load used for zone detection
  • 0x31BF: RPM zone value — RPM >> 4, 8-bit
  • 0x31C0: RPM-to-timing table value from ROM 0x4DC0
  • 0x31CB: combined knock correction
  • 0x323A: updated knock correction channel 1
  • 0x323C: updated knock correction channel 2
  • 0x3036: RPM/load zone flags (8 bits, hysteresis) — bits 0–3 for RPM zones, bits 4–7 for load zones
  • 0x3033: zone control flags — bits 0, 1, 2 aggregated from zone detection
  • 0x3034 bits 0–2: ignition control flags

Calls

  • 0x6080 — map_lookup_16x16: bilinear interpolation (called twice: base map 0x5100, advance map 0x5000)
  • 0x85A7 — sub_85A7: knock correction calculation (called twice: channels at 0x4700 and 0x4708). Historical: "possibly knock reduction"
  • 0x630E — sub_630E: bit shift left 4 (historical: "Shift AL and BL 4 bits to the left")
