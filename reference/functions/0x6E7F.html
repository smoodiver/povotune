0x6E7F — Closed-Throttle Position Switch Check

Field Value
Address 0x6E7F
Size 128 bytes (0x6E7F–0x6EFE)
Blocks 25
Type subroutine (health check callee)
Callers 1 call site (health_check_dispatcher 0x6B14, slot 10)
Callees 0x634E (saturating_increment_y_indexed), 0x633D (saturating_increment) — walker reports 0 callees; 0x634E called via JSR at 0x6EB9
Confidence confirmed

Summary

Tests the closed-throttle position switch (idle contact) for stuck-open faults. Verifies that the throttle position switch closes when driving conditions indicate it should be closed (vehicle moving, engine running, no speed signal anomalies). Uses a Y-indexed saturating increment counter at 0x3180 with a 16-bit threshold of 0x024A (586 counts) before setting a pending fault. Fault confirmation promotes through 0x332F bit 6 after additional checks, gated by diagnostic timer 0x318C >= 0x030D.

Logic Flow

  1. Precondition gates (0x6E7F–0x6E93): Exit early if any prior fault is already set in 0x332F bit 2, 0x332E bit 6, or 0x3330 bit 0
  2. Engine state check (0x6E94–0x6E99): Exit if speed snapshot 0x3167 != 0 (vehicle not stationary — inverted logic: test requires vehicle moving, so 0x3167 must be 0 meaning "test passed")
  3. Running gate (0x6E99): BBC 0x3051 bit 2 — engine must be running
  4. Bypass check (0x6E9E): BBS 0x305A bit 4 — skip if system control flag set
  5. Throttle position test (0x6EA3–0x6EAF): LDA 0x3745, CMP #0x2F — throttle input must be >= 0x2F; LDA 0x306E, CMP #0x32 — knock sensor must be >= 0x32
  6. Idle switch check (0x6EB1): BBS 0x305A bit 0 — skip if idle switch already active
  7. Fault counter (0x6EB6–0x6EBF): Increment Y-indexed counter at 0x3180 via JSR 0x634E; compare X against 0x024A (16-bit threshold)
  8. Pending fault set (0x6EC1): SEB 0x31CF bit 4 — set pending closed-throttle fault
  9. Clear path (0x6EC5): On any precondition failure, clear counter 0x3180 to 0x0000

Fault Confirmation (0x6ECB–0x6EFE)

  • If 0x332F bit 6 already set OR 0x31CF bit 4 pending:
    • Check 0x3167 >= 2 → clear both 0x332F bit 6 and 0x31CF bit 4, set 0x3030 bit 2 (throttle state recovery)
    • If 0x31CF bit 4 still pending and diagnostic mode (0x3510 != 0): check timer 0x318C >= 0x030D before promoting to 0x332F bit 6 (confirmed fault)

Inputs

  • 0x332F bit 2: prior crank position fault (inhibits test)
  • 0x332E bit 6: prior ignition signal fault (inhibits test)
  • 0x3330 bit 0: prior VSS fault (inhibits test)
  • 0x3167: speed snapshot (must be 0 for test to run)
  • 0x3051 bit 2: engine running flag
  • 0x305A bit 4: system control bypass flag
  • 0x305A bit 0: idle switch state
  • 0x3745: throttle position input (>= 0x2F required)
  • 0x306E: knock sensor value (>= 0x32 required)
  • 0x3510: diagnostic mode control
  • 0x318C: diagnostic elapsed timer (16-bit)

Outputs

  • 0x31CF bit 4: pending closed-throttle fault
  • 0x332F bit 6: confirmed closed-throttle / ignition coil fault
  • 0x3030 bit 2: throttle state recovery flag
  • 0x3180: fault debounce counter (16-bit, Y-indexed via 0x634E)
