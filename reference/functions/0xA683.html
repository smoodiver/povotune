0xA683 — Knock Control Dispatcher (Phase 0)

Field Value
Address 0xA683
Size 104 bytes (0xA683–0xA6EA)
Blocks 20
Type subroutine
Callers 1 — Timer A ISR Phase 0 (0x98A1 at 0x99DC)
Confidence confirmed

Summary

Knock control dispatcher called during ISR Phase 0. Evaluates a chain of preconditions to determine whether to run knock learning (0xA826), knock retard ramp (0xA76F), or just the timing decay (0xA745). The precondition chain tests system mode flags, RPM zone state, error flags, and the knock multiplier state to select the appropriate sub-function.

When conditions are not met for active knock processing, the function defaults to calling the timing decay function (0xA745) and clearing the knock interval counter.

Disassembly

; --- early exits ---
A683  BBS $305C,#$06,$A6D3 ; if mode_select bits 1+2 set, -> hard reset
A688  BBS $31FC,#$40,$A6D2 ; if knock enable flag set, -> return (already active)
A68D  BBC $31A1,#$02,$A6C8 ; if not in high-RPM zone, -> soft disable
A692  BBS $3021,#$80,$A6C8 ; if system error flag set, -> soft disable
; --- precondition chain ---
A697  BBC $3034,#$02,$A6DF ; if ign_control bit 1 clear, -> default path
A69C  BBS $31EA,#$40,$A6DF ; if diag intermittent flag set, -> default
A6A1  LDA $322C           ; load isr_event_flags
A6A4  AND #$24            ; test bits 2 and 5
A6A6  BNE $A6DF           ; if either set, -> default path
A6A8  LDA $31CC           ; load af_ratio_mode
A6AB  BNE $A6DF           ; if nonzero, -> default path
; --- dispatch knock processing ---
A6AD  BBC $3033,#$40,$A6C0 ; if ign_zone_control bit 6 clear, -> try ramp
A6B2  BBC $3033,#$02,$A6DF ; if bit 1 also clear, -> default
A6B7  JSR $A826           ; call knock_learning_update
A6BA  LDM $31CA,#$00      ; clear knock interval counter
A6BE  BRA $A6CC           ; -> cleanup
; --- alternative: knock ramp ---
A6C0  BBC $3033,#$01,$A6DF ; if bit 0 clear, -> default
A6C5  JSR $A76F           ; call knock_retard_ramp
; --- soft disable / cleanup ---
A6C8  SEB $3034,#$08      ; set ign_control bit 3 (knock processed flag)
A6CC  LDY #$00
A6CF  STY $31B8           ; clear timing_correction to 0
A6D2  RTS
; --- hard reset ---
A6D3  CLB $3033,#$40      ; clear ign_zone_control bit 6
A6D7  CLB $3034,#$04      ; clear ign_control bit 2
A6DB  LDM $3503,#$00      ; clear knock_multiplier to 0
A6DF  JSR $A745           ; call timing_decay
A6E2  LDM $31CA,#$00      ; clear knock interval counter
A6E6  SEB $3034,#$08      ; set ign_control bit 3
A6EA  RTS
