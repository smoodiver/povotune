0xB2B1 — AN2 Boost Pressure Reader

Field Value
Address 0xB2B1
Size 47 bytes (0xB2B1–0xB2DF)
Blocks 6
Type subroutine (ISR callee)
Callers 1 — scheduler P0 (0xAAD1 at 0xAC0D)
Callees 1 — iir_filter (0xB2E0)
Confidence confirmed

Summary

Reads ADC channel 2 (AN2) result and stores it as the boost pressure value at $3314. Called from scheduler priority 0 (1/20 ticks, lowest priority, last in P0 group). Uses an alternating-call pattern controlled by $302D bit 1: on even calls it reads and processes the ADC result; on odd calls it simply toggles the flag and returns. This halves the effective sampling rate for the boost pressure channel compared to other ADC readers.

Algorithm Detail

  1. BBC $20,#$08,$B2B1 — spin until ADC conversion complete
  2. Check $302D bit 1:
    • If set (odd call): clear $302D bit 1 and return immediately (skip processing)
    • If clear (even call): proceed with processing
  3. Switch to 16-bit mode (CLM), load result from dp+$22
  4. X = $3314 (destination)
  5. If $3166 bit 7 already set: call IIR filter (0xB2E0) to blend with previous
  6. Set $3166 bit 7 (init flag)
  7. Store filtered result to [X] ($3314)
  8. Shift right by 2, switch to 8-bit mode
  9. Store 8-bit value to $3794 (diagnostic mirror)
  10. Set $302D bit 1 (flag for next-call skip)

Inputs

  • dp+$20 — ADC status register (bit 3: conversion complete)
  • dp+$22 — ADC result (16-bit)
  • $302D — alternation flag (bit 1)
  • $3166 — initialization flags (bit 7: filter initialized)

Outputs

  • $3314 — filtered boost pressure value (16-bit)
  • $3794 — 8-bit diagnostic mirror (boost pressure >> 2)
  • $302D — bit 1 toggled each call
  • $3166 — bit 7 set after first call
