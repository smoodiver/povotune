0xD90A — Ignition Timing Retard Controller

Field Value
Address 0xD90A
Size 328 bytes (0xD90A–0xDA51)
Blocks 47
Type subroutine
Callers 1 — ignition_timing_output_calculator (0xA9ED at 0xAABB)
Confidence confirmed

Summary

Computes a timing retard amount that is subtracted from the ignition timing base value ($3278). Called from the ignition timing output chain after sub_DB51 stores the initial converted timing to $3278. The result in A on return is used as the final timing input to sub_DBBB. Uses a multi-phase state machine controlled by $3276 bits to manage transition entry, ramp-up, decay, and exit for the retard correction.

This function implements a load-transient timing retard — when a sudden change in airflow or RPM derivative is detected, it temporarily pulls back ignition timing to prevent knock, then gradually restores it. The retard magnitude is scaled by coolant temperature and airflow, and rate-limited by an RPM-acceleration-derived ramp controller.

Entry Conditions Gate (0xD90A–0xD930)

When $305C bit 5 is set, performs a window check on four parameters against ROM thresholds:

  • $3300 (learned_correction_1) must be <= ROM[$4452]
  • $309B (speed_value) must be within ROM[$444E]–ROM[$444F]
  • $306F (airflow_load) must be within ROM[$4450]–ROM[$4451]

If any check fails, falls through to the activation check. If $305C bit 5 is clear, skips directly to activation.

Activation Check (0xD931–0xD97E)

If $305A bit 0 is set, aborts (clears $3276 bit 1, returns $3278). Otherwise checks $3276 bit 5 (transition trigger):

  • On trigger (bit 5 set): Initializes the retard state machine:
    • Clears bit 5, sets bit 1 (active), clears bits 4 and 6
    • Zeros working variables $322D, $31FD, $327A
    • Sets airflow threshold: $3274 = $306F + 0x1E (clamped at 0xFF)
  • While active (bit 1 set): Checks if airflow ($306F) has dropped below $3274 — if so, retard has served its purpose and aborts. Also checks ROM[$4442] against counters $322D and $31FD to gate whether enough time has elapsed.

Retard Computation (0xD988–0xD9DE)

Computes two working values:

  1. $327B (gear-derived offset): Extracts gear position from $3044 (throttle_position >> 4, masked to 4 bits), subtracts ROM[$4440] base offset
  2. $3273 (retard magnitude): Starts from $3278 (timing base), then scaled by coolant temperature:
    • If $3092 < ROM[$4445]: shift right twice (divide by 4)
    • If $3092 < ROM[$4446]: shift right once (divide by 2)
    • Full value if coolant is warm enough
  3. Airflow scaling: If $306F >= ROM[$444B], shifts $3273 right once more (halves retard at high airflow). Otherwise subtracts ROM[$444C] from both $3278 and $3273 (with additional ROM[$444D] subtraction on $3273).
  4. Low speed guard: If $309B < 3, forces $3273 to 0 (no retard at very low speed).

Ramp Controller (0xD9E9–0xDA51)

Uses $326E (signed RPM acceleration from 0xDA52) to control the retard application rate:

  • $326E < 0 (decelerating): Returns $3278 directly unless $3276 bit 6 is set, in which case manages a two-phase exit via bits 4 and 6 toggles
  • $326E >= 0 (accelerating): Sets bit 6, then:
    • Phase 1 (bit 4 clear): Tracks peak $326E in $327A. If $326E-1 >= $327B, applies full $3273 retard; else applies 0
    • Phase 2 (bit 4 set): Computes decay cap = $327A * ROM[$4441], shifted left by 8 (JSR $6311). Clamps $3273 to this decaying cap
  • Final output: A = $3278 - $3273 (clamped at 0)

Disassembly (key sections)

; Entry gate — window check
D90A  BBC $305C,#$20,$D931   ; if mode bit 5 clear, skip window
D90F  LDA $4452              ; load learned correction max
D912  CMP $3300              ; compare against learned_correction_1
D915  BCC $D97E              ; if exceeded, abort
; ...speed and airflow range checks...

; Activation trigger
D936  BBC $3276,#$20,$D964   ; if trigger bit not set, check active
D93B  CLB $3276,#$20         ; clear trigger
D93F  SEB $3276,#$02         ; set active
D943  CLB $3276,#$10         ; clear phase 2
D947  CLB $3276,#$40         ; clear ramp flag
D94B  LDM $322D,#$00         ; zero counter
D94F  LDM $31FD,#$00         ; zero counter
D953  LDM $327A,#$00         ; zero peak tracker
D957  LDA $306F              ; airflow
D95A  CLC
D95B  ADC #$1E               ; add offset (30)
D961  STA $3274              ; store threshold

; Retard computation — coolant scaling
D997  LDA $3278              ; load timing base
D99A  STA $3273              ; copy to working retard
D99D  LDA $3092              ; load coolant temp
D9A0  CMP $4446              ; warm threshold
D9A3  BCS $D9B0              ; if warm enough, no scaling
D9A5  CMP $4445              ; mid threshold
D9AA  LSR $3273              ; shift right (divide by 2)
D9AD  LSR $3273              ; shift right again (÷4 total if cold)

; Ramp controller — RPM acceleration
D9E9  LDA $326E              ; load RPM acceleration
D9EC  DEA                    ; decrement
D9ED  BPL $DA0C              ; if >= 0, -> ramp logic

; Final output
DA46  LDA $3278              ; load timing base
DA49  SEC
DA4A  SBC $3273              ; subtract retard amount
DA4D  BCS $DA51              ; if no underflow, return
DA4F  LDA #$00               ; clamp at 0
DA51  RTS
