0xB7C8 — Final Fuel Pulse Width Blender

Field Value
Address 0xB7C8
Size 73 bytes (0xB7C8–0xB810)
Blocks 5
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 via 0xAB5C)
Callees none
Confidence confirmed

Summary

Blends the base fuel pulse width (from 0xB728) with the fuel correction factor (from 0xB68F) to produce the final engine load / injection pulse width value. The function computes a weighted interpolation: result = blend_ratio * fuel_correction + (1 - blend_ratio) * base_fuel, where blend_ratio is 0x3372 (from 0xB68F). The output is clamped to the airflow ceiling 0x30B0 and stored to 0x313C (engine load, 16-bit), with 8-bit mirrors to 0x3745, IO 0x010C, and 0x327C.

Flow

  1. Switches to 16-bit mode (CLM).
  2. Corrected component: Multiplies 0x3372 (blend ratio) by 0x3378 (fuel correction multiplier). Stores low word to scratch 0x3008, high word to 0x300A. Pushes the middle byte (0x3009) for later.
  3. Base component: Computes 0x0100 - 0x3372 (complement of blend ratio). Multiplies by 0x31D6 (base fuel pulse width). Stores to scratch.
  4. Summation: Pops the corrected component's middle byte, adds to the base component's middle byte. Saturates to 0xFFFF on carry.
  5. Ceiling clamp: Compares against 0x30B0 (airflow output scaled). If result exceeds ceiling, uses ceiling value.
  6. Output: Stores final 16-bit value to 0x313C (engine load). Shifts right 4 to get 8-bit. Switches to 8-bit mode (SEM). Stores to:
    • 0x3745 — diagnostic mirror
    • 0x010C — IO port mirror
    • 0x327C — RAM copy for other subsystems
