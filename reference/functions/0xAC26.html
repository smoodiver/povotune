Summary

Speed pulse counter that accumulates pulses within a fixed 345-tick measurement window. Called every scheduler ISR entry (pre-dispatch). Counts speed sensor edges on alternating ISR ticks using a toggle flag at 0x31A9 bit 1, providing 2:1 dividing of the raw edge rate. When the tick counter (0x3038) reaches 345 (0x0159), the accumulated pulse count (0x3035) is snapshotted to 0x3167 and the counters are reset. The final speed value is written to 0x309B, with a diagnostic override that forces the value to 5 when 0x332F bit 6 is set.

Disassembly

AC26  BBS $0C,#$02,$AC31     ; if dp+$0C bit 1 set, skip to odd-edge path
AC2A  BBC $31A9,#$02,$AC41   ; if toggle flag clear, skip counting
AC2F  BRA $AC36              ; go to increment
AC31  BBS $31A9,#$02,$AC41   ; if toggle flag set, skip counting
AC36  INC $3035              ; increment speed pulse count
AC39  LDA $31A9
AC3C  EOR #$02               ; toggle bit 1
AC3E  STA $31A9
AC41  LDX $3038              ; load tick counter
AC44  INX                    ; increment
AC45  CPX #$0159             ; compare against 345
AC48  BCC $AC57              ; if < 345, skip snapshot
AC4A  LDA $3035              ; snapshot pulse count
AC4D  STA $3167              ; to speed_snapshot
AC50  LDX #$00               ; reset tick counter
AC53  LDM $3035,#$00         ; reset pulse counter
AC57  STX $3038              ; store tick counter
AC5A  BBC $332F,#$40,$AC63   ; if bit 6 clear, use snapshot
AC5F  LDA #$05               ; diagnostic override: force 5
AC61  BRA $AC66
AC63  LDA $3167              ; load speed snapshot
AC66  STA $309B              ; write final speed value
AC69  RTS

Logic

  1. Edge gating (0xAC26–0xAC3E): The function checks dp+$0C bit 1 (IO port edge state) against the toggle flag 0x31A9 bit 1. Only when they disagree does it increment the pulse counter, effectively dividing the edge rate by 2. The toggle flag is flipped after each count.
  2. Window counter (0xAC41–0xAC57): The tick counter 0x3038 is incremented on every call. When it reaches 345 (0x0159), the current pulse count 0x3035 is snapshotted to 0x3167 and both counters reset.
  3. Output (0xAC5A–0xAC69): The final speed value at 0x309B comes from the snapshot 0x3167, unless 0x332F bit 6 is set (diagnostic/test mode), in which case it is forced to 5.
