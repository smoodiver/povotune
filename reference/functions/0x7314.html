0x7314 — Ignition Signal / Decel Detect

Summary

Monitors the ignition signal circuit and sets a fault flag when the ignition signal is absent during conditions where it should be present. First checks two bail-out conditions: if 0x3510 has any bits set (diagnostic mode active, BBC with mask 0xFF), or if 0x3030 bit 7 is already set (stall/decel flag), clears the fault bit and exits.

When conditions are met, checks 0x305C bit 5 to select between two paths that both test Port 0x2000 bit 0 (ignition signal line). If the signal state contradicts expectations, sets 0x3030 bit 7 (stall/decel detected) and clears the fault. If the signal matches expectations, checks 0x3031 bit 3 (engine running confirmation). When engine is running, reads 0x318C (diag_elapsed_timer, 16-bit) and compares against 0x030D (781). If the timer exceeds the threshold, sets 0x332E bit 6 (ignition signal fault).

Inputs

  • 0x3510 (control_reg_3510): must be 0x00 (no diagnostic mode)
  • 0x3030 (throttle_state_flags): bit 7 (stall/decel flag)
  • 0x305C (mode_select_flags): bit 5 (path selection)
  • Port 0x2000: bit 0 (ignition signal)
  • 0x3031: bit 3 (engine running)
  • 0x318C (diag_elapsed_timer): 16-bit, compared to 0x030D

Outputs

  • 0x332E (diag_fault_flags_A): bit 6 set/cleared (ignition signal fault)
  • 0x3030 (throttle_state_flags): bit 7 set (stall/decel detected)

None.
