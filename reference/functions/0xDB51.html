0xDB51 — Ignition Timing Recovery Gate

Field Value
Address 0xDB51
Size 106 bytes (0xDB51–0xDBBA)
Blocks 16
Type subroutine
Callers 1 — scheduler ISR at 0xAAB5
Confidence confirmed

Summary

Controls the slow ignition timing recovery flag ($3276 bit 7) based on a multi-condition qualification window. When all operating conditions are within range and the learned correction counter $3300 is below a threshold, enables slow timing advance recovery. Otherwise disables the recovery path and clears the ramp counter.

The qualification window checks (all must pass to reach the enable path):

  1. $305C bit 5 must be set (engine running in normal mode)
  2. $3092 (coolant temperature) >= ROM $4470 (minimum coolant for recovery)
  3. $306F (airflow load) within ROM range $4473–$4474
  4. $3071 (vehicle speed) within ROM range $4471–$4472
  5. $305A bit 0 must be clear (no stall condition)
  6. $3044 (throttle position) < ROM $4475 (maximum throttle for recovery)
  7. ROM $4477 >= $3301 (learned correction 2 within limit)

When all conditions pass and ROM $4477 >= $3301:

  • Loads ROM $4476 into $309A (overwrites dwell/cranking value with recovery timing base)
  • Clears ramp counter $3132 to 0

After condition checking, compares $3300 (learned correction 1) against ROM $4478:

  • If $3300 < $4478: sets $3276 bit 7 (slow recovery active)
  • Otherwise: clears $3276 bit 7 and clears ramp counter $3132

Finally, calls saturating increment on $3301 (learned correction 2 accumulator).
