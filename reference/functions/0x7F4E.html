0x7F4E — Duty Cycle Complement

Field Value
Address 0x7F4E
Size 41 bytes
Blocks 3
Type subroutine
Callers 1 call site
Confidence likely

Summary

Computes a duty cycle complement by taking the high byte of 0x316A, multiplying by ROM constant 0x401A, subtracting from ROM constant 0x401C, and clamping to zero. Stores the 16-bit result in 0x3056 and the 8-bit result (divided by 8) in 0x3743 and I/O port 0x0106.

Inputs

  • 0x316A (duty_input_value): source value (high byte used)

Outputs

  • 0x3056 (duty_complement_16): 16-bit complement result
  • 0x3743 (duty_complement_8): 8-bit scaled result (÷8)
  • 0x0106 (I/O port): hardware output

RAM Reads

  • 0x316A (duty_input_value)

RAM Writes

  • 0x300C (tracker_lower_bound): used as scratch for multiply intermediate
  • 0x3056 (duty_complement_16)
  • 0x3743 (duty_complement_8)

ROM References

  • 0x401A (duty_scale_factor): multiplication constant
  • 0x401C (duty_max_value): maximum from which product is subtracted
