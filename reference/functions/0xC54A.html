0xC54A — O2 Heater Control

Field Value
Address 0xC54A
Size 90 bytes (0xC54A–0xC5A3)
Blocks 14
Type subroutine
Callers 1 — multi_rate_timer_dispatcher (0xC458)
Confidence confirmed

Summary

Controls the O2 sensor heater relay via bit 4 of port_2000_shadow (0x35A5). Has two operating modes selected by diag_self_test_status ($35A4) bit 1:

Normal mode (bit 1 clear): Computes the elapsed time as ROM[$4630] - ROM[$4631] (= 20 - 10 = 10). Increments a timer counter (0x30D5) via saturating_increment. If elapsed < timer: loads a second threshold from 0x30D6 and compares against ROM[$4631] (10). If elapsed >= threshold, clears bit 4 (heater off); otherwise sets bit 4 (heater on). Writes result to port 0x2000 via port_2000_shadow_write (0x920B). Also checks control_reg_3510 ($3510) for nonzero to optionally skip to the self-test path when map_mode_control bit 2 is set.

Self-test/diagnostic mode (bit 1 set): Increments a diagnostic counter (0x31CE). Every 15 ticks (0x0F), toggles bit 4 of port_2000_shadow via EOR #$10 under interrupt protection, then resets counter to 0. Sets timer 0x30D5 to 0xFF to prevent normal mode from re-triggering.

Inputs

  • 0x35A4 (diag_self_test_status): bit 1 selects operating mode
  • 0x3510 (control_reg_3510): nonzero triggers diagnostic skip
  • 0x3051 (map_mode_control): bit 2 gates diagnostic skip
  • 0x30D5: heater timer counter (saturating increment descriptor)
  • 0x30D6: second threshold descriptor

Outputs

  • 0x35A5 (port_2000_shadow): bit 4 set/cleared/toggled
  • 0x2000 (external I/O port): updated via 0x920B
  • 0x31CE: diagnostic toggle counter
  • 0x30D5: timer counter or set to 0xFF

Calls

  • 0x633D — saturating_increment (up to 2 calls)
  • 0x920B — port_2000_shadow_write

ROM References

  • 0x4630: heater timer upper bound (value = 0x14 = 20)
  • 0x4631: heater timer lower bound / threshold (value = 0x0A = 10)

RAM Reads

  • 0x35A4 (diag_self_test_status)
  • 0x3510 (control_reg_3510)
  • 0x3051 (map_mode_control)
  • 0x30D5: timer counter
  • 0x35A5 (port_2000_shadow)
  • 0x31CE: diagnostic counter

RAM Writes

  • 0x30D5: saturating incremented or set to 0xFF
  • 0x30D6: via saturating_increment
  • 0x35A5 (port_2000_shadow): bit 4 modified
  • 0x2000 (external I/O): via shadow write
  • 0x31CE: incremented or cleared to 0
