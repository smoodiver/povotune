0xA3F4 — RPM Zone Classifier (Phase 1)

Field Value
Address 0xA3F4
Size 84 bytes (0xA3F4–0xA447)
Blocks 14
Type subroutine
Callers 1 — Timer A ISR Phase 1 (0x98A1 at 0x9A21)
Confidence confirmed

Summary

Two-level RPM zone classifier called during ISR Phase 1. Tests the RPM period ($3062) against ROM thresholds with hysteresis to manage two state bits: $31B4 bit 1 (primary RPM zone) and $31B5 bit 4 (secondary RPM zone). When the engine enters the active RPM range, injection timer setup values are loaded. When below the active range, four RAM locations ($31FE–$3201) are zeroed, disabling injection output.

Disassembly

A3F4  LDX $3062           ; load rpm_period
A3F7  BBS $305A,#$04,$A430 ; if system_control_flags bit 4 set, -> inactive
A3FC  BBS $31B4,#$01,$A42B ; if already in primary zone, check exit
; --- entry check ---
A401  CPX $4000           ; compare against primary entry threshold
A404  BCC $A434           ; if period < threshold (too fast), -> inactive path
A406  SEB $31B4,#$01      ; set primary zone active
A40A  BBS $31B5,#$10,$A41A ; if secondary zone already active, check exit
A40F  CPX $46EC           ; compare against secondary entry threshold
A412  BCC $A423           ; if period < threshold, -> enable injection
A414  SEB $31B5,#$10      ; set secondary zone active
A418  BRA $A434           ; -> inactive path
A41A  CPX $46EE           ; compare against secondary exit threshold
A41D  BCS $A434           ; if period >= threshold, -> inactive path
A41F  CLB $31B5,#$10      ; clear secondary zone
; --- enable injection ---
A423  LDM $F6,#$07        ; load injection timer with 0x07
A426  CLB $67,#$0F        ; clear output-complete flags bits 0-3
A429  BRA $A447           ; -> return
; --- exit check ---
A42B  CPX $4002           ; compare against primary exit threshold
A42E  BCS $A40A           ; if period >= threshold (still in zone), check secondary
; --- deactivate ---
A430  CLB $31B4,#$01      ; clear primary zone
A434  LDM $F6,#$00        ; clear injection timer
A437  LDM $31FE,#$00      ; clear injection workspace 0
A43B  LDM $31FF,#$00      ; clear injection workspace 1
A43F  LDM $3200,#$00      ; clear injection workspace 2
A443  LDM $3201,#$00      ; clear injection workspace 3
A447  RTS

Inputs

  • $3062 (rpm_period) — current RPM period

Outputs

  • $31B4 bit 1 — primary RPM zone active flag
  • $31B5 bit 4 — secondary RPM zone active flag
  • $F6 — injection timer (direct page), set to 0x07 when entering active zone or 0x00 when exiting
  • $67 — output-complete flags (direct page), bits 0–3 cleared on zone entry
