0x7928 — Fault Code Bit Scanner

Field Value
Address 0x7928
Size ~127 bytes (0x7928–0x79A6)
Blocks 25
Type subroutine
Callers 2 (MIL lamp driver 0x77E6 at 0x78A8 and 0x78E3)
Confidence confirmed

Summary

Scans a 4-byte fault flag register (pointed to by X) to find the next set bit, using a rotating bit position counter (0x3194) and a walking bitmask (0x3195). When a set bit is found, looks up the corresponding DTC code from a ROM table at 0x79A7 and stores it in 0x3192 for the blink pattern generator to output.

The scanner implements a round-robin search across 32 bit positions (4 bytes x 8 bits). On each call:

  1. Increments the bit position counter (0x3194), wrapping from 0x1F back to 0x00
  2. Shifts the walking bitmask (0x3195) left by one position (ASL with ROL for carry propagation)
  3. At each byte boundary (positions 0, 8, 16, 24), loads the corresponding byte from the fault register (X+0 through X+3) and checks if any bits are set
  4. ANDs the walking bitmask with the current fault byte
  5. If a match is found, uses the bit position as an index into the DTC lookup table at 0x79A7
  6. The table entry becomes the DTC code stored in 0x3192

The byte boundary detection at 0x7941–0x7961 handles the case where position counter is exactly at a boundary (0, 8, 16, 24). The mid-byte path at 0x7963–0x7976 handles positions between boundaries by comparing against 0x08, 0x10, 0x18 to select the correct byte.

When no match is found (AND result = 0), the function loops back to 0x7928 to try the next bit position.

DTC Table (0x79A7–0x79C6): 32 bytes mapping bit positions to DTC codes. The scanner scans registers 0x332E (byte 0), 0x332F (byte 1), 0x3330 (byte 2), 0x3331 (byte 3). Bit 0 of each byte maps to the lowest position in that group.

Inputs

  • X register: pointer to 4-byte fault flag register (0x332E for test mode, 0x350A for clear-codes mode)
  • 0x3194: current bit position counter (0–31, wrapping)
  • 0x3195: walking bitmask

Outputs

  • 0x3194: updated bit position counter
  • 0x3195: updated walking bitmask
  • 0x3192: DTC code from lookup table (for blink pattern generator)

ROM References

  • 0x79A7–0x79C6: 32-byte DTC code lookup table (fully populated — see table above)
