0xA9ED — Ignition Timing Output Calculator

Field Value
Address 0xA9ED
Size 228 bytes (0xA9ED–0xAAD0)
Blocks 40
Type subroutine
Callers 1 — Timer A ISR secondary chain (0x98A1 region at 0x9BF2)
Confidence confirmed

Summary

Computes the final ignition timing output value ($30BA) from the dwell start angle ($304F), applies knock correction offsets, rate-limiting (slew control), and stores the smoothed result to $309A (dwell_cranking). Then calls three sub-functions to compute derived timing outputs for the ignition system.

The function has four main paths:

  1. System error ($3021 bit 7 set): Clears af_ratio_mode and jumps to output.
  2. Knock correction fully enabled ($31C2 = 0xFF): Clears af_ratio_mode, subtracts a ROM offset from dwell angle, clamps to zero.
  3. Active knock processing: Evaluates multiple preconditions (ign_control bits, zone flags, event flags), subtracts timing offsets, applies knock-direction-dependent correction.
  4. Fallback: Subtracts base timing offset, clamps to zero.

After computing the raw output, applies rate-limiting (slew control) against the previous value ($309A), using ROM step limits ($4480, $4481) and a slow-ramp counter ($3132/$4479) gated by $3276 bit 7. The final value is clamped to a maximum of 0x3C (60 degrees), stored to three outputs, and three sub-functions are called.

Key Computation Path (Active Knock Processing)

A9ED  LDA $304F           ; load dwell_start_angle
A9F0  BBS $3021,#$80,$AA5C ; if system error, -> clear and output
; ...precondition checks...
AA1E  LDB $322C           ; check event flags
AA22  ANDB #$24           ; bits 2+5 must be clear
AA27  LDB $31CC           ; af_ratio_mode must be 0
; ...
AA32  SEC
AA33  SBC $31B9           ; subtract timing_advance_offset
AA36  BCC $AA43           ; clamp to 0 on underflow
AA3A  LDB $31B8           ; load timing_correction (signed)
AA3E  BPL $AA45           ; if positive, skip extra subtraction
AA40  DEA                 ; subtract 1 (signed adjustment)
; ...
AA45  BBC $31FC,#$40,$AA52 ; if knock feedback not enabled, skip
AA4A  SEC
AA4B  SBC $445C           ; subtract ROM knock offset
AA4E  BCS $AA52           ; clamp to 0
AA50  LDA #$00
; ...
AA52  BBC $31CC,#$FF,$AA60 ; if af_ratio_mode not active, skip
AA57  DEC $31CC           ; decrement af_ratio_mode counter

Rate Limiting (0xAA60–0xAAB2)

AA60  STA $30BA           ; store raw output
AA63  SEC
AA64  SBC $309A           ; compare against previous smoothed value
AA67  BCS $AA7E           ; if new >= old (increasing), -> ramp up
; --- decreasing ---
AA69  EOR #$FF            ; negate difference
AA6B  INA
AA6C  CMP $4481           ; compare against max decrease step
AA6F  BCC $AAAF           ; if within limit, use raw value
AA71  LDA $309A           ; else: old - step
AA74  SEC
AA75  SBC $4481
AA78  BCS $AAB2
AA7A  LDA #$00            ; clamp to 0
; --- increasing ---
AA7E  BBS $3276,#$80,$AA95 ; if slow-ramp mode, -> slow path
AA83  CMP $4480           ; compare against max increase step
AA86  BCC $AAAF           ; if within limit, use raw value
AA88  LDA $309A           ; else: old + step
AA8B  CLC
AA8C  ADC $4480
AA8F  BCC $AAB2
AA91  LDA #$3C            ; clamp to max (60)
; --- slow ramp ---
AA95  LDA $309A           ; load previous value
AA98  INC $3132           ; increment slow-ramp counter
AA9B  LDB $3132
AA9F  CMPB $4479          ; compare against slow-ramp threshold
AAA3  BCC $AAB2           ; if not reached, hold previous value
AAA5  LDM $3132,#$00      ; reset counter
AAA9  INA                 ; increment by 1
AAAA  CMP $30BA           ; compare against raw target
AAAD  BCC $AAB2           ; if still below target, use incremented
AAAF  LDA $30BA           ; else use raw value
AAB2  STA $309A           ; store smoothed result

Final Output (0xAAB5–0xAAD0)

AAB5  JSR $DB51           ; sub_DB51 (timing conversion 1)
AAB8  STA $3278           ; store converted value
AABB  JSR $D90A           ; sub_D90A (timing conversion 2)
AABE  JSR $DBBB           ; sub_DBBB (timing conversion 3)
AAC1  CMP #$3C            ; clamp to max 60
AAC3  BCC $AAC7
AAC5  LDA #$3C
AAC7  STA $3050           ; store to dwell_output
AACA  STA $01A4           ; mirror to diagnostic output
AACD  STA $3763           ; store to timing_output_mirror
AAD0  RTS
