0xAFF5 — Throttle Rate-of-Change Processor

Field Value
Address 0xAFF5
Size 361 bytes (0xAFF5–0xB15D)
Blocks 41
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 at 0xAB4D)
Callees 2 — adc_dual_capture (0xAF02), graph_lookup_2d (0x6000)
Confidence confirmed

Summary

Processes throttle position rate-of-change data to compute a fuel correction factor. Called from scheduler priority 4 (1/20 ticks). First calls adc_dual_capture (0xAF02) to acquire fresh ADC readings, then classifies the throttle rate into one of 6 severity bands using a cascade of ROM threshold comparisons. The band index selects a fuel enrichment multiplier from a ROM table, which is then combined with coolant temperature compensation and an RPM-based scaling factor to produce a 16-bit fuel correction value stored at 0x30CA. Also handles a deceleration path that computes an alternative correction using a separate ROM table lookup. A delta-based safety check at the end can zero out the correction if the throttle position change between successive samples exceeds a ROM threshold.

Algorithm Detail

  1. ADC Capture: Calls 0xAF02 (adc_dual_capture) to read fresh throttle position
  2. Stall check: If $305A bit 4 is set, clears $314F and zeros $3094, jumps to exit
  3. Rate classification (acceleration path):
    • Computes index into circular throttle history buffer at $3044+offset (offset from ROM $4070)
    • Subtracts historical throttle value from current $3044
    • If delta is positive and exceeds threshold $4075:
      • Classifies delta against ROM thresholds $4072/$4073/$4074 (ascending) to produce band B = 0..3
    • If delta is negative or below $4075: switches to deceleration path
  4. Rate classification (deceleration path):
    • Uses offset from ROM $4071 and threshold $4076/$4077
    • Produces band B = 4 or 5
  5. Band processing:
    • Clamps combined band to 0..5, stores to $3749 (diagnostic mirror) and IO $0128
    • If band <= ROM $4D70: loads $4D71 into $31CC (A/F ratio mode control)
    • Adds $30C7 (temp_zone_index) to band, uses as index into ROM table at $4220
    • Loads enrichment factor from table into $3009
  6. Fuel correction computation:
    • Reads $3008 (16-bit), shifts right 1 (divide by 2)
    • Multiplies by $3181 using MPY hardware multiply
    • Stores high byte result back to $3008
    • Looks up coolant correction from ROM table $4380 using $306F via 1D interpolation (JSR $6000)
    • Stores correction to $300B
    • Multiplies $300A by $3008, extracts upper bits
    • Saturates to 0xFFFF if overflow, otherwise shifts left 2 to produce 16-bit correction
  7. Deceleration alternative: If on decel path, computes from $313C via ROM table $4350 (JSR $6000), stores $3182, forces band=6 in diagnostic mirrors, zeros the 16-bit correction
  8. Table selection: Selects between ROM tables $4270 and $4260 based on $305C bit 4, indexes by $30C7/4 to get $30CC
  9. Delta safety check: Computes |$3044 - $3046|; if exceeds ROM $4029, zeros $3094 and skips update
  10. Output clamping: If $3023 bit 1 is set, skips the max-update. Otherwise updates $3094 only if new value >= old

Inputs

  • dp+$20/$22 — ADC control/result registers (passed to adc_dual_capture)
  • $305A — system flags (bit 4: stall/cranking state)
  • $3044 — current throttle position (16-bit)
  • $3046 — previous throttle position
  • $30C7 — temperature zone index
  • $306F — coolant temperature derivative
  • $3181 — RPM-based scaling factor (16-bit)
  • $313C — deceleration input value
  • $3023 — control flags (bit 1: skip max-update gate)
  • $305C — mode flags (bit 4: table selection)
  • $314F — stall persistence counter

Outputs

  • $30CA — 16-bit fuel correction value (acceleration enrichment / deceleration enleanment)
  • $30CC — temperature-compensated enrichment base
  • $3094 — clamped correction output (max of successive values unless zeroed)
  • $3095 — mirrored to IO $012C at exit
  • $3009 — enrichment factor from ROM table $4220
  • $300B — coolant correction factor
  • $3182 — deceleration correction (from $4350 table)
  • $3749 — diagnostic mirror of rate band (0–6)
  • $31CC — A/F ratio mode flag (set when band <= $4D70 threshold)
  • $314F — cleared on stall detect
