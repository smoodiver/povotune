0xB2F7 — Engine Load Calculator

Type subroutine (ISR callee)
Callers 1 — scheduler P5 (0xAAD1 at 0xABA7)

Confidence confirmed

Summary

Computes the engine load value from an ADC reading using a hardware multiply, IIR smoothing, and a 6-deep circular history buffer for rate-of-change detection. Called from scheduler priority 5 (1/20 ticks). Reads ADC result, applies a scaling factor via MPY to convert raw ADC counts to a calibrated load value, subtracts a ROM offset, then applies multi-stage exponential smoothing controlled by a ROM iteration count. Maintains a 6-byte circular history buffer for computing a signed rate-of-change metric, which is itself smoothed by the IIR filter. Writes the final 8-bit load value to $3304, IO port $0224, and diagnostic mirror $3792.

Algorithm Detail

  1. ADC read: BBC $20,#$08,$B2F7 — spin until conversion complete. Read 16-bit result from dp+$22.
  2. Raw storage: Store 16-bit to $3308 (aux sensor input). Shift right 2, store 8-bit to $3327.
  3. Scaling: Multiply $3308 by constant 0x6AA8 using MPY. Extract upper word: ASL A, ROL B, XAB (take high 16 bits).
  4. Offset subtract: Subtract ROM constant $486B. Clamp to 0 if underflow.
  5. Exponential smoothing:
    • If $3166 bit 4 not set (first time): skip smoothing, use raw value
    • If $3331 bit 4 set: force value to 0x0119 (override)
    • Load iteration count B from ROM $4826 (masked to 8-bit)
    • Loop B times: A = (A + $3306) / 2 (CLC; ADC $3306; ROR A; DEB; BNE loop)
    • This is a multi-pass averaging filter with exponentially increasing weight on old value
  6. Store smoothed value: Write 16-bit to $3306
  7. 8-bit conversion: Shift right 1 (LSR A). Clamp to 0xFF if >= 0x0100.
  8. Output: Store 8-bit to $3304 (engine_load_value), IO $0224, $3792 (diagnostic mirror)
  9. History buffer initialization (first time, $3166 bit 4 not set):
    • Fill all 6 bytes $333C–$3341 with current value
    • Set $3166 bit 4
  10. History buffer shift (subsequent calls):
    • Shift bytes down: [$3340→$3341, $333F→$3340, ..., $333D→$333E, $333C→$333D]
    • Store current value to $333C (newest)
  11. Rate-of-change:
    • Compute delta = $333C - $3340 (newest minus 6-samples-ago)
    • Add signed offset 0x80 to center around midpoint
    • Clamp to 0x00–0xFF range
  12. Rate smoothing: X = $3342, call IIR filter (0xB2E0) to smooth the rate value
  13. Rate output: Store to $3342 and IO $01D6

Inputs

  • dp+$20 — ADC status register (bit 3: conversion complete)
  • dp+$22 — ADC result (16-bit)
  • $3166 — init flags (bit 4: first-time initialization)
  • $3331 — control flags (bit 4: force override to 0x0119)
  • $3306 — previous smoothed load value (16-bit)

Outputs

  • $3308 — raw 16-bit ADC result
  • $3327 — raw 8-bit ADC result (>> 2)
  • $3306 — smoothed 16-bit load value
  • $3304 — final 8-bit engine load value
  • $3792 — 8-bit diagnostic mirror of engine load
  • $333C–$3341 — 6-byte circular history buffer
  • $3342 — smoothed rate-of-change value (centered at 0x80)
  • $3166 — bit 4 set after initialization
