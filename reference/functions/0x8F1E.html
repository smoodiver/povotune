0x8F1E — RPM Rate-of-Change Classifier

Field Value
Address 0x8F1E
Size 46 bytes (0x8F1E–0x8F4B)
Blocks 7
Type subroutine
Callers 1 call site (sensor processing at 0xBDD7)
Confidence confirmed

Summary

Classifies the RPM rate of change into three states based on the difference between current RPM period (0x306C) and previous RPM period (0x3064). Computes delta = 0x306C - 0x3064 (positive means RPM is increasing, since these are timer periods that decrease with rising RPM — but the subtraction here is reversed, so positive delta means deceleration). Sets state flags in 0x305F bits 1 and 3:

  • No acceleration (delta negative or zero): clears both bits 1 and 3
  • High rate (delta >= 0x100 or >= ROM[0x4B0E]): sets bit 1, leaves bit 3 alone
  • Moderate rate (delta >= ROM[0x4B0F] but < ROM[0x4B0E]): clears bit 1, sets bit 3

Disassembly

8F1E  CLM                    ; 16-bit mode
8F1F  LDA $306C              ; load current RPM period
8F22  SEC
8F23  SBC $3064              ; delta = current - previous
8F26  BCC $8F38              ; if negative (RPM rising), no accel flags
8F28  CMP #$0100             ; compare delta vs 0x100
8F2B  SEM                    ; 8-bit mode
8F2C  BCS $8F3E              ; if >= 0x100, high rate
8F2E  CMP $4B0E              ; compare vs high threshold
8F31  BCS $8F3E              ; if >= threshold, high rate
8F33  CMP $4B0F              ; compare vs moderate threshold
8F36  BCS $8F43              ; if >= threshold, moderate rate
; --- No acceleration ---
8F38  SEM                    ; 8-bit mode
8F39  CLB $305F,#$0A         ; clear bits 1 and 3
8F3D  RTS
; --- High rate ---
8F3E  SEB $305F,#$02         ; set bit 1 (high rate)
8F42  RTS
; --- Moderate rate ---
8F43  CLB $305F,#$02         ; clear bit 1
8F47  SEB $305F,#$08         ; set bit 3 (moderate rate)
8F4B  RTS

Inputs

  • 0x306C: current RPM period (16-bit timer value)
  • 0x3064 (rpm_full_precision): previous RPM period

Outputs

  • 0x305F bits 1, 3: rate-of-change classification flags:
    • bits 1=0, 3=0: no deceleration / RPM rising
    • bit 1=1: high deceleration rate
    • bit 3=1, bit 1=0: moderate deceleration rate

RAM Reads

  • 0x306C, 0x3064

RAM Writes

  • 0x305F (bits 1 and 3)

ROM References

  • 0x4B0E -- high rate threshold (8-bit, compared after SEM)
  • 0x4B0F -- moderate rate threshold (8-bit)
