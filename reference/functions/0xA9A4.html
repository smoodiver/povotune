0xA9A4 — Knock Timing Interval Counter

Field Value
Address 0xA9A4
Size 73 bytes (0xA9A4–0xA9EC)
Blocks 12
Type subroutine
Callers 1 — Background scheduler ISR bit 5 handler (0xAAD1 at 0xABAA)
Confidence confirmed

Summary

Periodic counter function called from the background scheduler. Manages two counters:

  1. Decay counter ($31C5): Incremented each call. When it reaches the ROM threshold ($4D96), resets to zero and performs a timing correction update: if the ISR is in the high-RPM zone ($31A1 bit 2), subtracts a ROM decay step ($4D94) from the 16-bit timing correction ($31B8), clamping to zero. The result is mirrored to $01B8.

  2. Timing interval counter ($31BA): Incremented each call under SEI. When it reaches the ROM threshold ($4D9F), resets to zero and sets $3033 bit 4 (timing interval flag), signaling the knock_learning_update (0xA826) that an interval has elapsed.

Disassembly

; --- decay counter ---
A9A4  INC $31C5           ; increment decay counter
A9A7  LDA $31C5
A9AA  CMP $4D96           ; compare against threshold
A9AD  BCC $A9D7           ; if not reached, skip
A9AF  LDM $31C5,#$00      ; reset counter
A9B3  SEI                 ; disable interrupts
A9B4  BBC $31A1,#$02,$A9CE ; if not in high-RPM zone, -> passthrough
; --- active decay (in RPM zone) ---
A9B9  CLM                 ; 16-bit mode
A9BA  LDA $31B8           ; load timing_correction (16-bit)
A9BD  BEQ $A9CB           ; if already 0, skip
A9BF  SEC
A9C0  SBC $4D94           ; subtract decay step
A9C3  BCS $A9C8           ; if no underflow, keep
A9C5  LDA #$0000          ; clamp to 0
A9C8  STA $31B8           ; store updated correction
A9CB  CLI                 ; re-enable interrupts
A9CC  BRA $A9D3           ; -> mirror output
; --- passthrough (not in RPM zone) ---
A9CE  CLM                 ; 16-bit mode
A9CF  LDA $31B8           ; just load current value
A9D2  CLI
A9D3  STA $01B8           ; mirror to diagnostic output
A9D6  SEM                 ; back to 8-bit
; --- timing interval counter ---
A9D7  SEI                 ; disable interrupts
A9D8  INC $31BA           ; increment interval counter
A9DB  LDA $31BA
A9DE  CMP $4D9F           ; compare against threshold
A9E1  BCC $A9EB           ; if not reached, skip
A9E3  LDM $31BA,#$00      ; reset counter
A9E7  SEB $3033,#$10      ; set timing interval flag
A9EB  CLI                 ; re-enable interrupts
A9EC  RTS
