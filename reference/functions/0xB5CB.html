0xB5CB — Acceleration Enrichment Calculator

Field Value
Address 0xB5CB
Size 196 bytes (0xB5CB–0xB68E)
Blocks 21
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 via 0xAB53)
Callees JSR 0x6000 (1D table interpolation), JSR 0x6307 (shift left B:A by 2)
Confidence confirmed

Summary

Computes acceleration enrichment (fuel adder) based on the fuel map index and engine load. This function determines how much extra fuel to inject during throttle tip-in events by looking up a per-cylinder enrichment duration from a ROM table indexed by fuel map selection, then computing an enrichment amount using a 1D table lookup scaled by engine load. The result is stored to 0x312A (injection enrichment adder) and mirrored to IO 0x018C. On first entry after a tip-in, it initializes the 4-cylinder enrichment countdown timers (0x31D2–0x31D5) from a ROM constant.

Flow

  1. Mode gate: Checks 0x31B4 bits 0+1 (both must be set = 0x03). If not both set, clears 0x31D2–0x31D5 (all 4 enrichment countdowns) and 0x018C, returns. This requires the engine to be in a running state with active injection.
  2. RPM ceiling: If 0x306E >= ROM 0x4402 (0x50 = 80), clears all enrichment and returns. Above this RPM threshold, acceleration enrichment is disabled.
  3. First-entry detection: Checks all four countdown timers (0x31D2–0x31D5). If all are zero and 0x3749 (fuel map raw) < 6, sets 0x3020 bit 7 (enrichment init request flag).
  4. Fuel map index: Reads 0x3749 (fuel map raw), clamps to max 5. Uses this as an index into ROM table at 0x4280 (6-entry byte table: 0x20, 0x1E, 0x1C, 0x00, 0x00, 0x00). Stores the per-map enrichment duration to 0x312D.
  5. Load-based enrichment amount: Takes 0x313C (engine load, 16-bit), shifts right 3 to get 8-bit (clamped to 0xFF). Uses this as input to 1D table lookup at ROM 0x4290, storing result to 0x312E.
  6. Enrichment computation: Multiplies 0x312D * 0x312E, stores intermediate to scratch. Then multiplies the high byte of 0x312E (fractional part) by the scratch result, and — if 0x302A bit 3 is set — applies an additional correction factor from ROM 0x4405 (0x95 = 149) via JSR 0x6307 (multiply by 4). XAB gives the final enrichment value.
  7. Output: Stores result to 0x312A (injection enrichment adder, 16-bit). Shifts right 4 to get 8-bit, stores to IO 0x018C.
  8. Countdown initialization: If 0x3020 bit 7 is set (from step 3), loads ROM 0x4403 (0x01) and initializes all four countdown timers (0x31D2–0x31D5) to this value. Clears 0x3020 bit 7.
