0x6C2B — Idle/Load Range Check

Field Value
Address 0x6C2B
Size 75 bytes
Blocks 17
Type subroutine
Callers 1 (health_check_dispatcher 0x6B14)
Confidence confirmed

Summary

Checks whether the engine is operating within a valid idle/load range by validating coolant temperature, RPM, and load values against expected bounds. If the coolant warmup flag (0x3330 bit 2) or the prior starter fault flag (0x3330 bit 0) are not set, clears the fault and returns. Otherwise validates: coolant temp (0x3092 >= 0x3C), RPM (0x3062 in range 0x0020..0x0167), and load (0x303E in range 0x003C..0x03E8). If any value is out of range, resets the counter and exits. If all checks pass, increments the debounce counter at 0x317C via saturating_increment (0x633D). Once the counter reaches 2, sets the idle control fault flag (0x332F bit 2).

Inputs

  • 0x3330 bit 2: coolant sensor warmup qualified (from 0x6C1A)
  • 0x3330 bit 0: starter/battery fault flag (from 0x6BC1)
  • 0x332F bit 0: VSS/idle combined fault (inhibits check)
  • 0x3092: coolant temperature (raw)
  • 0x3062: RPM (timer period, 16-bit)
  • 0x303E: load variable (16-bit)

Outputs

  • 0x332F bit 2: idle control debounce fault flag
  • 0x317C: fault debounce counter (reset to 0 or incremented)

Calls

  • 0x633D: saturating_increment (debounce counter at 0x317C)

RAM Reads

  • 0x3330, 0x332F, 0x3092, 0x3062, 0x303E

RAM Writes

  • 0x332F, 0x317C
