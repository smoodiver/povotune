0x922F — AC Relay Controller

Field Value
Address 0x922F
Size 468 bytes
Blocks 81
Type subroutine
Callers 1 call site
Confidence confirmed

Summary

AC compressor clutch relay controller via bit 6 of port_control_mirror (0x3275). Uses multiple hysteresis comparators on airflow, coolant temperature, engine load, and vehicle speed with several delay timers to prevent rapid relay cycling. In diagnostic mode, toggles the relay on a timed cycle. In normal mode, engages the AC compressor when all conditions are satisfied: sufficient airflow, adequate coolant temp, proper engine load (with speed-dependent threshold from table 0x4690), and all delay timers expired. Controls bit 6 of port_control_mirror, flushed to I/O 0x2200 via 0x9214.

Inputs

  • 0x306F (airflow_load): airflow hysteresis
  • 0x3092 (coolant_temp_raw): temperature hysteresis
  • 0x3044 (adj_map_input): engine load for engage/disengage
  • 0x3071 (vehicle_speed): speed-dependent threshold lookup
  • 0x305A (system_control_flags): bits 1,2,4 gate logic

Outputs

  • 0x3210 (ac_condition_flags): hysteresis state bits
  • 0x3282 (ac_engage_flags): engage state
  • 0x3275 (port_control_mirror): bit 6 updated
  • I/O port 0x2200: via 0x9214

Calls

  • 0x6000 — graph_lookup_2d: speed-to-threshold lookup (table 0x4690)
  • 0x9214 — port_2200_write: flushes port_control_mirror to hardware

ROM References

  • 0x4690 (ac_speed_lookup_table): speed-dependent cutoff threshold
  • 0x46B2–0x46CC: delay timers and hysteresis thresholds
