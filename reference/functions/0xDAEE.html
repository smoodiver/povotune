0xDAEE — Knock Feedback Delta Monitor

Field Value
Address 0xDAEE
Size 99 bytes (0xDAEE–0xDB50)
Blocks 13
Type subroutine
Callers 1 — scheduler ISR at 0xAC0A
Confidence confirmed

Summary

Monitors the rate of change of knock feedback correction values using two sliding-window buffers, and flags rapid-change conditions via $3276 bit 5. Also maintains a saturating counter at $322D and an accumulator at $31FD that advances when $3276 bit 6 is clear.

The function operates two parallel delta-check paths:

  1. Path A (indexed at $327C): computes current - previous from a 3-entry history buffer at $327C–$327F. If $3276 bit 1 is clear and the delta exceeds the ROM threshold at $4443, sets $3276 bit 5 (rapid knock change flag).
  2. Path B (indexed at $326E): computes previous - current from a 3-entry history buffer at $326E–$3271. If $3276 bit 1 is clear and the delta exceeds the ROM threshold at $4444, also sets $3276 bit 5.

Both deltas are biased by +0x80 (signed-to-unsigned offset) before being stored as the newest history entry. The history buffers shift older values down each call (FIFO).
