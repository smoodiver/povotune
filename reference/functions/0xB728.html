0xB728 — Base Fuel Pulse Width Calculator

Field Value
Address 0xB728
Size 160 bytes (0xB728–0xB7C7)
Blocks 19
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 via 0xAB56)
Callees JSR 0x6307 (shift left B:A by 2), JSR 0x6304 (shift left B:A by 3)
Confidence confirmed

Summary

Computes the base fuel injection pulse width by multiplying the airflow measurement by a volumetric efficiency scaling factor. The function has two paths depending on whether a 16-bit extended airflow value is available (0x320C != 0): a simple 8-bit path and a 16-bit extended-precision path. Both paths multiply the airflow value by ROM constant 0x400A (0x9353), apply a correction divisor selected from either 0x301A or 0x3023 (gated by 0x332F bit 2), and store the result to 0x31D6 (base injection pulse width). The result feeds directly into the fuel correction blender at 0xB7C8.

Flow — 8-Bit Path (0x320C == 0)

  1. Reads 0x320C with interrupts disabled (SEI). If zero, takes the simple path.
  2. Reads 0x320A (8-bit airflow measurement) with CLI to re-enable interrupts.
  3. Multiplies by ROM 0x400A (0x9353) via MPY, XAB to get result in B.
  4. Selects divisor: if 0x332F bit 2 set, uses 0x3023 >> 1 (high byte shifted right); otherwise uses 0x301A directly. Stores to 0x301C.
  5. Multiplies airflow*scaling by the divisor (MPY 0x301C).
  6. If high word (B) >= 0x4000, saturates to 0xFFFF. Otherwise, shifts left 2 (JSR 0x6307 = multiply by 4).
  7. Stores B to 0x31D6 (base fuel pulse width). SEM, RTS.

Flow — 16-Bit Path (0x320C != 0)

  1. Reads 0x3209:0x320B as a 24-bit value (0x3209 in scratch, 0x320B as high word).
  2. Performs 5 iterations of ASL/ROL to shift the 24-bit value left by 5, effectively multiplying by 32. Saturates to 0xFFFF on overflow.
  3. Re-enables interrupts (CLI).
  4. Multiplies by ROM 0x400A (0x9353), XAB.
  5. Same divisor selection as 8-bit path (0x332F bit 2 gates 0x301A vs 0x3023).
  6. Multiplies by divisor. If high word (B) >= 0x0800, saturates to 0xFFFF. Otherwise, shifts left 5 (JSR 0x6304 then JSR 0x6307 = multiply by 32).
  7. Stores B to 0x31D6. SEM, RTS.
