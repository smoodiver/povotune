0x9481 — Adaptive Learning

Field Value
Address 0x9481
Size 486 bytes
Blocks 58
Type subroutine
Callers 1 call site
Confidence confirmed

Summary

EEPROM-backed adaptive learning function. In normal mode, performs an 8×8 bilinear interpolation of learned values stored in RAM at 0x35B0 (EEPROM mirror) using RPM and engine_load through axes 0x4680/0x4688. In speed-limited mode, reads directly from stored calibration values at 0x351A. The learning engine tracks min/max values, counts events, and when sufficient stable observations accumulate, updates the learned values in the 0x35B0 map. Multiple conditions gate learning: warm engine, stable mode, correct operating regime.

Inputs

  • 0x305A (system_control_flags): bits 1,3,4 gate various paths
  • 0x35A4 (diag_self_test_status): bit 0 disables learning
  • 0x306F (airflow_load): speed-limited path threshold
  • 0x3062 (rpm): map lookup RPM axis
  • 0x313C (engine_load): map lookup load axis
  • 0x3092 (coolant_temp_raw): warm-enough check
  • 0x3028: interrupt-protected learning input
  • 0x3040: interrupt-protected AFM input
  • 0x30A7 (alt_mode_index): current operating mode

Outputs

  • 0x30A6 (startup_control): current map lookup result or 0x80 (disabled)
  • 0x3752: mirror of startup_control
  • 0x35B0 (EEPROM map): learned values updated when conditions met
  • 0x302B (learn_control_flags): state tracking
  • 0x30A9 (learn_event_counter): event count
  • 0x30AD, 0x30AE (learn_min/max): observation window

Calls

  • 0x608D — graph_lookup_3d (alternate entry): bilinear interpolation of learned values map
  • 0x96AF — adaptive_learning_integrator: integrates observation into EEPROM workspace

ROM References

  • 0x4680 (learn_rpm_axis): RPM axis
  • 0x4688 (learn_load_axis): load axis
  • 0x4648, 0x464A (learn_thresholds): comparison thresholds
  • 0x464C (learn_temp_threshold): minimum temperature
  • 0x4659 (learn_airflow_threshold): airflow path selector
