cross-reference Heritage Warning

This function is labeled "Rear O2 Sensor Processor" in the cross-reference SVX cross-reference. On the BD5A (EJ20H twin turbo), the ADC channel feeding this function (AN5, read by 0xB3A9) is connected to a turbo system feedback sensor, not a rear O2 sensor. The output (0x32AE) serves as the proportional controller feedback for the twin turbo duty controller (0xCC9F). The exact physical sensor identity (likely a pressure transducer related to the twin turbo exhaust routing) needs board-level pin trace verification. The sub-calls to 0x6E1D and 0x6E38 are generic sensor range validators that may have been repurposed from O2 checking on the SVX to turbo sensor checking on the BD5A.

Summary

Turbo feedback sensor acquisition and processing. This large function has two major branches controlled by 0x35A4 bit 0: when set (secondary mode), it performs a quick output-port update and returns; when clear (normal mode), it runs the full sensor processing pipeline. The normal path increments sample counters, waits for a threshold number of samples, computes a nonlinear voltage conversion, accumulates a sensor voltage integrator, and — once sufficient samples are collected — produces an averaged reading that feeds the twin turbo duty controller's closed-loop proportional controller as feedback (0x32AE). It also manages the transition from open-loop to closed-loop feedback based on RPM and sensor readiness thresholds.

Flow — Short Path (0x35A4 bit 0 set)

  1. Updates output port shadow: copies 0x35A4 bit 7 to 0x35A5 bit 2 (controls an external IO line).
  2. Writes 0x35A5 to IO port 0x2000 (atomic SEI/CLI bracketed).
  3. Computes 0x3360 >> 2 and stores to 0x376A (throttle position 8-bit mirror).
  4. Returns.

Flow — Normal Path (0x35A4 bit 0 clear)

  1. Counter management: If 0x3051 bit 2 is set, clears 0x31DF bit 2. Increments 0x32A6 (sample counter, saturating at 0xFF) and 0x32B7 (ISC activity counter, saturating at 0xFF, compared against 0x48C6 threshold).

  2. Early exit: If 0x32B7 < 0x48C6 (10), returns — not enough samples yet.

  3. Branching on 0x31DF bit 3:

    • Bit 3 set (closed-loop active): Performs nonlinear scaling of 0x3360 via (0xA6AA * [0x3360]) << 1 - 0x01D5, clamped 0–0xFF. Stores to 0x32A2 (turbo feedback voltage scaled). Increments 0x32A3 (integration counter, saturating at 0xFF). Accumulates 0x32A2 into 16-bit integrator 0x32A4. When counter reaches ROM threshold (0x48CE=5 normal, 0x48CF=3 for advanced mode gated by 0x31DF bit 4), computes averaged turbo feedback reading 0x32A4 / 0x32A3, updates 0x31FB/0x32C6 (turbo feedback reference/current values with intermittent fault tracking via 0x31EA bit 2). Calls JSR 0x6E1D (sensor range check). If 0x332E bit 4 is set (fault flag), overrides with 0xF0. Stores final turbo feedback value to 0x3279 (turbo speed threshold). Resets counters and clears output port bit, clears 0x31DF bit 3.
    • Bit 3 clear (open-loop): Falls through to averaging path at 0xB544.

  4. Averaging path (0xB544): Performs iterative averaging of 0x3362 (ISC scaled sample) with 0x32B8 using count iterations (ROM 0x48CD = 5): avg = (avg + new) ROR repeated. If 0x332E bit 4 set, clamps to 0x0550. Stores to 0x32B8 (sensor accumulator). Shifts right 3, clamps to 0xFF, stores to 0x32AE (turbo duty feedback), IO 0x01CC, and 0x3781.

  5. Open-to-closed-loop transition: Two-stage qualification gated by 0x31DF bit 2:

    • Stage 1 (bit 2 clear): Requires 0x306E >= ROM 0x48C7 (0x0C = RPM threshold). Sets 0x31DF bit 2, clears bit 4.
    • Stage 2 (bit 2 set): Requires 0x306E >= ROM 0x48C8 (0x28), 0x32B8 >= ROM 0x48C9 (0x0C03), and 0x32A6 >= ROM 0x48CB (0xAA44). When all met, sets 0x31DF bit 4 (advanced mode). Sets 0x35A5 bit 2, writes to IO 0x2000. Sets 0x31DF bit 3 (closed-loop active). Resets 0x32B7 and 0x32A6. Clears 0x3025 bit 7.

  6. Always: Calls JSR 0x6E38 (turbo feedback sensor range check) before returning.
