Summary

Per-cylinder injection timing calculator for channel 1. One of four identical-structure sub-functions (0x9E1E, 0x9F41, 0xA064, 0xA187) that compute injection timing parameters and arm the hardware timer for a specific injector channel. Called in sequence by the injection dispatch chain (0x9E11).

The function operates in three modes based on $31B4 (injection state) and $31B5 (output channel flags):

  1. Cranking mode ($31B4 bit 0 clear): Loads a fixed cranking pulse width from $30BE, sets $3004 = 0x0A (fixed dwell), and jumps to timer arm.
  2. Re-fire mode ($31B5 bit 4 set): Checks if this is the correct cylinder for re-fire ($31A1 bit 7 and $319C == 1), loads $3170 as pulse width, fixed dwell 0x0A, arms timer.
  3. Running mode (normal): Computes injection pulse width from base value $3170 (with optional enrichment $312A added if $31D2 != 0 and $31E1 == 0), then enters the sequencing state machine.

Running Mode State Machine

Reads direct-page $68 bits 4–5 to determine current timer state:

  • State 0x20 (active injection): Checks $3218 (injection step counter) against expected values with cylinder-specific gating via $31A1 bit 2 and $319C. If conditions met, enters re-fire path. Otherwise clears $31B5 bit 0 and returns.
  • State 0x30 (post-injection): Decrements countdown timers $31E1 and $31D2. Indexes into ROM table at $A2B9 (base) using $319C as cylinder selector. Loads per-cylinder timing compare value, compares against $31FE. Updates $3218 and $3197 accordingly.
  • Default (idle): Loads $3197 directly, jumps to timer calculation.

Timer Arm Sequence (0x9F0F–0x9F40)

  1. Stores injection schedule value to $31FE, decrements $3197.
  2. Calls $A2CE (injection period scaling) to compute final period, stores to $3004.
  3. Tests direct-page $0D bit 0 and $301F bit 3 for abort conditions.
  4. Loads $63 = 0xFE, clears $62 bit 0.
  5. Writes $3004 to direct-page $40 (timer period), $3006 to $42 (timer compare).
  6. Writes 0x02 to $68 (timer control), sets $62 bit 0, clears $63 to 0.

Inputs

  • $3170 — base injection pulse width (channel 1)
  • $312A — injection enrichment adder
  • $31D2 — enrichment countdown (channel 1)
  • $31E1 — enrichment delay counter
  • $31B4 — injection sequencing state
  • $31B5 — output channel flags (bits 0, 1, 4)
  • $3197 — injection schedule (channel 1)
  • $319C — cylinder counter
  • $31A1 — engine running flags
  • $31FE — timer compare reference (channel 1)
  • $3218 — injection step counter (channel 1)
  • $30BE — cranking injection pulse width
  • $35A4 — alternate mode flags
  • $3574 — alternate injection pulse width
  • $68 — direct-page timer control (bits 4–5: state)

Outputs

  • $40 — timer period register (channel 1)
  • $42 — timer compare register (channel 1)
  • $68 — timer control register (channel 1)
  • $62 / $63 — timer enable/status
  • $31FE — updated timing compare value
  • $3197 — decremented injection schedule
  • $3218 — updated/cleared injection step counter
  • $31B5 — output channel flags updated
  • $31D2 — enrichment countdown decremented
  • $31E1 — enrichment delay decremented
  • $3004 — computed injection period (scratch)
  • $3006 — injection pulse width (scratch)
