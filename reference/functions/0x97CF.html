Summary

Restores or initializes diagnostic state during ECU reset, depending on whether the EEPROM learning system was previously initialized (0x3030 bit 6). This function has two paths:

Warm-start path (bit 6 set — EEPROM valid): Restores previously saved diagnostic state from EEPROM-backed variables into the active diagnostic registers. Copies diagnostic counters (0x3506 to 0x332E, 0x3508 to 0x3330), diagnostic init state (0x3504 to 0x3032), and knock/sensor diagnostic bits (0x3512 to 0x31EB, 0x3513 to 0x31F8). Clears 0x3331 bit 5. Calls diagnostic_mode_initializer (0x6B89) to set up diagnostic subsystem.

Cold-start path (bit 6 clear — fresh EEPROM): Determines diagnostic mode from 0x305C bit 1 (mode_select_flags): if set, stores 0xFF to 0x3510; if clear, stores 0x00. Clears diagnostic counters 0x3506, 0x3508, 0x350A, 0x350C to zero.

Common tail (0x981A): Sets MIL blink timer (0x3190) to 0x23 and MIL scan bitmask (0x3195) to 0x01. Then checks AC/ignition input state: if 0x305C bit 5 is set, tests port 0x2000 bit 0 and sets 0x3031 bit 3 if clear; if bit 5 is clear, tests port 0x2000 bit 0 and sets 0x3031 bit 3 if set. Finally, if diagnostic mode is active (0x3510 == 0xFF), sets 0x31CF bits 4,2,7 (pending faults mask 0x94) and clears 0x332F bits 5,6 and 0x332E bit 6.

Inputs

  • 0x3030 (throttle_state_flags): bit 6 — EEPROM initialized flag
  • 0x305C (mode_select_flags): bit 1 — diagnostic mode select; bit 5 — AC/ignition polarity
  • 0x2000 (IO port): bit 0 — ignition/accessory input state
  • 0x3504 (diag_init_state_1): saved diagnostic state (warm-start)
  • 0x3506 (diag_counter_1): saved counter (warm-start)
  • 0x3508 (diag_counter_2): saved counter (warm-start)
  • 0x3512 (diag_init_state_2): saved knock sensor bits (warm-start)
  • 0x3513 (diag_init_state_3): saved sensor status bits (warm-start)

Outputs

  • 0x332E (diag_fault_flags_A): restored from 0x3506 (warm) or unchanged; bit 6 cleared if diag mode
  • 0x3330: restored from 0x3508 (warm) or unchanged
  • 0x3032: restored from 0x3504 (warm) or unchanged
  • 0x31EB (diag_knock_sensor_bits): restored from 0x3512 (warm) or unchanged
  • 0x31F8 (o2_sensor_status): restored from 0x3513 (warm) or unchanged
  • 0x3331: bit 5 cleared (warm path)
  • 0x3510 (control_reg_3510): set to 0xFF (diag) or 0x00 (normal) on cold path
  • 0x3506, 0x3508, 0x350A, 0x350C: zeroed on cold path
  • 0x3190 (mil_blink_timer): set to 0x23
  • 0x3195 (mil_scan_bitmask): set to 0x01
  • 0x3031: bit 3 set conditionally (ignition detect)
  • 0x31CF (diag_pending_faults): bits 4,2,7 set if diagnostic mode
  • 0x332F (temp_override_flags): bits 5,6 cleared if diagnostic mode
  • 0x332E (diag_fault_flags_A): bit 6 cleared if diagnostic mode

Calls

  • 0x6B89 (diagnostic_mode_initializer): warm-start path only
