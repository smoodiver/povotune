0xA2CE — Injection Period Scaling

Field Value

Size 155 bytes (0xA2CE–0xA368)
Blocks 15
Type subroutine
Callers 4 — 0x9F15 (ch1), 0xA018 (ch2), 0xA13B (ch3), 0xA27E (ch4)
Confidence confirmed

Summary

Common injection helper called by all four injection timing calculators to convert a schedule index into a hardware timer period. Takes the schedule value in A, performs two multiply-and-scale passes using engine timing data and ROM constants, then applies timing compensation and clamping to produce the final injection timer period.

Algorithm

  1. Input normalization: Clears M flag (16-bit mode), masks A to 8-bit, pushes schedule value to stack.

  2. First multiply pass (dwell offset calculation):

    • Loads $4016 (timer capture register / engine period reference).
    • Selects timing multiplier based on $31A1 bit 2: uses $31AE (timing_result_A) or $31B2 (timing_result_B).
    • Multiplies via MPY instruction, then calls $A3C1 (shift right B:A by 4, i.e. divide by 16).
    • Adds injection pulse width from $3006, stores 32-bit result to $3004/$3002 (high/low).

  3. Second multiply pass (injection end-point calculation):

    • Recovers schedule value from stack, doubles it (ASL A), adds ROM table base $A368 to form index.
    • Loads 16-bit ROM constant from table at $A368 + (schedule * 2).
    • Multiplies by same timing multiplier ($31AE or $31B2 per $31A1 bit 2).
    • Calls $A3C1 (divide by 16), stores result to $3260/$3262.

  4. Period calculation:

    • Subtracts first-pass result ($3002/$3004) from second-pass result.
    • If result is negative (borrow), clamps to minimum 0x14 (20).

  5. Timing compensation (positive result path):

    • Sets SEM (8-bit mode), configures direct-page control registers $FE, $B4 (timer sequencing flags).
    • Reads direct-page $A0 (current timer position), computes compensation offset: ($A0 >> 4) + 0x0C.
    • Subtracts compensation from period result.
    • If high byte nonzero after subtraction, saturates to 0xFFFF.
    • Clamps minimum to 0x14 (20 timer ticks).

Inputs

  • A — schedule index (0–4, cylinder-dependent)
  • $4016 — timer capture / engine period reference
  • $31AE — timing_result_A (multiplier for $31A1 bit 2 set)
  • $31B2 — timing_result_B (multiplier for $31A1 bit 2 clear)
  • $31A1 — engine running flags (bit 2 selects multiplier)
  • $3006 — injection pulse width (from caller)
  • $A0 — direct-page timer position (for compensation)
  • $A368 — ROM scaling constant table base

Outputs

  • A — computed timer period (16-bit, minimum 0x14)
  • $3002 — intermediate dwell+pulse result (low word)
  • $3004 — intermediate result (high byte)
  • $3260 — injection endpoint high byte
  • $3262 — injection endpoint low word
  • $FE — timer sequencing control
  • $B4 — timer channel flags (bits 2, 4 toggled)
