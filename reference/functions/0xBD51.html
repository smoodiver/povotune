0xBD51 — RPM Processor

Field Value
Address 0xBD51
Size 138 bytes (0xBD51–0xBDDA)
Blocks 11
Type subroutine (ISR callee)
Callers 1 — scheduler P6 (0xAAD1 at 0xAB80)
Confidence confirmed

Summary

Converts raw timer-derived RPM data into multiple scaled RPM representations used throughout the ECU, maintains a 5-entry RPM history FIFO, and calls the RPM rate-of-change classifier. The function has three phases:

  1. Timer-to-RPM conversion (0xBD51–0xBDA0): Atomically reads (SEI/CLI) the ISR-produced timer period values $320A (B register) and $320C (A register, low byte only). If the low byte of $320C >= 5, clamps the output to a fixed value of 0x0008 (very high RPM / overflow). Otherwise, performs a 24-bit right-shift-by-3 of the B:A pair using LSR A / ROR $3008 three times, then applies a constant mask: LDB #$04, LDA #$93E0, AND $3008, producing a scaled RPM value. Pushes this value into a 5-entry 16-bit FIFO at $3064–$306C (with interrupt protection via SEI/CLI), and stores a right-shifted-by-2 copy to $3062 (the standard RPM axis value used by table lookups).

  2. Multi-resolution RPM outputs (0xBDA5–0xBDD4): Produces three 8-bit-clamped RPM values at different resolutions:

    • $3070 = RPM >> 2, clamped to 0xFF (airflow rate)
    • $306F = RPM >> 3, clamped to 0xFF (airflow load), also written to SSM port $012A and $3778
    • $306E = RPM >> 4, clamped to 0xFF (knock sensor resolution)

  3. Rate-of-change (0xBDD7): Calls JSR $8F1E (RPM rate-of-change classifier) to classify the current RPM acceleration/deceleration state.

Inputs

  • $320A (derived_rpm_low): ISR-produced timer period value (low word)
  • $320C: ISR-produced timer period value (high byte, only low byte used)

Outputs

  • $3062 (rpm): standard RPM axis value (full precision >> 2)
  • $3064 (rpm_full_precision): full-precision 16-bit RPM (FIFO head)
  • $3066–$306C: RPM FIFO entries [1]–[4] (16-bit each)
  • $306E (knock_sensor): RPM >> 4, 8-bit clamped
  • $306F (airflow_load): RPM >> 3, 8-bit clamped
  • $3070 (airflow_rate): RPM >> 2, 8-bit clamped
  • $012A: SSM diagnostic — airflow load RPM
  • $3778: mirror of airflow load RPM
