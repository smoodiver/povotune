0x7714 — Fault Output Stage

Field Value
Address 0x7714
Size ~210 bytes (0x7714–0x77E5)
Blocks 24
Type subroutine
Callers 1 (health_check_dispatcher 0x6B14, slot 24)
Confidence confirmed

Summary

Aggregates per-sensor fault flags from the diagnostic scan cycle into latched fault registers and distributes them to I/O output ports and RAM mirrors for the SSM diagnostic interface. Also manages a diagnostic elapsed timer and a "persistent fault with cleared codes" detection mechanism.

The function has four phases:

Phase 1 — Snapshot current state (0x7714–0x7731): Copies current fault flags (0x332E, 0x3330) to diagnostic snapshot registers (0x3506, 0x3508). Copies engine state (0x3032) to 0x3504 and status bytes (0x31EB, 0x31F8) to 0x3512/0x3513.

Phase 2 — Fault latching (0x7732–0x779B): Branches on 0x3510 (diagnostic mode control):

  • Normal mode (0x3510 = 0): Switches to 8-bit mode (CLM), ORs current fault flags into latched registers (0x350A |= 0x332E, 0x350C |= 0x3330). Resets elapsed timer (0x318C = 0). If airflow (0x306F) >= 0x10, increments persistent fault counter (0x3216) with saturation.
  • Test mode (0x3510 != 0): Increments elapsed timer (0x318C) if airflow >= 0x28. Checks if faults have cleared AND various engine conditions are met (0x3030 bits 2,3,7 set, 0x31EA bit 7 set, 0x305C bit 2 set). If all clear conditions met, zeros the latched fault registers (0x350A = 0, 0x350C = 0).

Phase 3 — Persistent fault detection (0x779D–0x77AB): If elapsed timer < 0x030D (781) and faults are present, sets 0x378F bit 7 (persistent fault flag). Otherwise clears it.

Phase 4 — Output distribution (0x77AC–0x77E5): Copies fault data to I/O output ports and RAM mirrors:

  • 0x332E → 0x01AC + 0x3787 (current faults A)
  • 0x3330 → 0x01AE + 0x3789 (current faults B)
  • 0x350A → 0x01B0 + 0x378B (latched faults A)
  • 0x350C → 0x01B2 + 0x378D (latched faults B) Finally sets 0x31EA bit 6 if either 0x332F bits 2+3 or 0x3330 bits 3-6 are set, indicating active diagnostic fault condition.

Inputs

  • 0x332E, 0x3330: current scan-cycle fault flags
  • 0x332F, 0x3031: control/status flags
  • 0x3032: engine state
  • 0x31EB, 0x31F8: status registers
  • 0x306E, 0x306F: airflow values
  • 0x3030: engine condition flags (bits 2,3,7)
  • 0x31EA bit 7: intermittent fault detection ready
  • 0x305C bit 2: fault clear gate
  • 0x3510: diagnostic mode control

Outputs

  • 0x3504, 0x3506, 0x3508: diagnostic state snapshots
  • 0x3512, 0x3513: status snapshots
  • 0x350A, 0x350C: latched fault accumulator registers (OR-accumulate in normal mode)
  • 0x318C: diagnostic elapsed timer
  • 0x3216: persistent fault counter
  • 0x01AC, 0x01AE, 0x01B0, 0x01B2: I/O output ports (SSM parameter block)
  • 0x3787–0x378D: RAM mirrors of fault outputs
  • 0x378F bit 7: persistent fault detected flag
  • 0x31EA bit 6: active fault aggregate flag
