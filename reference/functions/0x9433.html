0x9433 — Limiter Output Controller

Field Value
Address 0x9433
Size 78 bytes
Blocks 16
Type subroutine
Callers 1 call site
Confidence likely

Summary

Controls bit 0 of port_2000_shadow (0x35A5) based on two parallel hysteresis comparators. Each compares a limiter input value (0x32E0, 0x3314) against ROM thresholds (0x46C5/0x46C3). If map_mode_control bit 2 is set and both comparators indicate the condition is met, the output is cleared; otherwise it is set. Skips all checks when self-test is active.

Inputs

  • 0x35A4 (diag_self_test_status): bit 0 bypasses to return
  • 0x32E0 (limiter_input_0): first comparator input
  • 0x3314 (limiter_input_1): second comparator input
  • 0x302C (engine_status_flags): bits 1,2 hysteresis state
  • 0x3051 (map_mode_control): bit 2 gates the logic

Outputs

  • 0x302C (engine_status_flags): bits 1,2 updated
  • 0x35A5 (port_2000_shadow): bit 0 set/cleared

ROM References

  • 0x46C5 (limiter_threshold_high): default threshold
  • 0x46C3 (limiter_threshold_low): alternate threshold
