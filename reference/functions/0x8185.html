0x8185 — Ignition Output Controller

Field Value
Address 0x8185
Size 337 bytes (0x8185–0x82D5)
Blocks 53
Type subroutine
Callers 1 call site (main loop at 0x680C)
Confidence confirmed

Summary

Top-level ignition controller that selects between three operating modes (cranking, speed-limited, normal), computes the coil dwell start angle, and manages knock learning resets. In normal mode, calls ignition_timing (0x82D6) for spark angle and knock_retard_calc (0x85DB) for the knock-proportional dwell reduction, then computes: dwell_start = spark_angle + dwell_duration(coolant_temp) - knock_reduction - corrections. The result at 0x304F tells the hardware timer when to begin charging the coilpack relative to TDC. Also responsible for clearing the knock learned values table (0x3630–0x366F) when zone conditions require a reset.

Inputs

  • 0x31A1 bit 1: engine running flag — if clear, use cranking mode
  • 0x305A bit 4: speed limiting active — if set, use speed-limited mode
  • 0x302A bit 3: speed limit enforced (from speed_control)
  • 0x3093: coolant temperature (processed AD value)
  • 0x31BB: final ignition timing angle (from ignition_timing)
  • 0x330F: knock-proportional reduction (from knock_retard_calc)
  • 0x3518: correction term (capped at 5)
  • 0x3326: correction term
  • 0x31B6: raw timing value (used in bypass mode)
  • 0x3033 bit 3: knock learning reset trigger
  • ROM[0x448F]: bypass flag — if 0xFF, skip normal calculation and use raw 0x31B6

Outputs

  • 0x304F: dwell start angle — when coilpack charging begins relative to TDC
  • IO 0x019C: dwell start angle mirror (diagnostic output)
  • 0x309A: dwell value copy (set during cranking/speed-limit modes)
  • 0x3021 bit 7: cranking/abnormal mode flag
  • 0x3500: copy of 0x3033 (zone control flags, under interrupt protection)
  • 0x3501: copy of 0x3034 (ignition control flags, under interrupt protection)
  • 0x3630–0x366F: knock learned values table (zeroed when 0x3033 bit 3 set)

Calls

  • 0x82D6 — ignition_timing: computes spark angle → 0x31BB (normal mode only)
  • 0x85DB — knock_retard_calc: computes knock-proportional reduction → 0x330F (normal mode only)
  • 0x6000 — graph_lookup_2d: called 1x in normal mode (coolant temp → dwell), 3x in speed-limited mode

ROM References

  • 0x44F0 (coolant_temp_to_dwell): 16-entry table, values 20 to 13, more dwell when cold
  • 0x4485 (fixed_dwell_speed_cut): fixed dwell for speed-limited + speed-cut-active mode
  • 0x448A (max_dwell_speed_limited): max dwell clamp in speed-limited mode
  • 0x448B (max_dwell_normal): max dwell clamp in normal mode
  • 0x448D (coolant_temp_threshold_dwell): coolant temp threshold for speed-limited dwell
  • 0x448C (vehicle_speed_threshold_dwell): vehicle speed threshold for speed-limited dwell
  • 0x448F (timing_bypass_flag): 0xFF = use raw 0x31B6 value
  • 0x44A0 (rpm_to_dwell_speed_limited): RPM-to-dwell table in speed-limited mode
  • 0x4490 (load_to_correction_speed_limited): load-to-correction table in speed-limited mode
  • 0x4B90 (coolant_temp_to_dwell_speed_limited): coolant-temp-to-dwell table in speed-limited mode
