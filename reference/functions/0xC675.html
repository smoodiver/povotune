0xC675 — Coolant Fan Relay Qualifier

Field Value
Address 0xC675
Size 176 bytes (0xC675–0xC724)
Blocks 35
Type subroutine
Callers 1 — scheduler P1 (0xAAD1 at 0xABF2)
Confidence confirmed

Summary

Multi-input hysteresis comparator that qualifies whether the coolant fan relay should be engaged. Evaluates four sensor inputs against ROM-defined thresholds with individual hysteresis bits, then combines the results to drive port 0x2200 bit 1 and the fan relay command flag.

Entry gate: Returns immediately if diag_self_test_status ($35A4) bit 0 is set (self-test active), or if map_mode_control ($3051) bit 2 is set (engine running in certain mode), or if control_reg_3510 ($3510) is 0xFF. Also requires a warmup timer ($316C) > 0x14 (20).

Comparator channels (each uses hysteresis via bits in $323F):

Channel Input Bit Set Threshold Clear Threshold
Airflow 0x306F (airflow_load) bit 1 ($323F) ROM[$4875] = 60 ROM[$4874] = 56
Throttle 0x3044 (throttle_position) bit 2 ($323F) ROM[$4877] = 40 ROM[$4876] = 32
Limiter 0 0x32E0 (limiter_input_0) bit 3 ($323F) ROM[$487E] = 132 ROM[$487C] = 66
Limiter 1 0x3314 (limiter_input_1) bit 5 ($323F) ROM[$487E] = 132 ROM[$487C] = 66

Output logic: If any of bits 1, 2, 3, 5 (mask 0x2E) of $323F are set AND temp_override_flags ($332F) bit 7 is clear: sets port_control_mirror ($3275) bit 1. Then checks external port ($2000) bit 5 to determine whether to set $323F bit 0 as a feedback flag. If none of the comparator bits are set: clears $3275 bit 1 and $323F bit 0.

Final output written to port 0x2200 via port_2200_shadow_write (0x9214).

Inputs

  • 0x35A4 (diag_self_test_status): bit 0 bypasses
  • 0x3051 (map_mode_control): bit 2 bypasses
  • 0x3510 (control_reg_3510): 0xFF bypasses
  • 0x316C: warmup timer (must exceed 20)
  • 0x306F (airflow_load): comparator channel 1
  • 0x3044 (throttle_position): comparator channel 2
  • 0x32E0 (limiter_input_0): comparator channel 3
  • 0x3314 (limiter_input_1): comparator channel 4
  • 0x323F: hysteresis state register
  • 0x332F (temp_override_flags): bit 7 inhibits output
  • 0x2000 (external I/O port): bit 5 read for feedback

Outputs

  • 0x323F: hysteresis bits 0–5 updated
  • 0x3275 (port_control_mirror): bit 1 set/cleared
  • 0x2200 (external I/O port): updated via 0x9214

Calls

  • 0x9214 — port_2200_shadow_write

ROM References

  • 0x4874: airflow clear threshold (value = 0x38 = 56)
  • 0x4875: airflow set threshold (value = 0x3C = 60)
  • 0x4876: throttle clear threshold (value = 0x20 = 32)
  • 0x4877: throttle set threshold (value = 0x28 = 40)
  • 0x487C: limiter clear threshold (value = 0x42 = 66, 16-bit compare)
  • 0x487E: limiter set threshold (value = 0x84 = 132, 16-bit compare)
