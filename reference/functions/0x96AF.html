Summary

Integrates the current learning observation into the EEPROM-backed adaptive learning workspace. This is the core accumulator for the adaptive learning system: it takes the current observation value, computes a signed delta from center (0x8000), scales it by a rate-of-change factor selected from a 3-tier speed bracket, and adds/subtracts the scaled result to/from the current workspace cell. The workspace cell is clamped against per-mode upper and lower bounds stored in ROM.

The function operates in two distinct phases:

  1. Scaling phase (0x96AF-0x96ED): Subtracts 0x8000 from the input (A), takes the absolute value, stores the magnitude in scratch 0x300C. Then selects a rate multiplier from a 3-level bracket table (0x4652+N indexed by 0x30A7) based on the current speed value in 0x300D against three thresholds at ROM 0x4663/0x4665/0x4667. Multiplies magnitude by the selected rate byte (high byte of result), producing a scaled delta. A zero result is bumped to 1 (minimum step).

  2. Accumulation phase (0x96ED-0x9790): Tests flag 0x302B bit 3 to select workspace addressing mode. In the normal mode (bit 3 clear), uses 0x30A7 as an index into the EEPROM learn table base at 0x35B0. In the speed-limited mode (bit 3 set), uses fixed addresses 0x351A/0x351C selected by 0x302B bit 4. Then adds or subtracts the scaled delta depending on the carry flag (original sign of the observation - 0x8000 subtraction). The result is clamped to ROM upper bounds at 0x4EF8/0x4EF9/0x4EFA (per-mode, selected by 0x302B bits 3,4) and lower-bounded by the Y-indexed floor value from the ROM table at 0x4EB6. The final value is stored back to the workspace cell and mirrored to 0x3752.

Inputs

  • A (accumulator): current observation value (16-bit, centered around 0x8000)
  • 0x300D (temp_scaled_hi): current speed/load for rate bracket selection
  • 0x30A7 (alt_mode_index): operating mode index (selects workspace cell and ROM floor table entry)
  • 0x302B (interp_mode_control): bit 3 = speed-limited mode, bit 4 = secondary address select

Outputs

  • Workspace cell at 0x35B0+offset or 0x351A/0x351C: updated learning value
  • 0x300C/0x300D (scratch): residual scaled delta (B:A)
  • 0x3752: mirror of final accumulated value

ROM References

  • 0x4652 (learn_rate_base): base address for 3-byte rate multiplier table (indexed by 0x30A7)
  • 0x4663 (learn_speed_bracket_3): highest speed threshold
  • 0x4665 (learn_speed_bracket_2): middle speed threshold
  • 0x4667 (learn_speed_bracket_1): lowest speed threshold
  • 0x4EB6 (learn_floor_table): Y-indexed floor values per operating mode (index from 0x30A7 >> 1)
  • 0x4EF8 (learn_ceiling_mode_0): upper clamp for normal mode (bit 3 clear)
  • 0x4EF9 (learn_ceiling_mode_1): upper clamp for speed-limited mode, bit 4 clear
  • 0x4EFA (learn_ceiling_mode_2): upper clamp for speed-limited mode, bit 4 set
