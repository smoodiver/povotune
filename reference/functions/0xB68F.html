0xB68F — Fuel Correction Blender

Field Value
Address 0xB68F
Size 153 bytes (0xB68F–0xB727)
Blocks 16
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 via 0xAB59)
Callees JSR 0x6000 (1D table interpolation), JSR 0x63E0 (16-bit table interpolation)
Confidence confirmed

Summary

Computes a composite fuel correction factor by blending three separate lookups: an RPM-based correction, an O2-feedback-based correction, and a load-based correction from a 16-bit table. These are combined through multiplication and scaling to produce a final fuel multiplier stored to 0x3378 (16-bit) and mirrored to IO 0x0124. This function also produces intermediate outputs at 0x3372 (blend ratio, 16-bit), 0x3374 (O2 correction factor), 0x3376 (load correction), and IO 0x0120/0x0122.

Flow

  1. Precondition check: Reads 0x30B6 (engine run counter, 16-bit) and compares against ROM 0x4710 (0x0500 = 1280). If below, skips to zero-output path. Also checks 0x3092 against ROM 0x4712 (0x64 = 100) — must be strictly greater. Finally checks 0x332E bit 4 — if set (sensor fault), skips to zero-output.
  2. RPM-based blend ratio: Reads 0x3064 (RPM value), subtracts 0xC0 (clamped to 0), doubles it (ASL), clamps to 0xFF. Uses this as input to 1D table lookup at ROM 0x42A0. Stores result to 0x3008 (scratch). This produces a blend ratio that decreases at higher RPM.
  3. Battery/alternator factor: Reads 0x3007, masks high byte, uses as a secondary factor. Combines with blend ratio via AND to produce 0x3372 (blend ratio output, 16-bit). Also stores 0x3008 to IO 0x0120.
  4. O2 correction factor: Reads 0x3279 (O2 feedback value from 0xB44B). Uses as input to 1D table lookup at ROM 0x42B0. Stores result to 0x3374.
  5. Load correction factor: Reads 0x3370 (ISC average from 0xB3A9). Uses as input to 16-bit table interpolation at ROM 0x5500 (with ROM pointer 0x300A = 0x5500). Stores result to 0x3376 and IO 0x0122.
  6. Final multiplier: Takes high byte of 0x3373 (fractional part of O2 correction), multiplies by 0x3376 (load correction). Doubles the result (ASL/ROL) with saturation to 0xFFFF. Stores to 0x3378 (16-bit fuel correction multiplier). Shifts right 3 to get 8-bit, stores to IO 0x0124.
  7. Zero-output path: If preconditions fail, sets 0x3008 = 0 and 0x3372 = 0 (16-bit), then falls through to the O2 and load lookups (which will produce baseline values).
