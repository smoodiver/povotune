0xC5A4 — Coolant Fan Speed Controller

Field Value
Address 0xC5A4
Size 147 bytes (0xC5A4–0xC636)
Blocks 27
Type subroutine
Callers 1 — scheduler P5 (0xAAD1 at 0xABAD)
Confidence confirmed

Summary

Controls the coolant fan speed output (0x325D) based on knock sensor reading (0x306E), RPM (0x303C), and additional sensor conditions. Uses an adaptive threshold system with hysteresis to ramp the fan speed up or down.

Entry gate: Checks knock_sensor ($306E). If >= 0x28 (40) or < ROM[$488D] (10), clears fan_speed_output (0x325D) to 0 and returns.

Engine running check: Requires system_control_flags ($305A) bit 3 set (engine running) AND bit 2 clear. Also requires lookup_input_b ($316B) > 0x64 (100). If any condition fails, sets output to 0.

RPM-based control (using ad_initial_ch1 at $303C as 16-bit):

  • If RPM < 0x0028: below minimum, loads base thresholds from ROM and returns 0
  • If RPM >= 0x0028 and < 0x03E0: in active range, runs adaptive controller
  • If RPM >= 0x03E0: high RPM, checks sensor_status_flags and runs comparator

Adaptive controller: Maintains two 16-bit threshold values:

  • 0x357A (adaptive_base_value): lower threshold
  • 0x32AA: upper threshold (base + ROM[$4888] offset of 30)

Base is initialized from ROM[$4886] (100). When RPM crosses thresholds, the output counter increments or resets. A secondary qualifier checks system_control_flags bit 0, knock_sensor bounds (ROM[$488C] = 20 upper, 0x0A lower), and coolant_temp_raw ($3092) against ROM[$488B] (130) to conditionally lower the threshold.

Inputs

  • 0x306E (knock_sensor): primary input, must be in valid range
  • 0x303C (ad_initial_ch1): RPM input (16-bit)
  • 0x305A (system_control_flags): bits 0, 2, 3 gate operation
  • 0x316B (lookup_input_b): must exceed 100
  • 0x31CD (sensor_status_flags): bit 5 gates high-RPM path
  • 0x3092 (coolant_temp_raw): secondary qualifier

Outputs

  • 0x325D (fan_speed_output): 8-bit output level (0 = off)
  • 0x357A (adaptive_base_value): lower RPM threshold
  • 0x32AA: upper RPM threshold

ROM References

  • 0x4886: base RPM threshold (value = 0x64 = 100)
  • 0x4888: RPM threshold offset (value = 0x1E = 30)
  • 0x488A: output increment limit (value = 0x0A = 10)
  • 0x488B: coolant temp qualifier (value = 0x82 = 130)
  • 0x488C: knock sensor upper bound (value = 0x14 = 20)
  • 0x488D: knock sensor minimum (value = 0x0A = 10)

RAM Reads

  • 0x306E (knock_sensor), 0x303C (ad_initial_ch1), 0x305A (system_control_flags)
  • 0x316B (lookup_input_b), 0x31CD (sensor_status_flags), 0x3092 (coolant_temp_raw)
  • 0x325D (fan_speed_output), 0x357A (adaptive_base_value), 0x32AA

RAM Writes

  • 0x325D (fan_speed_output): output level
  • 0x357A (adaptive_base_value): threshold updated
  • 0x32AA: upper threshold updated
