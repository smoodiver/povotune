0xA826 — Knock Learning Update

Field Value
Address 0xA826
Size 146 bytes (0xA826–0xA8B7)
Blocks 19
Type subroutine
Callers 1 — knock_control_dispatcher (0xA683 at 0xA6B7)
Confidence confirmed

Summary

Updates the per-cylinder knock learning table based on current knock state. Called when the knock retard ramp has found the knock boundary (ign_zone_control bit 6 set). The function has three operating modes:

  1. Knock correction fully enabled ($31C2 = 0xFF): Clears the timing interval counter, sets ign_control bit 3, loads B=0, stores cylinder ID to diagnostic mirror. Minimal update.
  2. Sensor active ($31C1 nonzero): Performs a ROM table lookup using airflow load ($306F) against table at $44B0 to compute a knock correction base value ($31BD). Then indexes into the per-cylinder knock learning RAM table at $3630 + cylinder_id to read the current learning value, adds the correction, passes through the knock clamp function (0xA8B8), and writes the result back.
  3. Normal path: Checks if the cylinder has changed since last update, waits for the knock timing interval flag, and conditionally loads a ROM default correction value from $4D9E.
