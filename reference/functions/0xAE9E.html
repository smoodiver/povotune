0xAE9E — Cranking Initialization

Field Value
Address 0xAE9E
Size 100 bytes (0xAE9E–0xAF01)
Blocks 13
Type subroutine
Callers 1 — Timer A ISR (0x98A1) at 0x9A27
Callees 0
Confidence confirmed

Summary

One-shot initialization routine that loads cranking injection parameters into all four cylinder registers when the engine transitions from stall to cranking. Called from the Timer A ISR (crank/cam timing handler), not from the scheduler ISR. The function gates on multiple preconditions — cranking-active flag, engine phase detection, and per-cylinder threshold checks — and only fires once per cranking event. When triggered, it loads the boost startup value ($3130) into all 4 cylinder injection base registers, zeros the fractional accumulators, sets initial dwell times, and enables injection outputs for all cylinders.

Detailed Flow

Gate checks (0xAE9E–0xAECC):

AE9E  BBS $31B4,#$03,$AEA5 ; if engine phase bit 3 set → proceed
AEA3  BRA $AF01            ; else → return (not in correct phase)
AEA5  BBC $3020,#$08,$AF01 ; if cranking-active not set → return
AEAA  LDA #$01
AEAC  CMP $3197            ; compare 1 against cylinder 1 threshold
AEAF  BCS $AF01            ; if 1 >= threshold → return (already initialized)
AEB1  BBS $0D,#$01,$AF01   ; if IO port bit 0 set → cyl 1 already firing
AEB5  CMP $3198            ; cylinder 2 threshold
AEB8  BCS $AF01
AEBA  BBS $0D,#$02,$AF01   ; IO port bit 1 = cyl 2
AEBE  CMP $3199            ; cylinder 3 threshold
AEC1  BCS $AF01
AEC3  BBS $0D,#$04,$AF01   ; IO port bit 2 = cyl 3
AEC7  CMP $319A            ; cylinder 4 threshold
AECA  BCS $AF01
AECC  BBS $0D,#$08,$AF01   ; IO port bit 3 = cyl 4

The function requires ALL of the following:

  1. $31B4 bit 3 set (engine phase/position detected)
  2. $3020 bit 3 set (cranking-active, from stall/cranking detector)
  3. All four cylinder thresholds ($3197–$319A) must be > 1 (not yet initialized)
  4. None of the IO port $0D bits 0–3 set (no cylinder already firing)

If any condition fails, the function returns immediately (15 possible early-exit paths).

Initialization (0xAED0–0xAEFE):

AED0  LDA $43F6           ; load cranking dwell initial value from ROM
AED3  STA $31E1           ; store as current dwell value
AED6  LDX $3130           ; load boost_startup_value (from stall/cranking detector)
AED9  STX $40             ; cylinder 1 injection base (dp)
AEDB  STX $44             ; cylinder 2 injection base
AEDD  STX $48             ; cylinder 3 injection base
AEDF  STX $4C             ; cylinder 4 injection base
AEE1  LDX #$0000
AEE4  STX $42             ; cylinder 1 injection fraction = 0
AEE6  STX $46             ; cylinder 2 injection fraction = 0
AEE8  STX $4A             ; cylinder 3 injection fraction = 0
AEEA  STX $4E             ; cylinder 4 injection fraction = 0
AEEC  CLB $3020,#$08      ; clear cranking-active flag (consumed)
AEF0  SEB $301F,#$08      ; set cranking-initialized flag
AEF4  LDA #$0C            ; initial dwell count value
AEF6  STA $68             ; cylinder 1 dwell counter
AEF8  STA $69             ; cylinder 2 dwell counter
AEFA  STA $6A             ; cylinder 3 dwell counter
AEFC  STA $6B             ; cylinder 4 dwell counter
AEFE  SEB $62,#$0F        ; enable injection outputs for all 4 cylinders
AF01  RTS

Inputs

Source Address Description
RAM $31B4 Engine phase flags — bit 3 = position detected
RAM $3020 Engine state flags — bit 3 = cranking active
RAM $3197 Cylinder 1 event threshold
RAM $3198 Cylinder 2 event threshold
RAM $3199 Cylinder 3 event threshold
RAM $319A Cylinder 4 event threshold
IO $0D Injection output port — bits 0–3 = cylinder firing status
RAM $3130 Boost startup value (from stall/cranking detector 0xAE20)
ROM $43F6 Initial dwell value for cranking

Outputs

Address Description
$31E1 Current dwell value (loaded from ROM $43F6)
dp $40/$44/$48/$4C Cylinder 1–4 injection base values (loaded from $3130)
dp $42/$46/$4A/$4E Cylinder 1–4 injection fractions (zeroed)
$3020 Bit 3 cleared (cranking-active consumed)
$301F Bit 3 set (cranking-initialized)
dp $68/$69/$6A/$6B Cylinder 1–4 dwell counters (set to 0x0C)
dp $62 Bits 0–3 set (injection outputs enabled for all 4 cylinders)
