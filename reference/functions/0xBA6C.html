0xBA6C — RPM/AFM Rate-of-Change History

Field Value
Address 0xBA6C
Size 55 bytes (0xBA6C–0xBAA2)
Blocks 1
Type subroutine (ISR callee)
Callers 1 — scheduler P4 (0xAAD1 at 0xAB62)
Confidence confirmed

Summary

Maintains two 3-sample history buffers and computes rate-of-change deltas for RPM and AFM signals. The function is split into two halves:

RPM history (0xBA6C–0xBA8A): Uses a 3-entry FIFO at $32D2–$32D7 (16-bit values). On each call, shifts entries: [2] = [1], [1] = [0], [0] = $3064 (current full-precision RPM). Computes two deltas:

  • $3268 = $3064 - $32D4 (current minus 1-sample-ago = first derivative)
  • $326A = $3064 - $32D6 (current minus 2-samples-ago = second derivative)

AFM history (0xBA8B–0xBAA2): Uses a 3-entry FIFO at $32D8–$32DD (8-bit values, indexed via Y register in SEM mode). On each call, shifts entries: [2] = [1], [1] = [0], [0] = $3748 (AFM diagnostic processed). Computes:

  • $32DD = $3748 - $32DA (current minus 1-sample-ago)

Inputs

  • $3064 (rpm_full_precision): current 16-bit RPM value
  • $3748 (afm_diag_processed): current 8-bit AFM diagnostic value

Outputs

  • $32D2–$32D7: RPM 3-entry history FIFO (16-bit each)
  • $32D8–$32DD: AFM 3-entry history FIFO (8-bit each)
  • $3268: RPM first derivative (signed, current - previous)
  • $326A: RPM second derivative (signed, current - 2-ago)
  • $32DD: AFM rate-of-change (signed, current - previous)
