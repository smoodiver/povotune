Summary

Two-phase diagnostic check combining an RPM-range sensor test (Phase 1) and a coolant-temperature/load sensor test (Phase 2). Unlike the uniform solenoid checks at 0x73AF–0x74C3, this function tests two different conditions and merges them into a single summary fault bit. Uses a shared counter base at 0x3324 (two consecutive bytes for the two phases).

Phase 1 (0x751F–0x7545): RPM-range sensor check. Loads Y from 0x3308 (an RPM or speed-related value) and compares against a ROM-defined window: if Y <= ROM[0x4C68] or Y >= ROM[0x4C6A], the value is within the acceptable range and the counter is incremented. When the counter reaches ROM[0x4C6C], sets 0x333B bit 0. If Y falls within the exclusion window (between 0x4C68 and 0x4C6A), the counter is cleared and the fault bit is cleared.

Phase 2 (0x7546–0x7587): Coolant-gated RPM window check. Requires 0x316B >= 0x64 (lower threshold than the solenoid checks), 0x305A bit 2 set, and 0x3330 bit 1 clear. Loads Y (retained from Phase 1) and checks if it falls within RPM window 0x0154–0x01CC. If within range, increments second counter; when count reaches 0x0A (10), sets 0x333B bit 1. Outside the window clears counter and fault bit.

Final aggregation: if 0x333B bits 0+1 are both set, sets 0x3331 bit 4; otherwise clears it.

Per-Phase Logic

; Phase 1: RPM-range sensor
X = $3324
Y = [$3308]

if Y <= ROM[$4C68] or Y >= ROM[$4C6A]:
    JSR $633D (increment counter at X)
    if count >= ROM[$4C6C]: SEB $333B bit 0
else:
    LDM $00,X,#$00 (clear counter)
    CLB $333B bit 0

; Phase 2: Coolant-gated RPM window
if $316B < $64: skip
if $305A bit 2 clear: skip
if $3330 bit 1 set: skip

INX (X = $3325)
if Y < $0154 or Y == $0154: check range
    if Y < $01CC: goto out_of_range
in_range:
    JSR $633D (increment counter at X+1)
    if count >= $0A: SEB $333B bit 1
    BRA exit
out_of_range:
    LDM $00,X,#$00 (clear counter)
    CLB $333B bit 1

exit:
    if $333B bits 0+1 both set:
        SEB $3331 bit 4
    else:
        CLB $3331 bit 4
    RTS

Inputs

  • 0x3308: RPM or speed-related input value (16-bit, loaded into Y)
  • 0x316B (lookup_input_b): >= 0x64 required for Phase 2 (lower threshold than solenoid checks)
  • 0x305A (system_control_flags): bit 2 must be set for Phase 2
  • 0x3330: bit 1 must be clear for Phase 2

Outputs

  • 0x3324 (aux_sensor_counter_1): Phase 1 debounce counter
  • 0x3325 (aux_sensor_counter_2): Phase 2 debounce counter
  • 0x333B bit 0: Phase 1 fault (RPM-range sensor out of spec)
  • 0x333B bit 1: Phase 2 fault (coolant-gated RPM window fault)
  • 0x3331 bit 4: summary fault (both phases confirmed)
