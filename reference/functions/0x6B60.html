0x6B60 — Health Check Precondition Gate

Field Value
Address 0x6B60
Size 41 bytes
Blocks 7
Type subroutine
Callers 1 (health_check_dispatcher 0x6B14)
Confidence confirmed

Summary

Precondition gate that checks whether the engine is running and warmed up enough for health checks to begin. If engine is not running (0x3051 bit 4 clear), clears the health-check initialized flag (0x3031 bit 7) and returns. Otherwise, checks three readiness conditions: knock sensor (0x306E >= 0x08), warmup counter (0x316B >= 0x64), and engine run timer (0x30B6 >= 0x000A). When all pass, sets the initialized flag (0x3031 bit 7) and falls through into the diagnostic mode initializer at 0x6B89.

Inputs

  • 0x3051 bit 4: engine running flag (master gate)
  • 0x3031 bit 7: health check initialized flag (checked to skip re-init)
  • 0x306E: knock sensor value (>= 0x08 required)
  • 0x316B: warmup counter (>= 0x64 required)
  • 0x30B6: engine run timer, 16-bit (>= 0x000A required)

Outputs

  • 0x3031 bit 7: set when all preconditions pass; cleared when engine not running

Calls

  • Falls through to 0x6B89 (diag_mode_initializer) when preconditions pass
