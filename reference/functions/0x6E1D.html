0x6E1D — Front O2 Sensor Range Check

Field Value
Address 0x6E1D
Size 27 bytes (0x6E1D–0x6E37)
Blocks 6
Type subroutine
Callers 1 call site (0xB510)
Confidence confirmed

Summary

Computes the absolute difference between two O2 sensor readings (0x32C6 and 0x31FB) and compares it against a fixed threshold of 0x9C (156). If the difference exceeds the threshold, sets the front O2 sensor range fault flag (0x31F8 bit 0). Otherwise clears it.

This is a simple deviation check: if the two values diverge by more than 156 counts, the front O2 sensor is considered out of range.

Disassembly

6E1D  AD C6 32       LDA $32C6           ; Load O2 sensor value A
6E20  38             SEC                 ; Set carry for subtraction
6E21  ED FB 31       SBC $31FB           ; Subtract O2 sensor value B
6E24  B0 03          BCS $6E29           ; If result >= 0, skip negate
6E26  49 FF          EOR #$FF            ; Negate (one's complement)
6E28  3A             INA                 ; +1 = two's complement (absolute value)
6E29  C9 9C          CMP #$9C            ; Compare abs diff vs threshold 156
6E2B  90 06          BCC $6E33           ; If below -> no fault
6E2D  0C F8 31 01    SEB $31F8,#$01      ; Set front O2 range fault: bit 0
6E31  80 04          BRA $6E37           ; -> return
6E33  1C F8 31 01    CLB $31F8,#$01      ; Clear front O2 range fault
6E37  60             RTS

Inputs

  • 0x32C6: O2 sensor reading A (front O2 current value)
  • 0x31FB: O2 sensor reading B (front O2 reference/expected value)

Outputs

  • 0x31F8 bit 0: front O2 sensor range fault

RAM Writes
