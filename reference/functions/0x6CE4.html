0x6CE4 — Intake Air Temp Sensor Check

Field Value
Address 0x6CE4
Size 49 bytes (0x6CE4–0x6D14)
Blocks 9
Type subroutine
Callers 1 call site (health_check_dispatcher 0x6B14, slot 6)
Confidence confirmed

Summary

Checks the intake air temperature sensor (ADC reading at 0x303C) for open/short circuit conditions. Compares the 16-bit ADC value against low threshold 0x0014 (shorted/grounded) and high threshold 0x03D4 (open circuit). If within the valid range, the fault counter at 0x317F is cleared and the fault flag is removed. If out of range, the counter is incremented via saturating increment (0x633D); when it reaches 4, the fault flag 0x332F bit 4 is set.

An additional sub-check distinguishes low vs high out-of-range: values at or below 0x0014 set 0x31CD bit 5 (sensor shorted), while values in the normal-low range clear that bit.

Disassembly

6CE4  AE 3C 30       LDX $303C           ; Load intake air temp ADC (16-bit)
6CE7  E0 14 00       CPX #$0014          ; Compare against low threshold
6CEA  F0 1A          BEQ $6D06           ; If equal -> shorted path
6CEC  90 18          BCC $6D06           ; If below -> shorted path
6CEE  1C CD 31 20    CLB $31CD,#$20      ; Clear sensor-shorted flag (bit 5)
6CF2  E0 D4 03       CPX #$03D4          ; Compare against high threshold (open circuit)
6CF5  90 15          BCC $6D0C           ; If below -> value in range, clear fault
6CF7  A2 7F 31       LDX #$317F          ; Load counter address
6CFA  20 3D 63       JSR $633D           ; Saturating increment counter
6CFD  C9 04          CMP #$04            ; Trip threshold = 4
6CFF  90 04          BCC $6D05           ; Not yet tripped -> return
6D01  0C 2F 33 10    SEB $332F,#$10      ; Set fault: 0x332F bit 4 = intake air temp fault
6D05  60             RTS
6D06  0C CD 31 20    SEB $31CD,#$20      ; Set sensor-shorted flag (bit 5)
6D0A  80 EB          BRA $6CF7           ; Jump to increment counter path
6D0C  9C 7F 31 00    LDM $317F,#$00      ; Clear fault counter
6D10  1C 2F 33 10    CLB $332F,#$10      ; Clear fault flag
6D14  60             RTS

Inputs

  • 0x303C (16-bit): intake air temperature ADC reading
  • 0x317F: fault debounce counter
  • 0x31CD: sensor status flags

Outputs

  • 0x332F bit 4: intake air temp sensor fault (set when confirmed)
  • 0x31CD bit 5: sensor shorted indicator
  • 0x317F: updated fault counter

RAM Reads

Address Name Use
0x303C ad_initial_ch1 Intake air temp ADC value (16-bit)
0x317F iat_fault_counter Debounce counter for fault detection

RAM Writes

Address Name Use
0x317F iat_fault_counter Incremented on fault, cleared on valid reading
0x31CD sensor_status_flags Bit 5: sensor shorted indicator
0x332F temp_override_flags Bit 4: intake air temp sensor fault

ROM References

None (thresholds are immediate operands: 0x0014 and 0x03D4).
