0xC9DD — Coolant-Based Fuel Correction

Field Value
Address 0xC9DD
Size 90 bytes (0xC9DD–0xCA36)
Blocks 10
Type subroutine
Callers 1 call site (irq_handler_AAD1 main loop at 0xABE0)
Confidence confirmed

Summary

Computes a coolant-temperature-dependent fuel correction value and selects an engine state correction factor. The first half (gated by system_control_flags bit 2) looks up coolant_temp through a 2D table at 0x4310 via graph_lookup_2d, extracts the high byte shifted right 1, stores to 0x3124, then selects one of three correction factors (ROM 0x403C, 0x403E, or 0x4030) based on coolant_temp_raw compared against thresholds 0x403B and 0x4037, storing the result to 0x30CE. The second half (gated by system_control_flags bit 4) optionally scales 0x3124 by a ROM multiplier at 0x403F; otherwise halves it. The final result goes to fuel_correction_addend (0x3126), and the low byte is mirrored to IO port 0x0132.

Inputs

  • 0x305A (system_control_flags): bit 2 gates the lookup section, bit 4 gates the scaling section
  • 0x3093 (coolant_temp): 2D table lookup input
  • 0x3092 (coolant_temp_raw): threshold comparisons for factor selection

Outputs

  • 0x3009 (scratch): table lookup result (low byte)
  • 0x3124 (coolant_correction_base): intermediate 16-bit correction value
  • 0x30CE (coolant_correction_factor): selected correction factor from ROM
  • 0x3126 (fuel_correction_addend): final scaled correction output
  • 0x0132 (IO port): low byte of correction result

Calls

  • 0x6000 — graph_lookup_2d (1 call: coolant_temp through table 0x4310)

RAM Reads

  • 0x305A (system_control_flags)
  • 0x3093 (coolant_temp)
  • 0x3092 (coolant_temp_raw)
  • 0x3124 (coolant_correction_base)

RAM Writes

  • 0x3009 (scratch)
  • 0x3124 (coolant_correction_base)
  • 0x30CE (coolant_correction_factor)
  • 0x3126 (fuel_correction_addend)

ROM References

  • 0x4310 (coolant_correction_table): 2D lookup table for coolant temp to correction base
  • 0x403B (coolant_factor_threshold_1): first comparison threshold for coolant_temp_raw
  • 0x4037 (coolant_factor_threshold_2): second comparison threshold for coolant_temp_raw
  • 0x403C (coolant_factor_low): correction factor when coolant_temp_raw < threshold_1
  • 0x403E (coolant_factor_mid): correction factor when threshold_1 <= coolant_temp_raw < threshold_2
  • 0x4030 (coolant_factor_high): correction factor when coolant_temp_raw >= threshold_2
  • 0x403F (correction_scale_multiplier): 16-bit multiplier for the scaling path
