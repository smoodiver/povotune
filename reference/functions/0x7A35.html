0x7A35 — Closed-Loop Readiness Classifier

Size ~92 bytes (0x7A35–0x7A90)
Blocks 20
Type subroutine
Callers 1 (turbo boost solenoid check at 0x7205, inside health_check_dispatcher slot 13)

Summary

Classifies the engine's closed-loop fuel control readiness based on vehicle speed, coolant temperature, and airflow, producing a 0–3 readiness level stored in 0x3211. Uses hysteresis flags in 0x31EA (bits 3, 4) to prevent oscillation at zone boundaries.

The function starts by loading B = 0 (base readiness level) and checking two entry conditions:

Path A — Speed sensor active, low speed (0x305A bit 4 set, speed < 10): Checks airflow (0x306F) against three zones:

  • Airflow >= 0x38 (56): high airflow — sets 0x31EA bit 3 (warm flag), readiness = 3
  • Airflow >= 0x18 (24) and < 0x38: medium airflow — uses 0x31EA bit 3 as hysteresis (if previously warm, readiness = 2; if not, readiness = 2)
  • Airflow < 0x18 (24): low airflow — clears 0x31EA bit 3, readiness = 0

Path B — Speed sensor inactive OR speed >= 10: Checks speed (0x3071) to further classify:

  • Speed < 10: clears 0x31EA bit 4, classifies based on coolant temp (0x3044): < 10 = readiness 1, < 0xDB (219) = readiness 2, >= 0xDB = readiness 3
  • Speed >= 10 and < 30: uses 0x31EA bit 4 as hysteresis — if previously set, falls through to coolant classification; otherwise readiness = 2
  • Speed >= 30: sets 0x31EA bit 4, readiness = 2

The readiness level (B register, 0–3) is stored in 0x3211 via STB $3211.

Inputs

  • 0x3071 (vehicle_speed): current vehicle speed
  • 0x306F (airflow): current airflow reading
  • 0x3044 (coolant_temp): coolant temperature
  • 0x305A bit 4: speed sensor validity flag
  • 0x31EA bits 3, 4: hysteresis flags

Outputs

  • 0x3211: closed-loop readiness level (0–3)
  • 0x31EA bit 3: airflow zone hysteresis
  • 0x31EA bit 4: speed zone hysteresis
